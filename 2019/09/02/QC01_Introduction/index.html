前言

如果用量子力学的规则替换二进制逻辑来计算，某些用经典计算机难以攻克的计算任务将得到解决。这就是量子计算的核心问题。

所谓的量子力学的规则，在量子计算的三大核心体现：叠加、量子测量和纠缠。对于超越经典计算机的计算能力起名“量子霸权（quantum supremacy）”。

运行量子规则的物理实现有：离子阱，超导量子比特，光子，拓扑超导的任意子等。

  • 2013年5月D-Wave 系统公司宣称NASA和Google共同预定了一台采用512量子位的D-Wave Two量子计算机。
  • 2017年5月，中国科学院宣布制造出世界首台超越早期经典计算机的光量子计算机。
  • 2017年7月，哈佛大学米哈伊尔·卢金（Mikhail Lukin）宣布：用激光冷却的原子完成51个量子比特的量子计算机模拟器[24]。
  • 2019年1月8日，IBM展示了20量子比特的量子计算机IBM Q System One

需要澄清的一点是，目前没有证明在所有计算领域量子计算机会都比经典计算机有更强的计算能力，所以有一个分领域是研究有哪些算法上，量子计算机比经典计算机有更强的计算能力。目前常见的量子算法有

  • Shor算法：质数分解
  • Grover算法：搜索算法
