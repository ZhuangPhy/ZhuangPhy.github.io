2.1 模 型 序 列

如果我们能够定量同时定性地说明临界现象如何能够用第一性原理从微观模 型导出, 我们就有了一个完整的理论.

然而，在做这样的尝试之前, 我们需要考察 各个模型的优点。在这个考察的过程中, 我们将引出一些基本的想法。 微观模型的判据并不是固定的, 它依赖于所感兴趣的现象。这一点在下面能够 导出临界现象的模型序列中演示。为了具体起见, 我们将讨论限制在给定的晶 体中的铁磁临界现象。

模型(1): 万有模型

模型(1)：由库仑力相互作用的电子和原子核。

这肯定是能够导出几乎所有现象, 包括临界现象的微观模型。但作为研究临 界现象的出发点, 这个模型显然是不切实际的。

模型 (2): 晶格模型

模型 (2): 在一个指定晶格内的具有有效相互作用的电子。

现在, 我们假定晶格结构为已知的, 并且理所当然地认为指定电子-电子相互作用. 能带结构、晶体场等的参数已经由原子物理和固体物理学家基于模型得到。

这个模型比模型(1) 更适合分析临界现象, 这是由于晶格结构的形成和原子 的内层电子离开临界现象太远. 我们期望这些已经是给定的。

由于临界现象是电子自旋在大尺度上的集体行为, 除了对于电子自旋之间相 互作用的总体贡献外, 我们或许不需要知道能带结构以及许多细节。这个模型还 可以进一步简化。

模型(3): 自旋模型

模型 (3): 在给定的晶格的每个原胞中放置一个经典自旋, 并指明 自旋之间的 相互作用.

这里, 量子本质、电子运动以及许多模型 (2) 的细节都忽略了。自旋-自旋相 互作用由若干参数指定, 通过使得尽可能模拟模型 (2) 的结论来调整这些参数的值. 这种模拟是非平庸的 , 这些模型中最常用的一些版本, 如伊辛模型和海森伯模型 (会在后面定义这些模型) 是非常粗橙的。然而, 我 们认为重要的物理在于 大量的电子自旋一起表现出来的行为。因此在单个原胞上有点粗粒超无关紧要. 这个模型对临界现象而言与模型 (1) 和模型 (2) 同样微观。

模型 (4): 块自旋模型

模型 (4): 在每个由 \(2 \times 2 \times 2\) 个原胞构成的块上放置一个经典自旋, 并指明 自旋 之间的相互作用。

这在消去细节上比模型(3) 又前进了一步. 这里的每个“自旋”是模型(3) 的8 个自旋的网格。同样, 如果只涉及大尺度的行为, 除了总体效应外, 我们不在意 每个块的细节以及块上自旋之间的相互作用。这个模型显然不比模型 (3) 不微观.

模型(5): 更大块上的自旋。

代替 \(2 \times 2 \times 2\) 的块, 我们可以取 \(3 \times 3 \times 3\) 甚至 \(10 \times 10 \times 10\) 的块. 在构成更 大的块上我们能够走多远, 同时还能声称是微观模型？这里没有清楚的答案。但 定性而言, 答案是块的尺寸必须远小于临界现象的特征尺寸，即第 1 章讲过的自 旋斑块的平均尺寸。如果我们认为这些自旋斑块在 \(T_{ }\) 接近于 \(T_{\mathrm{c}}\) 时不断变大, 那么块的尺寸在 T 非常接近于 \(T_{\mathrm{c}}\) 时可以取得非常大. 块的构造的想法和应用是由Kadanoff (1966,1967) 给出的。

模型的选择

在上述的模型序列中, 那些我们认为与临界现象无关的细节被逐步消除。当 完成这样的消除过程时, 为了避免巨大的数学上的复杂性, 简化是一个实际上的必要手段。

通常，模型中的参数由实验数据来确定，因为计算是不实际的。无需指明的是, 例如模型 (1) 中的电子电荷就是由实验确定的. 模型 (2) 中的晶格结构可以用 X 射线散射确定. 对于模型 (3) (5), 我们可以通过拟合在温度不太靠近 \(T_{\mathrm{c}}\) 处的 实验数据来定出参数。

传统上, 临界现象的大多数理论研究始于模型 (3) 的不同版本。我们将主要 考察模型 (5). 金兹堡-朗道模型可以认为是模型 (5) 的一个粗彭版本. 当然, 先介 绍模型 (3) 并演示如何由此推出模型 (4) 和模型 (5) 的方式, 具有示范意义。这就 是本章所用的方式。

2.2 原胞哈密顿的经典模型

2.2.1 伊辛模型

每个原胞内有一个自旋变量 \(\sigma_{c},\) 它可以衡量原胞 \(c\) 的总自旋. 共有 \(L^{3}\) 个原胞。

伊辛模型的原胞哈密顿: \[ \hat{H}[\sigma]=\frac{1}{2} J \sum_{c} \sum_{r}^{\prime}\left(\sigma_{c}-\sigma_{c+r}\right)^{2} \]

  • 对 \(r\) 的求和只对 \(c\) 的近邻进行。
  • 自旋变量只取两个值: \(\sigma_{c}=\pm 1\)

自旋与其近邻同向具有比反向更低能量，是对于单轴铁磁体的模拟。

推广 : \[ \hat{H}[\sigma]=\frac{1}{2} J \sum_{c} \sum_{r}^{\prime}\left(\sigma_{c}-\sigma_{c+r}\right)^{2}+\sum_{c} U\left(\sigma_{c}^{2}\right) \]

  • 认为每个 \(\sigma_{c}\) 是一个连续变量.
  • 附加的能量 \(U\left(\sigma_{c}^{2}\right)\) 很大

通过调节 \(U\left(\sigma_{c}^{2}\right)\) 的形状, 我们可以计人 电子扩展到近邻原胞时能量的增加. 还可以 进一步推广到包含次近邻相互作用以及其他 效应, 我们暂不讨论这些推广。

2.2.2 $ X Y$ 模型和海森伯模型

如果没有限定自旋的指向只能沿着某个轴, 而是允许指向任意方向, 我们需 要用下面的矢量来描述每个自旋: \[ \sigma _{ c }=\left(\sigma_{1 c }, \sigma_{2 c }, \sigma_{3 c}\right) \] 则 \(\hat{H}[\sigma]=\frac{1}{2} J \sum_{c} \sum_{r}^{\prime}\left(\sigma_{c}-\sigma_{c+r}\right)^{2}\) 就变为海森伯模型的原胞哈密顿. 它模拟各向同性的铁磁体。注意我们把 \(\sigma_{c}\) 当作一个普通矢量, 而不是泡利矩阵.

\(X Y\) 模型介于伊辛模型和海森伯模型之间. 它模拟自旋指向基本上在一个平 面内的磁体. 我们可以把 \(\sigma_{c}\) 看作一个二分量的矢量 \(\left(\sigma_{1 c}, \sigma_{2 c}\right) .\) 原胞哈密顿仍然 具有 \(\hat{H}[\sigma]=\frac{1}{2} J \sum_{c} \sum_{r}^{\prime}\left(\sigma_{c}-\sigma_{c+r}\right)^{2}\) 的形式.

2.2.3 空间维数 d 和自旋维数 \(n\)

我们引人记号 n 标记自旋矢量 \(\sigma_{c}\) 的分量数. 从而

  • 伊辛模型、
  • XY 模型
  • 海 森伯模型,

分别对应于 \(n=1,2\) 和 3.

记号 \(d\) 用来标记晶体占据的空间维数.

  • 通 常, \(d=3 .\)
  • 对于一个单层的自旋, \(d=2 ;\)
  • 对于一个链, \(d=1\)

在以后的讨论中, 我们将会考虑 d 和 \(n\) 的非物理的值. 由于每个自旋具有 n 个分量, 显然自旋变量的总数是 \(n L^{d} .\) 在接下去的讨论中, 我们将把 n 和 d 作为参数, 于是关于伊辛模型、XY 模型或海森伯模型的表述可以通过对 \(n\) 和 d 赋以合适的数值而得到

2.3 统 计 力 学

在概率论的语言下, \(n L^{d}\) 个自旋变量的每一个都是一个随机变量. 这些随机 变量的联合概率分布 P 由统计力学给出: \[ P=Z^{-1} e ^{-\hat{H}(\sigma) / T} \] 这里 Z 是一个归一化的常数, 称为配分函数: \[ Z=\int e ^{-\hat{H}(\sigma) / T} \prod_{i, c} d \sigma_{i c} \] 这里乘积对所有原胞和所有分量进行. 这个积分是有 \(n L^{d}\) 个变量的多重积分.

平均值

作为自旋变量的函数的任何一个物理量的平均值, 可以通过对于 P 的积分来计算. 例如 \[ \left\langle\sigma_{i a} \sigma_{j a^{\prime}}\right\rangle=\int \sigma_{i a} \sigma_{j a^{\prime}} P \prod_{i, c} d \sigma_{i c} \] 自旋矢量的傅里叶分量 \(\sigma _{k}\) 将在其后的讨论中起重要作用: \[ \sigma_{c}=L^{-d / 2} \sum_{k} e ^{ i k \cdot c} \sigma_{k} \]

\[ \sigma_{k}=L^{-d / 2} \sum_{c} e ^{- i k \cdot c } \sigma_{c} \]

对波矢量 \(k\) 的求和取第一布里渊区的 \(L^{d}\) 个分立值. 由于 \(L^{ }\) 是一个非常大的数, 故 \(k\) 空间中点的密度 \(L^{d}(2 \pi)^{-d}\) 是非常大的.

  • 我们既可以把原胞哈密顿 \(\hat{H}(\sigma)\) 看 作是 \(\sigma_{c}\) 的函数
  • 也可以把它看作是 \(\sigma_{k}\) 的函数

关联函数

散射实验测量关联函数 \[ \left\langle\left|\sigma_{i k}\right|^{2}\right\rangle \equiv G( k ) \] 如果原胞哈密顿对于自旋变量各向同性, 则它与 \(i\) 无关.

注意到原胞哈密顿对自旋位形的描述达到了最小的距离，即单胞的大小。在 单胞内自旋的变化超出了原胞哈密顿的描述. 由式 \(\sigma _{ k }=L^{-d / 2} \sum_{ c } e ^{- i k \cdot c } \sigma _{c}\) 定义的 \(\sigma_{k},\) 与由式\(\sigma_{k}=V^{-1 / 2} \int d ^{3} x e ^{- i k \cdot x} \sigma(x)\) 定义的一样, 足以说明自旋变化在大于单胞的尺度上的变化, 且是哈密顿中仅有 的随机变量.

热力学性质

让我们在哈密顿中加一项 \[ -h \sum_{c} \sigma_{1 c} \] 即在方向 1 加上一个磁场 \(h\). 令单位体积的自由能 \(F\) 通过式 \(Z=\int e ^{-\hat{H}(\sigma) / T} \prod_{i, c} d \sigma_{i c}\) 定义 \[ Z= e ^{-F L^{d} / T} \] \(F\) 是 \(h, T\) 以及指明原胞哈密顿的参量的函数. 一旦知道了 \(F\) , 热力学性质就确定 \[ \begin{aligned} &\text { 熵 } \quad S=-\frac{\partial F}{\partial T}\\ &\text { 比热 } \quad c=T \frac{\partial S}{\partial T}=-T \frac{\partial^{2} F}{\partial T^{2}}\\ &\text {磁化 } \quad m=-\left(\frac{\partial F}{\partial h}\right)_{T}\\ &\text {磁化率 } \quad \chi=\left(\frac{\partial m}{\partial h}\right)_{T}=-\left(\frac{\partial^{2} F}{\partial h^{2}}\right)_{T} \end{aligned} \] 注意 \(\chi / T=G(0)\) 来自 \(G( 0 )=\int d ^{3} x \langle[\sigma( x )-\langle\sigma\rangle][\sigma( 0 )-\langle\sigma\rangle]\rangle\) 和 \(\chi=\left(\frac{\partial m}{\partial h}\right)_{T}=-\left(\frac{\partial^{2} F}{\partial h^{2}}\right)_{T}\)

奇异性

这些公式与类似于式 \(Z=\int e ^{-\hat{H}(\sigma) / T} \prod_{i, c} d \sigma_{i c}\) 和式 \(\left\langle\sigma_{i a} \sigma_{j a^{\prime}}\right\rangle=\int \sigma_{i a} \sigma_{j a^{\prime}} P \prod_{i, c} d \sigma_{i c}\) 的积分一起, 构成一套用于计算的完备数学框架. 求得的物理量 是 \(h\) 和 \(T\) 的函数. 如果这一框架确实描述了自然, 则计算得到的物理量必须与 观察相符. 特别地, 观察到的奇异性必须出现在计算结果中.

原胞哈密顿 \(\hat{H}\) 是 \(T, h\) 和其他参数的光滑函数, 对于 \(\exp (-\hat{H} / T)\) 的积分是 很好定义了的. 奇异性是如何从这些公式中得出来的呢？如果只有有限数目的 自旋, 确实不会出现奇异性。它们只有在自旋变量的数目趋于无限时才会出现. 很难看清这个极限是如何产生奇异性的. 事实上, 解释一个光滑的哈密顿如何在 \(L^{d} \rightarrow \infty\) 时的极限下出现奇异性, 就是临界现象理论的基本任务.

种方法是精确计算然后取 \(L^{d} \rightarrow \infty\) 时的极限. 这实际上非常困难. 临界现 象的大部分理论工作用在这个任务上. 这个方向上最早和最重要的进展是伊辛模 型在 d=2 时的昂萨格解。它的重要性在于: 这个解明确和定量地显示了 T_ 附近 观察到的奇异行为, 确实可以通过统计力学规律从一个非奇异的哈密顿得到, 尽 管得到的过程非常复杂. 昂萨格计算积分的技巧得到了推进和进一步的发展. 对 于无法得到精确解的模型，人们也做了广泛的数值计算. \[ \begin{array}{c|c|c|c|c|} \hline & \text { 空间维数 } d & \text { 序参量维数 } n & \alpha & \beta \\ \hline \text { 伊辛 } & 2 & 1 & 0^{ O }(\text { 对数 }) & 0.125^{ O } \\ \hline \text { 海森伯 } & 2 & 3 & & \\ \hline \text { 伊辛 } & 3 & 1 & 0.013 \pm 0.01 & 0.312_{-0.005}^{+0.002} \\ \hline X Y, \text { 自旋 } 1 / 2 & 3 & 2 & & \\ \hline X Y & 3 & 2 & -0.02 \pm 0.03 & \\ \hline \text { 海森伯, 自旋 } 1 / 2 & 3 & 3 & -0.20 \pm 0.08^{(2)} & \\ \hline \text { 海森伯 } & 3 & 3 & -0.14 \pm 0.06 & 0.38 \pm 0.03 \\ \hline \text { 球模型 } & 3 & \infty & -1^{ O } & 0.5^{ O } \\ \hline \end{array} \] 表 2.1 列出了一些结果. 尽管这样的数学工作为我们提供了精确的数值和解析信息, 但它不能给出很多物 理理解, 从而对如何取近似无法提供更多启示. 实际情况大致是, 我们或者通过巨 大的数学工作得到精确答案, 或者得到很少甚至是坏的答案. 没有通过较少的工 作量获得近似答案的方案. 这不是我们期望的状况. 我们想要计算的物理量, 如比 热和磁化率, 是温度的奇异函数. 经验告诉我们, 对于奇异函数, 近似方法通常是 不灵的. 进一步, 如观察结果所显示的, 这些奇异函数有一些普适的行为 (见 1.5 节)。由于答案不依赖于细节, 在哈密顿上做近似并不能使问题简化. 我们将不试 图探索精确计算自由能和其他物理量的方法, 这些方法在已有的书和文章中有广 泛的讨论.

在随后的讨论中, 经典模型仅仅作为研究定性行为的方便基础以及用于重正化群方法.

2.4 块哈密顿和卡丹诺夫少换

前面讨论的原胞哈密顿描述原胞自旋之间的相互作用. 原胞哈密顿的参数集 合了小于原胞尺度上各种细节的效应. 显而易见, 我们可以从原胞哈密顿构造一 个块哈密顿, 描述坏自旋之间的相互作用. 这里我们把晶体划分成立方体块, 每一 块包含 \(b^{d}\) 个原胞. 例如, \(b=2\) 或 5. 我们定义块自旋为块中的 b ^d 个原胞自旋之 和除以 \(b^{d},\) 即块中原胞自旋的平均值. 块哈密顿中的参数集合了 b 个格点常数中 相关细节的效应.

给定了原胞的哈密顿, 我们如何构造块哈密顿？

概率分布

原则上这是直截了当的. 在 写出答案之前，让我们记住下面的例子所演示的概率论中的一个简单规则。设 \(P\left(q_{1}, q_{2}\right)\) 是两个随机变量 \(q_{1}, q_{2}\) 的概率分布, 我们定义 \[ q=\frac{1}{2}\left(q_{1}+q_{2}\right) \] 我们如何构造 \(q\) 的概率分布呢？规则是 \[ \begin{aligned} P^{\prime}(q) &=\int d q_{1} d q_{2} \delta\left(q-\frac{1}{2}\left(q_{1}+q_{2}\right)\right) P\left(q_{1}, q_{2}\right) \\ &=\left\langle\delta\left(q-\frac{1}{2}\left(q_{1}+q_{2}\right)\right)\right\rangle_{P} \end{aligned} \] 如果只涉及与 \(q\) 相关的平均值, 则 \(P^{\prime}(q)\) 与 \(P\left(q_{1}, q_{2}\right)\) 的作用完全相同. 例如 \[ \left\langle q^{2}\right\rangle=\int d q q^{2} P^{\prime}(q)=\left\langle\frac{\left(q_{1}+q_{2}\right)^{2}}{4}\right\rangle_{P} \] 对 \(q\) 的积分消除 \(\delta\) 函数并把 \(q\) 替换为 \(\left(q_{1}+q_{2}\right) / 2 .\)

块自旋

我们用块的中心的坐标 \(x\) 来标志块. 如上所述, 块自旋定义为块内的净自旋 除以块内的原胞数: \[ \sigma_{x}=b^{-d} \sum_{c}^{x} \sigma_{c} \] 这里求和号中的上标 \(x\) 表示对块内的 \(b^{d}\) 个原胞求和. \(\sigma_{x}\) 的分量是 \(\sigma_{i x},\) 同前, \(i\) 从 1 到 \(n .\) 存在 \(L^{d} / b^{d}\) 个块自旋. 从而, \(\sigma_{x}\) 就是由 \(x\) 标记的块内原胞自旋的“均值”如果 \(b=1,\) 则块自旋与原胞自旋相同. 如果 \(b=L,\) 则块自旋就是整个系统中 原胞自旋的均值. 注意, 我们在这里使用“均值”而没有用“平均”来描述自旋的 和除以自旋数。“平均”这个词将只用于对正则系综的统计平均. 在块的尺寸坛于 无穷时, 也只有在此极限下, 我们期望块自旋的概率分布在平均值处有无穷尖锐 的峰. 按照式 (2.13) 所示的规则, 我们写出块自旋的概率分布 \(P^{\prime}(\sigma)\) 如下： \[ \begin{aligned} P^{\prime}(\sigma) &=\left\langle\prod_{i x} \delta\left(\sigma_{i x}-b^{-d} \sum_{c}{ }^{x} \sigma_{i c}\right)\right\rangle_{P} \\ & \propto \int e ^{-\hat{H}[\sigma] / T} \prod_{i, x} \delta\left(\sigma_{i x}-b^{-d} \sum_{c}{ }^{x} \sigma_{i c}\right) \prod_{j, c} d \sigma_{j c} \\ & \equiv e ^{-H[\sigma] / T} \end{aligned} \] 这里 c 取所有的原胞, \(i\) 和 \(j\) 从 1 到 \(n .\) 块自旋的函数 \(H[\sigma]\) 就是需要的块哈密顿. 如果没有 \(\delta\) 函数的限制，它就是总的自由能。注意到式 (2.5) 和式 (2.16) 的定义, 我们有 \[ Z=\int e ^{-H[\sigma] / T} \prod_{i, x} d \sigma_{i x} \] 块自旋 \(\sigma_{x}\) 可以描述在最小长度 b 之上的自旋变化, 即块哈密顿 \(H\) 的空间分 辨能力为 \(b\), 而原胞哈密顿 央 的空间分辨能力为 1. 块哈密顿的空间分辨能力较 弱. 但是, 如果我们对自旋在小于 \(b\) 的尺度上的变化没有兴趣, 那么, 在给出同样 的平均值这一点上, 块哈密顿和原胞哈密顿是等价的. 在临界现象的研究中, 我们 仅对自旋在远大于 1 个晶格常数的长波的变化感兴趣.

块哈密顿

我们可以定义另外一个块哈密顿，它与前面的定义相似但不完全相同。我们 把原胞哈密顿 \(\hat{H}[\sigma]\) 通过式 \(\sigma_{k}=L^{-d / 2} \sum_{c} e ^{- i k \cdot c} \sigma_{c}\) 用傅里叶分量 \(\sigma_{k}\) 写出. 这是一个简单的变量变 换. 这样, \(e ^{-\hat{H} / T}\) 为 \(n L^{d}\) 个新的随机变量 \(\sigma_{i k}\) 的概率分布. 同样, 让我们记住概率 论的一个简单的规则, 这个规则本质上与式 \(P^{\prime}(q)=\int d q_{1} d q_{2} \delta\left(q-\frac{1}{2}\left(q_{1}+q_{2}\right)\right) P\left(q_{1}, q_{2}\right)\) 演示的相同. 考虑两个随机变 量的概率分布 \(P\left(q_{1}, q_{2}\right) .\) 如果我们只对 \(q_{1}\) 感兴趣, 可以积分掉 \(q_{2}\) 而得到 \(q_{1}\) 的概 \(P^{\prime}\left(q_{1}\right)\) \[ P^{\prime}\left(q_{1}\right)=\int d q_{2} P\left(q_{1}, q_{2}\right) \] 因此, 如果我们只对 \(k\) 的幅度小于某个 \(\Lambda\) 的 \(\sigma_{k}\) 感兴趣, 可以从 \(e ^{-\hat{H} / T}\) 中积分掉 所有 \(k>\Lambda\) 的那些 \(\sigma_{i k}\) \[ P^{\prime} \propto \int e ^{-\hat{H}[\sigma] / T} \prod_{i, k>A} d \sigma_{i k} \equiv e ^{-H[\sigma] / T} \] 现在, \(P^{\prime}\) 是处于半径为 \(\Lambda\) 的球内的 \(k\) 对应的 \(\sigma_{i k}\) 的概率分布. 在下述意义下,

  • 这 些变量的函数 \(H[\sigma]\) 是一个块哈密顿: 没有 \(k>\Lambda\) 的傅里叶分量, 自旋在约短于 \(2 \pi A^{-1}\) 的尺度上的变化无法指定. 也就是说, \(H[\sigma]\) 的空间分辨能力是 \(2 \pi \Lambda^{-1} .\)
  • 如果我们指定 \(2 \pi \Lambda^{-1}\) 作为块的尺寸 b, 那么对于式 \(\int e ^{-\hat{H}[\sigma] / T} \prod_{i, k>A} d \sigma_{i k} \equiv e ^{-H[\sigma] / T}\) 的 \(H[\sigma]\) 的解释与由式 \(\int e ^{-\hat{H}[\sigma] / T} \prod_{i, k>A} d \sigma_{i k} \equiv e ^{-H[\sigma] / T}\) 定义的 \(H[\sigma]\) 的解释相似.
  • 在式 \(\int e^{-\hat{H}[\sigma] / T} \prod_{i, x} \delta\left(\sigma_{i x}-b^{-d} \sum_{c}^{x} \sigma_{i c}\right) \prod_{j, c} d \sigma_{j c}\equiv e ^{-H[\sigma] / T}\) 中, 我们积分掉每个块中的细节, 将 空间分辨能力退化到尺寸 b.
  • 在式 \(\int e ^{-\hat{H}[\sigma] / T} \prod_{i, k>A} d \sigma_{i k} \equiv e ^{-H[\sigma] / T}\) 中, 我们通过积分掉高阶傅里叶分量来 达到同样的效果. 定量上, 两个块哈密顿不是完全相同的. 对于我们的定性讨论, 我们将忽略掉这个差别.

Kadanoff 变换

类似于式 \(\sigma_{x}=b^{-d} \sum_{c}^{x} \sigma_{c}\) , 我们定义块自旋 \[ \sigma ( x )=L^{-d / 2} \sum_{k<\Lambda} \sigma _{k} e ^{i k \cdot x } \] 它指定了直到距离 \(b \approx A^{-1}\) 的自旋位形. 定性而言, \(\sigma(x)\) 与 \(\sigma_{x}\) 相同, 是一个块 内自旋的均值. 通过式 \(\int e^{-\hat{H}[\sigma] / T} \prod_{i, k>A} d \sigma_{i k} \equiv e^{-H[\sigma] / T}\) , 由原胞哈密顿 \(\hat{H}[\sigma]\) 得到块哈密顿 \(H[\sigma]\) 的程序将 称为卡丹诺夫 (Kadanoff) 变换. 我们将把这个程序用符号表示为 \[ H[\sigma] / T=K_{b} \hat{H}[\sigma] / T \] 这里下标 b 表示块的尺寸是原胞尺寸的 b 倍. 令 \(K_{1}=1\)

连续变换

显而易见, 给定了块哈密顿 \(H[\sigma]\), 我们可以对定义在更大的块上的块自旋构 造块哈密顿 \(H^{\prime \prime}[\sigma] .\) 例如, 我们可以合并 2 \(^{d}\) 个坏得到一个新的块. 构造新的块哈 密顿的方法与从原胞哈密顿构造老的块哈密顿的方式相同. 这意味着把空间分辨 能力继续下降一个因子 2, 即作用 \(K_{2}\). 我们把合并 \(s^{d}\) 个块的操作写为 \[ \begin{array}{l} H^{\prime \prime}[\sigma] / T=K_{s} H[\sigma] / T \\ e ^{-H^{\prime \prime}[\sigma] / T}=\int e ^{-H\left[\sigma^{\prime}\right] / T} \prod_{i, x} \delta\left(\sigma_{i x}-s^{-d} \sum_{y}^{x} \sigma_{i y}^{\prime}\right) \prod_{j, y} d \sigma_{j y}^{\prime} \end{array} \] 这里 \(s^{-d} \sum_{y}^{x} \sigma_{i y}^{\prime}\) 是中心位于 \(x\) 的新自旋块中 \(s^{d}\) 个老块的自旋均值. 相应地, \(K_{s}\) 也可以用傅里叶分量来定义： \[ e ^{-H^{\prime \prime}[\sigma] / T}=\int e ^{-H[\sigma] / T} \prod_{i, \Delta>k>\Lambda / s} d \sigma_{i k} \] 这一结果来自式 \(\int e ^{-\hat{H}[\sigma] / T} \prod_{i, k>A} d \sigma_{i k} \equiv e ^{-H[\sigma] / T}\), 并把截断波数从 \(\Lambda\) 下降到 \(\Lambda / s .\) 显然 \[ K_{s} H[\sigma] / T=K_{s} K_{b} \hat{H}[\sigma] / T=K_{s b} \hat{H}[\sigma] / T \] 最后一步是显而易见的, 因为合并 \(b^{d}\) 个原胞作为块, 再合并 \(s^{d}\) 个块成为大的块 与合并 (sb) \(^{d}\) 个原胞成为大的块是相同的. 一般地 \[ K_{s} K_{s^{\prime}}=K_{s s^{\prime}} \] 卡丹诺夫变换在构造重正化群时将起到重要的作用.

评论

让我们对几个技术上的问题做一些评论.

傅里叶分量 \(\sigma_{i k}\) 是复数. 在定义式 (2.19) 和式 (2.24) 中, 对 \(d \sigma_{i k}\) 积分时要 小心. 注意到 \(\sigma( x )\) 是实数, 于是 \(\sigma_{i k}^{*}=\sigma_{i-k} .\) 因而涉及 \(\sigma_{i k}\) 和 \(\sigma_{i-k}\) 中只有两个实 变量, 即 Re \(\sigma_{i k}\) 和 \(\operatorname{Im} \sigma_{i k},\) 而不是四个. 像式 (2.19) 和式 (2.24) 这样的积分应该 定义为对于 \(\operatorname{Re} \sigma_{i k}\) 和 \(\operatorname{Im} \sigma_{i k}\) 的积分, 即对于每一对 \(\sigma_{i k}\) 和 \(\sigma_{i-k}\) \[ d \sigma_{i k} d \sigma_{i-k}= d \left(\operatorname{Re} \sigma_{i k}\right) d \left( Im \sigma_{i k}\right) \] 代替分立值, 我们使用连续的自旋变量. 其主要原因是分立的自旋值在卡丹 诺夫变换下基本上变为连续值。例如, 假定每个块自旋可以取值 +1 和 -1. 我们 合并八个块构成新块. 那么新块的自旋, 即八个老自旋的均值, 可以取九个可能的 值: \(\pm 1, \pm 6 / 8, \pm 4 / 8, \pm 2 / 8,0 .\) 九个值使得新的块自旋基本上连续.. 与我们的直 觉一致, 粗粒化将摧毁分立性. 进一步, 如果利用连续变量, 则讨论傅里叶变换要 容易一些. 当然, 可以通过修改新块自旋的定义来保持其二值性, 但将失去新的自旋变 量的确切物理含义. 我们将在第 8 章讨论这一点.

作为总结, 通过模糊化或粗粒化过程, 从原胞哈密顿得到了块哈密顿. 它指定 了在 \(b \approx \Lambda^{-1}\) 尺度上的相互作用, 集中了 \(b^{d}\) 个原胞自旋的粗略特征. 由式 (2.16) 和式 (2.19), 它依赖于 T, 以及其他指定原胞哈密顿的参数, 如式 (2.1) 和式 (2.9) 中的 J 和 h. 由于下述理由, 这种依赖性是非奇异的: 坏自旋之间的相互作用是 短程的, 这是因为它来自原胞自旋的相互作用, 而原胞自旋的相互作用假定为短 程的. 从而块自旋间的相互作用依赖于有限个, 即约 \(b^{d}\) 个, 位于几个块中的原胞 自旋的行为. 我们曾经指出有限个原胞自旋不可能导致奇异行为. 从而块哈密顿 是 T 以及原胞哈密顿的其他参数的光滑函数. 我们在本章的开头演示了模型 (3)(原胞哈密顿) 与模型 (4) 和 (5)(块哈密顿) 之间的关系. 为了得到任意一个从模型 (4) 到 (5)、从模型(3) 到 (4)、从模型 (2) 到 (3) 以及从模型 (1) 到(2) 的明确计算都是非常困难的, 必须做合适的近似. 些明确的计算将在讨论重正化群时给出.

2.5 金兹堡-朗道形式

金兹堡-朗道 (Ginzburg-Landau) 理论在超导理论及其他临界现象中起到了 重要的作用. 它假定了一个块哈密顿的简单形式. 这个简单形式通常写为 \[ H[\sigma] / T=\int d ^{d} x \left(a_{0}+a_{2} \sigma^{2}+a_{4} \sigma^{4}+c(\nabla \sigma)^{2}- h \cdot \sigma \right) \] 这里 \[ \begin{array}{l} \sigma^{2} \equiv \sigma( x ) \cdot \sigma ( x ) \equiv \sum_{i=1}^{n}\left[\sigma_{i}(x)\right]^{2} \\ \sigma^{4} \equiv\left(\sigma^{2}\right)^{2} \\ (\nabla \sigma)^{2} \equiv \sum_{\alpha=1}^{d} \sum_{i=1}^{n}\left(\frac{\partial \sigma_{i}}{\partial x_{\alpha}}\right)^{2} \end{array} \] 系数 \(a_{0}, a_{2}, a_{4}, c\) 是 \(T\) 的函数, \(h\) 由外加磁场除以 \(T\) 得到.

傅里叶

除了外加磁场 \(h,\) 我 们假定系统在自旋空间是各向同性的, 同时, 在位置空间也是各向同性的. 块自旋 \(\sigma(x)\) 由式 \(\sigma ( x )=L^{-d / 2} \sum_{k<\Lambda} \sigma _{k} e ^{i k \cdot x }\) 定义. 它指明直到距离 \(b \approx \Lambda^{-1}\) 的自旋位形. 这个块哈密顿是一 个傅里叶分量 \(\sigma _{k}\) 的分立集合的函数： \[ \begin{aligned} H[\sigma] / T=& a_{0} L^{d}+\sum_{k<\Lambda} \sigma_{k} \cdot \sigma_{-k}\left(a_{2}+c k^{2}\right) \\ &+L^{-d} \sum_{k k^{\prime} k^{\prime \prime}<\Lambda} a_{4}\left(\sigma_{k} \cdot \sigma_{k^{\prime}}\right)\left(\sigma_{k^{\prime \prime}} \cdot \sigma_{-k-k^{\prime}-k^{\prime \prime}}\right)-L^{d / 2} \sigma_{0} \cdot h \end{aligned} \] \(P= e ^{-H[\sigma] / T}\) 是 \(k<\Lambda\) 时 \(\sigma_{k}\) 的概率分布. 我们可以写出一个类似的以 \(\sigma _{ x }\) 为变 量的块哈密顿: \[ H[\sigma] / T=b^{d} \sum_{x}\left[a_{0}+a_{2} \sigma_{x}^{2}+a_{4} \sigma_{x}^{4}+c \frac{b^{-2}}{2} \sum_{y}^{\prime}\left(\sigma_{x}-\sigma_{x+y}\right)^{2}-h \cdot \sigma_{x}\right] \] 这里 \(\sigma_{x}\) 是式 \(\sigma_{x}=b^{-d} \sum_{c}^{x} \sigma_{c}\) 定义的块自旋. 对 \(y\) 的求和对坏 \(x\) 的 2d 个近邻进行. 在一 阶近似下， \(\left( \sigma _{ x + y }- \sigma _{ x }\right) / b\) 是 \(\nabla \sigma\)

各项的意义

通过式上式，可以容易理解块哈密顿的金兹堡-朗道形式各项的意义. \(-h \cdot \sigma_{x}\) 的意义是清楚的，为简单起见可暂时将其忘掉。如果我们丢掉梯度项，即 \(\left( \sigma _{ x }- \sigma _{ x +y}\right)^{2}\) 项, 则上式成为 \[ b^{d} \sum_{ x } U\left( \sigma _{ x }\right) \] 其中 \[ U\left(\sigma_{x}\right)=a_{0}+a_{2} \sigma_{x}^{2}+a_{4} \sigma_{x}^{4} \] 式 \(b^{d} \sum_{x} U\left(\sigma_{x}\right)\) 中的每一项都只依赖于一个块的自旋 \(\sigma_{x},\) 也就是每个块的自旋与其他 块自旋统计独立，这是由于概率分布是一个连乘 \(\prod \exp \left(-U\left(\sigma_{x}\right)\right) .\) 于是, 我们的 系统由 \(L^{d} / b^{d}\) 个无相互作用的块组成. 每个块具有一个有效哈密顿 \(b^{d} U\left(\sigma_{x}\right),\) 这 就是块的自旋均值限制为 \(\sigma_{x}\) 时块的自由能。如前所述, 因为块内只有有限个原 胞自旋, 这个自由能必为 \(\sigma_{x}, T\) 和其他参数的光滑函数. 因而式 \(U\left( \sigma _{ x }\right)=a_{0}+a_{2} \sigma_{x}^{2}+a_{4} \sigma _{ x }^{4}\) 就是这 个自由能的级数展开的前几项. 系数 \(a_{0}, a_{2}, a_{4}\) 以及没有写出来的高阶系数必为 T 和其他参数的光滑函数. 由于只保留到四阶项, 我们认为金兹堡-朗道形式对于 大的 \(\sigma_{x}\) 不是一个好的近似. 如果 \(a_{4}\) 为负值, 这个式子一定是没有意义的, 因为 \(U\left( \sigma _{ x }\right)\) 在 \(\sigma_{ x } \rightarrow \infty\) 时趋于负无穷大, 概率分布 \(P \propto \exp (-H[\sigma] / T)\) 将发散.

当式 \(U\left( \sigma _{ x }\right)=a_{0}+a_{2} \sigma_{x}^{2}+a_{4} \sigma _{ x }^{4}\) 中包含梯度项 \(\left( \sigma _{ x }- \sigma _{ x +y}\right)^{2}\) 时, 块自旋将不再相互独立. 于是,金兹堡-朗道形式以这个简单的梯度项来描述相邻块之间的相互作用. 对于铁磁 体, 我们期望这个相互作用使得块自旋倾向于与其近邻块的自旋平行。梯度项描 述这一倾向, 其强度由系数 c 确定. 这一项只有在所有的块自旋具有相同值时才 为零. 块自旋之间的差别越大, H/T 越大, 从而概率越小. 我们认为块自旋之间的 相互作用并不是完全由梯度项决定的, 应该还有相邻自旋相乘的高阶项. 总而言之, 金兹堡-朗道形式的块哈密顿是最简单但不是最一般的形式. 最一 般的形式应该包括所有的块自旋乘积的景次. 金兹堡-朗道形式只保留了最低阶 项. 当然, 这个形式可以通过把 \(\sigma_{x}^{4}\) 的项丢掉而简化. 但是, 这 是一个过分的简化. 注意我们在块哈密顿中排除了分数景和其他形式的 \(\sigma _{ x }\) 的奇异函数. 排除的 理由在于我们假定自旋之间的相互作用是短程的, 从而块自旋中的项描述的是约 \(b^{d}\) 个自旋的“局部性质”, 因此必须是 \(\sigma_{ x }, h\) 和 \(T\) 的光滑函数.
