1 利用自编码器学习并表示量子态

1.1 自编码器

自编码器 (Autoencoder) 是一种用于数据降维的神经网络结构

一个自编码 器包含了

  • 编码器 (encoder)
  • 译码器 (decoder)

自编码的结构

  • 输入层和输出层
  • 前3层构成 编码器
  • 后3层构成 译码器
  • 中间的code layer的神经元称为潜变量 (latent variables)

应该保持 输出层和 输入层相等。我们可以定义潜变量的个数和输入维度的比值是压缩比 \[ 压缩比=\frac{潜变量个数}{输入维度} \] 当压缩比小于一时，自编码器将输人数据的维度做了降维。

1.2 变分自编码器

和传统的编码器相比，变分自编码器 (Variational autoencoder) 的最大不同之处是潜变量服从宽度是一的高斯分布。这个约束使得编码器能够产生高质量的编码。

作为生成器的译码器

假设有一个译码器

  • 它的任务是生成一张人能够识别的图片
  • 输入这个译码器的潜变量输入是独热码 (One hot)，比如输入 [1,0,0],[0,1,0],[0,0,1] 时这个译码器分别输出手马数字 0,1,2 。

这样我们就押有了一个手写数字图片的生成器。

然而由于独热编码，这个生成器只能最多生成三种图片类型，我们需要生成更多的图片类型，因此可以不使用独热码，而使用一个矢量作为潜变量，这个矢量里的每个数字都是随机实数。由于潜变量中的元素存在很多的组合，因此生成器可以生成很多类型的图片。

自编码器

当潜变量是随机数时，想要人为判别不同类型的输入是很困难的。因此我们使用一个编码器，这个编码器接受的输入是一张人可以识别的图片，输出的是元素是随机实数的潜变量。当我们把编码器和译码器一起使用时，就构成了一个标准的编码器。我们可以通过监督性学习，让编码器的输入和译码器的输出十分接近以至于人眼无法分辨出区别。这时我们就有了一个可以正常工作的自编码器。然而这个自编码器的功能是有限的，它不能生成新的图片，而只能输出和输入几乎一样的图片。

变分自编码器

对标准自编码器的潜变量做一些约束就可以使一个标准自编码器变成一个变分自编码器。这个约束就是潜变量里的随机实数服从宽度是一的高斯分布。为 了实现这个约束，我们在训练变分编码器时使用的损失函数包含两个部分： \[ \text { Loss }=\text { generation loss }+\text { latent loss } \] 其中

  • generation loss 是 译 码 器生成 的图 像和 输入的图像之前 的差异
  • generation loss = MSE(generated image, real image), MSE 是均方误差
  • latent loss 是潜变量和宽度是一的高斯分布之间的 KL 散度，又称为相对熵，它反映了潜变量服从的分布和高斯随机分布之间的距离。
