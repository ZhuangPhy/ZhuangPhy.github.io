入门

1 吴恩达《机器学习》

讲得非常细节，通俗易懂。但是内容涵盖的话题比较少，没有CNN，RNN等内容。主要内容：

  • 回归问题
    • 线性回归
    • 逻辑斯蒂回归
    • 多变量回归
  • 有中文版和英文版讲义，英文版的视频。
  • 作业有matlab和python代码

2 李沐《动手机器学习》

李沐老师作为MXNet的作者之一，教材自然也是使用MXNet框架写成的。但是很多学生使用的却是tensorflow和PyTorch。

  • MXNet框架
  • PyTorch版本 github
  • TensorFlow2版本
  • GLUON版本中文
