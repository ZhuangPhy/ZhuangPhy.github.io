1.1 Discrete Fourier transformation and Fourier series

Fourier series:

  • real space is discrete
  • continuous \(\mathrm{k}\) -space

typically, Fourier series is used to describe a periodic function in the real space, which will have a describe set of wave vectors in the \(\mathrm{k}\) space. Here, it is the opposite. We have a continuous \(\mathrm{k}\) -space and it is periodic (the Brillouin zone), but the real space is discrete. \[ \begin{aligned}&c_{k}=\frac{\sqrt{a}}{\sqrt{2 \pi}} \sum_{i} c_{i} e^{-i k x}\\&c_{i}=\frac{\sqrt{a}}{\sqrt{2 \pi}} \int_{\mathrm{BZ}} d k c_{k} e^{i k x}\end{aligned} \]

  • 对易性

Because \(\left\{c_{i}, c_{j}^{\dagger}\right\}=\delta_{\mathrm{ij}},\) it is easy to check that \[ \begin{aligned}\left\{c_{k}, c_{k}^{\dagger}\right\}&=\left\{\frac{1}{\sqrt{2 \pi / a}} \sum_{i} c_{i} e^{-i k x_{i}}, \frac{1}{\sqrt{2 \pi / a}} \sum_{j} c_{j}^{\dagger} e^{i k x_{j}}\right\} \\&=\sum_{i, j}\left\{c_{i}, c_{j}^{\dagger}\right\} \frac{1}{2 \pi / a} e^{-i k x_{i}} e^{i k x_{j}}=\sum_{i, j} \delta_{i, j} \frac{1}{2 \pi / a} e^{-i(k-k) x_{i}}\\&=\sum_{i} \frac{1}{2 \pi / a} e^{-i(k-k) i a}=a \delta\left(\left(k-k^{\prime}\right) a\right)=\delta\left(k-k^{\prime}\right)\end{aligned} \] If we know \(\left\{c_{k}, c_{k^{\dagger}}\right\}=\delta\left(k-k^{\prime}\right),\) we can also show that \(\left\{c_{i}, c_{j}^{\dagger}\right\}=\delta_{\mathrm{ij}}\) \[ \begin{aligned}\left\{c_{i}, c_{j}^{\dagger}\right\}&=\left\{\frac{1}{\sqrt{2 \pi / a}} \int_{\mathrm{BZ}} d k c_{k} e^{i k x_{i}}, \frac{1}{\sqrt{2 \pi / a}} \int_{\mathrm{B} \mathrm{Z}} d k^{\prime} c_{k}^{\dagger} e^{-i k x_{j}}\right\} \\&=\frac{1}{2 \pi / a} \int_{\mathrm{BZ}} d k \int_{\mathrm{BZ}} d k^{\prime}\left\{c_{k}, c_{k}^{\dagger}\right\} e^{i k x_{i}} e^{-i k x_{j}}\\&=\frac{1}{2 \pi / a} \int_{\mathrm{Bz}} d k \int_{\mathrm{BZ}} d k^{\prime} \delta\left(k-k^{\prime}\right) e^{i k x_{i}} e^{-i k x_{j}}\\&=\frac{1}{2 \pi / a} \int_{\mathrm{BZ}} d k e^{i k\left(x_{i}-x_{j}\right)}=\delta_{i j}\end{aligned} \]

Discrete Fourier transform.

Here, we briefly discuss about the discrete Fourier transform.

  1. Consider a discrete function \(f_{i}\), where \(i=1,2,3 \ldots N\) marks different lattice site.
  2. In addition, we assume the periodic boundary condition \(f_{N+i}=f_{i}\).

We can define \[ \tilde{f}_{k}=\frac{1}{\sqrt{N}} \sum_{j} f_{j} e^{-i k a j} \] where

  • \(a\) is the lattice constant
  • \(j\) marks different lattice sites

And it is easy to prove that \[ f_{j}=\frac{1}{\sqrt{N}} \sum_{k} \tilde{f}_{k} e^{i k a j} \] Same as \(j\), the wave vector \(k\) here is also a discrete variable.

compare the two equations \[ \begin{array}{l} f_{j+N}=\frac{1}{\sqrt{N}} \sum_{k} \tilde{f}_{k} e^{i k(a j+a N)}=\frac{1}{\sqrt{N}} \sum_{k} \tilde{f}_{k} e^{i k a j} e^{i k a N} \\ f_{j}=\frac{1}{\sqrt{N}} \sum_{k} \tilde{f}_{k} e^{i k a j} \end{array} \] because \(f_{N+j}=f_{N}\), we find that \(e^{i k a N}=1\), which implies \[ k a N=2\pi \]

\[ k=\frac{2 \pi }{N a}=\frac{2 \pi }{L} \]

Same as j, k also have a periodicity, and the periodicity is also \(N .\) This periodicity is just the Brillouin zone.

Define \(k_{m}=2 \pi m / L\) and by definition, \[ \tilde{f}_{k_{m}}=\frac{1}{\sqrt{N}} \sum_{j} f_{j} e^{-i k_{m} a j}=\frac{1}{\sqrt{N}} \sum_{j} f_{j} e^{-i\left(\frac{2 \pi}{N a} m\right) a j}=\frac{1}{\sqrt{N}} \sum_{j} f_{j} e^{-i \frac{2 \pi}{N} m j} \] For \(k_{m+N}\), by definition \[ \tilde{f}_{k_{m+N}}=\frac{1}{\sqrt{N}} \sum_{j} f_{j} e^{-i k_{m+N} a j}=\frac{1}{\sqrt{N}} \sum_{j} f_{j} e^{-i\left[\frac{2 \pi}{N a}(m+N)\right] a j}=\frac{1}{\sqrt{N}} \sum_{j} f_{j} e^{-i \frac{2 \pi}{N} m j} e^{-i 2 \pi j} \] Because \(j\) is an integer, the factor \(e^{-i 2 \pi j}=1\). Therefore \[ \tilde{f}_{k_{m+N}}=\frac{1}{\sqrt{N}} \sum_{j} f_{j} e^{-i \frac{2 \pi}{N} m j}=\tilde{f}_{k_{m}} \] As a result, we can limit the value of \(m\) to be \(-N / 2 \leq m<N / 2\). If \(m\) is not in this range, the value of \(\tilde{f}_{k_{m}}\) can be obtained using the periodic condition \(\tilde{f}_{k_{m+N}}=\tilde{f}_{k_{m}}\) \[ m=-\frac{N}{2},-\frac{N}{2}+1, \ldots \frac{N}{2}-1 \] For the wave vector \(k\), this means that \[ \tilde{f}_{k+2 \pi / a}=\tilde{f}_{k} \] So, we can confine the value of \(k\) into the range of \(-\pi / a \leq k<\pi / a\), which is the first Brillouin zone. For \(k\) outside the first Brillouin zone, we can find the corresponding \(\tilde{f}_{k}\) using the periodicity \(\tilde{f}_{k+2 \pi / a}=\tilde{f}_{k}\) \[ k=-\frac{\pi}{a},-\frac{\pi}{a}+1 \times \frac{2 \pi}{L},-\frac{\pi}{a}+2 \times \frac{2 \pi}{L}, \ldots \frac{\pi}{a}-\frac{2 \pi}{L}, \] Two useful Identities: \[ \begin{aligned} \frac{1}{N} \sum_{k} e^{i k a j} e^{-i k a j^{\prime}}=\delta_{j, j^{\prime}} \\ \frac{1}{N} \sum_{j} e^{i k a j} e^{-i k^{\prime} a j}=\delta_{k, k^{\prime}} \end{aligned} \]

1.2 Example 1: a one-band model

Lets consider a 1D lattice with one atom per unit cell. For each atom, we consider only one quantum state. The creation (annihilation) operator \(i^{\dagger}\left(c_{i}\right)\) creates and annihilates one particle on site \(i .\) The non-interacting Hamiltonian can be written as \[ H=-\sum_{i j}\left(t_{i j} c_{i}^{\dagger} c_{j}+t_{j i} c_{j}^{\dagger} c_{i}\right)+\sum_{i} V_{i} c_{i}^{\dagger} c_{i} \]

  • The first term \(\sum_{i j} t_{i j} c_{i}^{\dagger} c_{j}\) describes hoppings from site \(j\) to i.
  • The second term \(\sum_{i j} t_{j i} c_{j}^{\dagger} c_{i}\) describes the hopping from i to \(j\).
  • The last term is the potential energy, which tells us how much energy we need to put an electron on each site \(V_{i}\).

Because the lattice contains only one type of atoms, \(V_{i}=\) constant (i.e. translational symmetry).

Because \(H\) is Hermitian, we find that \(t_{i j}=t_{i i^{*}}\) and \(V_{i}\) is a real number.

Proof: Hermitian

\[ \begin{aligned} H^{\dagger} &=\left[-\sum_{i j}\left(t_{i j} c_{i}^{\dagger} c_{j}+t_{j i} c_{j}^{\dagger} c_{i}\right)+\sum_{i} V_{i} c_{i}^{\dagger} c_{i}\right]^{\dagger} \\ &=-\sum_{i j}\left(t_{i j}^{*} c_{j}^{\dagger} c_{i}+t_{j i}^{*} c_{i}^{\dagger} c_{j}\right)+\sum_{i} V_{i}^{*} c_{i}^{\dagger} c_{i} \\&=-\sum_{i j}\left(t_{i j} c_{i}^{\dagger} c_{j}+t_{j i} c_{j}^{\dagger} c_{i}\right)+\sum_{i} V_{i} c_{i}^{\dagger} c_{i} \\ &=H \end{aligned} \]

By comparing the two sides, we find that \(t_{\mathrm{ij}}^{*}=t_{\mathrm{ji}}\) and \(V_{i}=V_{i}^{*}\) Therefore, we can simplify the Hamiltonian \[ \begin{aligned} H&=-\sum_{\mathrm{ij}}\left(t_{\mathrm{ij}} c_{i}^{\dagger} c_{j}+t_{\mathrm{ij}}^{*} c_{j}^{\dagger} c_{i}\right)+V \sum_{i} c_{i}^{\dagger} c_{i} \\&=-\sum_{\mathrm{ij}}\left(t_{\mathrm{ij}} c_{i}^{\dagger} c_{j}+t_{\mathrm{ij}}^{*} c_{j}^{\dagger} c_{i}\right)+V N \end{aligned} \]

In the last term, \(N=\Sigma_{i} c_{i}^{\dagger} c_{i}\) is the total number of electrons in the system. Because \(V N\) is a constant, this term just shifts the total energy by a constant, and thus has no other physical contribution (can be ignored if we are not interested in the total energy).

nearest hopping

For tight-binding models, a typically approximation is to assume that electrons can only hop to its nearest-neighbor sites. In reality, long-range hopping is allowed, but their amplitudes are small (decay exponentially as distance increases). Therefore, in many cases, we just need the nearest-neighbor hopping terms to describe the systems. Due to the translational symmetry, all nearest-neighbor hoppings shall have the same hopping strength. If we assume that this hopping strength is a real number \(t,\) the Hamiltonian is \[ H=-t \sum_{\langle ij \rangle}\left(c_{i}^{\dagger} c_{j}+c_{j}^{\dagger} c_{i}\right)=-t \sum_{i} c_{i}^{\dagger} c_{i+1}+h . c . \] Here,\(\langle i j\rangle\) implies that i and j must be neighbors. On the right hand side, h.c. means Hermitian conjugate. Notice that the second term is in fact the Hermitian conjugate of the first term, so we just use h.c. to represent it.

Hamiltonian in k-space

We can transfer the Hamiltonian into the k-space. In real space, it looks like \[ H=-t \sum_{i} c_{i}^{\dagger} c_{i+1}+h . c . \] The first term is \[ \begin{aligned} \sum_{j} c_{j}^{\dagger} c_{j+1}&=\sum_{j} \frac{1}{\sqrt{2 \pi / a}} \int_{\mathrm{B} z} d k c_{k}^{\dagger} e^{-i k a j} \frac{1}{\sqrt{2 \pi / a}} \int_{\mathrm{B} \mathrm{Z}} d k^{\prime} c_{k} e^{i k^{\prime}(j+1) a} \\ &=\sum_{j} \frac{1}{2 \pi / a} \int_{\mathrm{BZ}} d k \int_{\mathrm{BZ}} d k^{\prime} c_{k}^{\dagger} c_{k k} e^{i k a} e^{-i(k-k) a j}\\ &=\int_{\mathrm{BZ}} d k \int_{\mathrm{BZ}} d k^{\prime} c_{k}^{\dagger} c_{k} \cdot e^{i k^{\prime} a} \delta\left(k-k^{\prime}\right)\\ &=\int_{\mathrm{BZ}} d k c_{k}^{\dagger} c_{k} e^{i k a} \end{aligned} \] The second term is the Hermitian conjugate of the first term, so \[ h . c .=\left(\sum_{j} c_{j}^{\dagger} c_{j+1}\right)^{\dagger}=\left(\int_{\mathrm{BZ}} d k c_{k}^{\dagger} c_{k} e^{i k a}\right)^{\dagger}=\int_{\mathrm{BZ}} d k c_{k}^{\dagger} c_{k} e^{-i k a} \] Hamiltonian in k-space \[ \begin{aligned} H&=-t \sum_{i} c_{i}^{\dagger} c_{i+1}+h . c . \\ &=-t \int_{\mathrm{BZ}} d k c_{k}^{\dagger} c_{k} e^{i k a}-t \int_{\mathrm{BZ}} d k c_{k}^{\dagger} c_{k} e^{-i k a} \\ &=-2 t \int_{\mathrm{BZ}} d k c_{k}^{\dagger} c_{k} \cos k a \\ &=\int_{\mathrm{BZ}} d k(-2 t \cos k a) c_{k}^{\dagger} c_{k} \end{aligned} \] For a solid, we know that the total energy of electrons (ignore interactions) is \[ E=\sum_{n} \int_{\mathrm{BZ}} d k \epsilon_{n}(k) n_{n}(k) \] where \(\Sigma_{n}\) sums over all bands, \(\epsilon_{n}(k)\) is the dispersion relation for band \(n\) and \(n_{n}(k)\) is the occupation number for the Bloch wave state in band \(n\) with momentum \(k\). In second quantization, this formula implies that the Hamiltonian is \[ H=\sum_{n} \int_{\mathrm{Bz}} d k \epsilon_{n}(k) \gamma_{n k}^{\dagger} \gamma_{n k} \] Here, \(\gamma_{n, k}^{\dagger}\) is the creation operator for a Bloch wave \(\psi_{n k}(r)=u_{n k}(r) e^{i k r}\). If we compare this formula with the tight-binding Hamiltonian (in \(\mathrm{k}-\)space), we find immediately that the tight-binding model we considered here has only one energy band. And our \(c_{k}^{\dagger}\) operator is in fact the creation operator for Bloch waves. And the dispersion relation for this band is \(\epsilon_{k}=-2 t \cos k a .\) Notice that \(\epsilon_{k}\) is a periodic function of \(k\) with periodicity \(2 \pi / a,\) which is exactly what we expect for Bloch waves.

1.3 Example 2: a two-band model in 1D

Now, let us consider a slightly more complicated situation, i.e. a \(1 d\) chain formed by two different types of atoms \((a\) and \(b)\). \[ H=-t \sum_{i}\left(a_{i}^{\dagger} b_{i}+b_{i}^{\dagger} a_{i+1}+\text { h.c. }\right)+V_{a} \sum_{i} a_{i}^{\dagger} a_{i}+V_{b} \sum_{i} b_{i}^{\dagger} b_{i} \] Here, the position of \(a\) sites in the jth unit cells is \[ r=a \times j+r_{a} \] and the position of \(b\) sites in the \(j\) th unit cells is \[ r=a \times j+r_{b} \] where, \(a\) is the lattice constant (the size of a unit cell)

Discrete Fourier transform

(as discussed in the previous section) \[ \begin{aligned} a_{k}&=\frac{1}{\sqrt{N}} \sum_{i} a_{i} e^{i k x} \\ a_{i}&=\frac{1}{\sqrt{N}} \sum_{k} a_{k} e^{-i k x} \\ b_{k}&=\frac{1}{\sqrt{N}} \sum_{i} a_{i} e^{i k x} \\ b_{i}&=\frac{1}{\sqrt{N}} \sum_{k} a_{k} e^{-i k x} \end{aligned} \] Therefore, we know that \[ \begin{aligned} \sum_{j} a_{j}^{\dagger} b_{j}=&\sum_{j} \frac{1}{\sqrt{N}} \sum_{k} a_{k}^{\dagger} e^{-i k\left(a j+r_{a}\right)} \frac{1}{\sqrt{N}} \sum_{k^{\prime}} b_{k^{\prime}} e^{i k^{\prime}\left(a j+r_{b}\right)}\\=&\frac{1}{N} \sum_{k} \sum_{k^{\prime}} a_{k}^{\dagger} b_{k^{\prime}} \sum_{j} e^{-i k\left(a j+r_{a}\right)} e^{i k^{\prime}\left(a j+r_{b}\right)} \\=& \sum_{k} \sum_{k^{\prime}} a_{k}^{\dagger} b_{k^{\prime}} \frac{1}{N} \sum_{j} e^{-i\left(k-k^{\prime}\right) a j} e^{-i k r_{a}} e^{i k^{\prime} r_{b}} \\=&\sum_{k} \sum_{k^{\prime}} a_{k}^{\dagger} b_{k^{\prime}} \delta_{k, k^{\prime}} e^{-i k r_{a}} e^{i k^{\prime} r_{b}} \\=&\sum_{k} a_{k}^{\dagger} b_{k} e^{i k\left(r_{b}-r_{a}\right)} \end{aligned} \] This conclusion is generic. In k-space, hoppings from \(b\) to a result in a term in Hamiltonian \(\sim \sum_{k} a _{ k }{ }^{\dagger} b _{ k } e^{ i i k\left(r_{ b }- r _{ a }\right)}\). The phase factor here is determined by how far the electron hop, i.e. \(r_{b}-r_{a}\). If we apply the same conclusion to the hopping from a to b, we find immediately that \[ \sum_{j} b_{j}^{\dagger} a_{j+1}=\sum_{k} b_{k}^{\dagger} a_{k} e^{i k\left(r_{a}-r_{b}\right)} \] If \(r_{b}-r_{a}=\frac{a}{2}\),

  • \(\sum_{j} a_{j}^{\dagger} b_{j}=\sum_{k} a_{k}^{\dagger} b_{k} e^{i k a / 2}\)
  • \(\sum_{j} b_{j}^{\dagger} a_{j+1}=\sum_{k} b_{k}^{\dagger} a_{k} e^{i k a / 2}\)

For the potential term, we also know that \[ \sum_{i} a_{i}^{\dagger} a_{i}=\sum_{k} a_{k}^{\dagger} a_{k} \]

\[ \sum_{i} b_{i}^{\dagger} b_{i}=\sum_{k} b_{k}^{\dagger} b_{k} \]

Therefore, in the k-space, the Hamiltonian looks like \[ \begin{aligned} H=&-t \sum_{k}\left(a_{k}^{\dagger} b_{k} e^{i k a / 2}+b_{k}^{\dagger} a_{k} e^{i k a / 2}+b_{k}^{\dagger} a_{k} e^{-i k a / 2}+a_{k}^{\dagger} b_{k} e^{-i k a / 2}\right)\\ &+V_{a} \sum_{k} a_{k}^{\dagger} a_{k}+V_{b} \sum_{k} b_{k}^{\dagger} b_{k}\\ =&-2 t \sum_{k}\left[\left(a_{k}^{\dagger} b_{k} \cos \left(\frac{k a}{2}\right)+b_{k}^{\dagger} a_{k} \cos \left(\frac{k a}{2}\right)\right)+V_{a} a_{k}^{\dagger} a_{k}+V_{b} b_{k}^{\dagger} b_{k}\right] \end{aligned} \]

2 Eigen-problem & Energy Band

we can write two band model in a matrix form: \[ H=\sum_{k}\left(a_{k}^{\dagger}, b_{k}^{\dagger}\right)\left(\begin{array}{cc} V_{a} & -2 t \cos \left(\frac{k a}{2}\right) \\ -2 t \cos \left(\frac{k a}{2}\right) & V_{b} \end{array}\right)\left(\begin{array}{c} a_{k} \\ b_{k} \end{array}\right) \] For any tight-binding models with two quantum states per unit cell, the Hamiltonian can be written in terms of a two-by-two matrix in the k space: \[ H=\sum_{k}\left(a_{k}^{\dagger}, b_{k}^{\dagger}\right) \mathcal{H}(k)\left(\begin{array}{l} a_{k} \\ b_{k} \end{array}\right) \] where

  • \(\mathcal{H}\) is a \(2 \times 2\) Hermitian matrix as a function of \(k\).

It is called the kernel of the Hamiltonian. Because \(\mathcal{H}(k)\) contains all the information of the Hamiltonian \(H,\) it is often called the Hamiltonian in literature. However, it is important to keep in mind that \(\mathcal{H}(k)\) is only part of the Hamiltonian.

The Hamiltonian \(H\) must be gauge invariant, but \(\mathcal{H}(k)\) is not.

For example, if we change \(a^{\dagger} \rightarrow a^{\dagger} e^{i \phi}\), the Hamiltonian \(H\) is invariant, but the kernel \(\mathcal{H}(k)\) is NOT.

For more generic cases, if one have \(m\) quantum states per unit cell, \(\mathcal{H}(\mathbf{k})\) will be a \(m \times m\) Hermitian matrix.

When we have a matrix, we know what to do. We need to find the eigenvalue and eigenfunctions of the Hamiltonian, i.e. diagonalizing the matrix. The next two sections discuss the physical meanings of the eigenvalues and eigenvectors of \(\mathcal{H}(k)\)

Eigenvalues

The eigenvalues of \(\mathcal{H}(k)\) give us the dispersion relations.

At each \(\mathrm{k}\) point, one can define a unitary transformation

\[ \left(\begin{array}{l} c_{k} \\ d_{k} \end{array}\right)=U_{k}^{-1}\left(\begin{array}{l} a_{k} \\ b_{k} \end{array}\right) \]

\[ \left(\begin{array}{ll}c_{k}^{\dagger} & d_{k}^{\dagger}\end{array}\right)=\left(\begin{array}{ll}a_{k}^{\dagger} & b_{k}^{\dagger}\end{array}\right) U_{k} \]

where \(U_{k}^{-1}=U_{k}^{\dagger}\) \[ H=\sum_{k}\left(c_{k}^{\dagger}, d_{k}^{\dagger}\right) U_{k}^{+} \mathcal{H} U_{k}\left(\begin{array}{c} c_{k} \\ d_{k} \end{array}\right) \] If we choose \(U_{k}\) such that \(U_{\mathrm{k}}^{+} \mathcal{H} \mathrm{U}_{\mathrm{k}}\) is a diagonal matrix, \[ U_{k} \mathcal{H} U_{k}^{-1}=\left(\begin{array}{cc} \epsilon_{c} & 0 \\ 0 & \epsilon_{d} \end{array}\right) \] Then, the Hamiltonian becomes \[ \begin{aligned} H&=\sum_{k}\left(c_{k}^{\dagger}, d_{k}^{\dagger}\right)\left(\begin{array}{cc} \epsilon_{c} & 0 \\ 0 & \epsilon_{d} \end{array}\right)\left(\begin{array}{l} c_{k} \\ d_{k} \end{array}\right) \\&=\sum_{k} \epsilon_{c}(k) c_{k}^{\dagger} c_{k}+\sum_{k} \epsilon_{d}(k) d_{k}^{\dagger} d_{k} \end{aligned} \] We know that using Bloch waves (which are eigenstates of the Hamiltonian), the total energy is \[ \begin{aligned} E&=\sum_{k, m} \epsilon_{m}(k) n_{m}(k) \\&=\sum_{k} \epsilon_{1} n_{1}(k)+\sum_{k} \epsilon_{2} n_{2}(k)+\sum_{k} \epsilon_{3} n_{3}(k)+\ldots \end{aligned} \] where

  • \(\epsilon_{m}(k)\) is the dispersion relation for the m-th band
  • \(n_{m}(k)\) is the occupation number for the quantum state in the m-th band with momentum \(k\).

Therefore, in terms of Bloch waves, the Hamiltonian should take the form: \[ H=\sum_{k} \epsilon_{1}(k) \gamma_{1, k}^{\dagger} \gamma_{1, k}+\sum_{k} \epsilon_{2}(k) \gamma_{2, k}^{\dagger} \gamma_{2, k}+\sum_{k} \epsilon_{3}(k) \gamma_{3, k}^{\dagger} \gamma_{3, k}+\ldots \] where \(\gamma_{n, k^{\top}}\) is the creation operator for a Bloch wave in band \(\mathrm{n}\) at momentum \(k\).

therefore we can interpreter the Hamiltonian \[ \begin{aligned} H=\sum_{k} \epsilon_{c}(k) c_{k}^{\dagger} c_{k}+\sum_{k} \epsilon_{d}(k) d_{k}^{\dagger} d_{k} \end{aligned} \]

  • \(c_{k}^{\dagger}\) creates a Bloch wave with momentum \(k\) in one of the bands
  • \(d_{k}^{\dagger}\) creates a Bloch wave with momentum \(k\) in the other band
  • For the bands created by \(c_{k}^{\dagger}\), its dispersion relation is \(\epsilon_{c}(k)\)

For two-band model in 1D, the dispersions are \[ \begin{aligned} &\epsilon_{c}=\frac{V_{a}+V_{b}}{2}-\sqrt{\left[2 t \cos \left(\frac{k a}{2}\right)\right]^{2}+\left(\frac{V_{a}-V_{b}}{2}\right)^{2}}\\ &\epsilon_{d}=\frac{V_{a}+V_{b}}{2}+\sqrt{\left[2 t \cos \left(\frac{k a}{2}\right)\right]^{2}+\left(\frac{V_{a}-V_{b}}{2}\right)^{2}} \end{aligned} \] which are the eigenvalues of the matrix \(\mathcal{H}(k)\)

If we have a model with m orbitals per unit cell, \(\mathcal{H}(k)\) would be a \(m \times m\) matrix. This matrix will have \(m\) eigenvalues, which results in a m-band model and each eigenvalue \(\epsilon_{n}(k)\) is the dispersion relation for one of the bands.

Eigenvectors

The eigenvectors \(\mathcal{H}(k)\) gives us the Bloch waves.

Let's start from the two band model discussed above as an example. Here, the two-by-two matrix \(\mathcal{H}(k)\) has two eigenvalues, \(\epsilon_{c}(k)\) and \(\epsilon_{d}(k)\). For each eigenvalue, the corresponding eignevector is a 2-component vector: \[ \begin{aligned} &\mathcal{H}\left(\begin{array}{c} v^{(-)} a(k) \\ v^{(-)}_{b}(k) \end{array}\right)=\epsilon_{c}\left(\begin{array}{c} v^{(-)} a(k) \\ v^{(-)}_{b}(k) \end{array}\right)\\ &\mathcal{H}\left(\begin{array}{c} v^{(+)} a(k) \\ v^{(+)}_{b}(k) \end{array}\right)=\epsilon_{d}\left(\begin{array}{c} v^{(+)} a(k) \\ v^{(+)}_{b}(k) \end{array}\right) \end{aligned} \] It can be easily checked that（其实就是一个旋转） \[ \begin{aligned} &c_{k}^{\dagger}=v^{(-)}_{a}(k) a_{k}^{\dagger}+v_{b}^{(-)}(k) b_{k}^{\dagger}\\ &d_{k}^{\dagger}=v^{(+)}_{a}(k) a_{k}^{\dagger}+v^{(+)}_{b}(k) b_{k}^{\dagger} \end{aligned} \] We know that

  • \(a^{\dagger}\) and \(b^{\dagger}\) are the creation operators for plane waves (because they come from Fourier transformations)
  • \(c_{k}^{\dagger}\) and \(d_{k}^{\dagger}\) create Bloch waves.

The relation between plane waves and Bloch waves is \[ \psi_{n, k}(x)=u_{n, k}(x) \frac{e^{i k x}}{\sqrt{N}} \] So, the coefficients \(v^{(\pm)}_{a}\) and \(v^{(\pm)}_{b}(k)\) here are in fact the discrete version of \(u_{n, k}(x)\)

Bottom line: eigenvectors of \(H ( k )\) tell us \(u _{n, k} .\)

Berry connection

We have learned that the Berry connection \(\mathcal{H}_{n}(k)\) is defined as \[ \overrightarrow{\mathcal{A}}_{n}=-i\left\langle u_{n, k}\left|\nabla_{-}\right| u_{n, k}\right\rangle \] For continuous models, this formula means \[ \overrightarrow{\mathcal{A}}_{n}=-i \int d x\left[u_{n, k}(x)\right]^{*} \nabla_{-} u_{n, k}(x) \] For lattice models (tight-binding models) \[ \begin{array}{l} \overrightarrow{\mathcal{A}}_{-}=-i\left(v^{(-)}_{a}(k)^{*} \quad v^{(-)}_{b}(k)^{*}\right) \nabla_{-}\left(\begin{array}{c} v^{(-)} a(k) \\ v^{(-)}_{b}(k) \end{array}\right) \\ \overrightarrow{\mathcal{A}}_{+}=-i\left(v^{(+)}_{a}(k)^{*} \quad v^{(+)}_{b}(k)^{*}\right) \nabla_{-}\left(\begin{array}{c} v^{(+)} a(k) \\ v^{(+)}_{b}(k) \end{array}\right) \end{array} \] In general, for a \(m\) band model, where \(\mathcal{H}(k)\) is a \(m \times m\) matrix, each of the eigenvector gives us the Bloch wavefunction for one of the \(m\) bands. We can use each eigenvector to compute the Berry connection, the Berry curvature and the Chern number for that band

Bottom line: the eigenvectors are Bloch wave functions.

Summary

  • The key properties of a tight-binding model is coded in \(H _{k}\), which is a \(m \times m\) Hermitian matrix. Each component is a function of \(k\). (In other words, \(H _{k}\) is a matrix function of \(k\).)

  • The eigenvalues if \(H\) (as a function of \(k\) ) gives the band structure (the dispersion relation) for each bands \(\epsilon_{n}(k)\) with \(n=1,2, \ldots m\)

  • The eigenvectors as a function of \(k\) gives the Bloch wave: \(u_{n, k}(\alpha)\), where \(n =1,2, \ldots, m\) is the band index and \(\alpha=1,2 \ldots m\) marks different site in a unit cell.

  • Using these \(u_{n, k}(\alpha)\), we can define the Berry connection, Berry curvature and the Chern number. Just replace the integral in the real space by summing over \(\alpha=1 \ldots m\)

Before 80s, people stop at step #2 (computing the eigenvalues), without calculating the eigenvectors. This is because eigenvectors are the wavefunction, which cannot be measured directly in experiments. After the discovery of topological insulators, physicists realized that although the wavefunction can not be measured directly, it contains the topological information, which is a physical observable (e.g. the Hall conductivity).

In a one-band model, the eigenvector is trivial, which is just identity. Therefore, the Berry connection \(\overrightarrow{ A }_{n}=-i \int d x\left[u_{n, k}(x)\right]^{*} \nabla_{\vec{k}} u_{n, k}(x)\) must be trivial \(\overrightarrow{ A }_{n}=0\). This tells us that a one-band model can never show quantum Hall effect. We need at least two bands. In fact, one can prove that the total Chern number summing over all bands must be zero.

3 topologically nontrivial insulator

In a one-band model, the eigenvector is trivial, which is just identity. Therefore, the Berry connection \(\overrightarrow{\mathcal{A}}_{n}=-i \int d x\left[u_{n, k}(x)\right]^{*} \nabla_{-} u_{n, k}(x)\) must be trivial \(\overrightarrow{\mathcal{A}}_{n}=0 .\) This tells us that a one-band model can never show quantum Hall effect. We need at least two bands. In fact, one can prove that the total Chern number summing over all bands must be zero.

Let's consider a 2-band model, whose kernel i \[ \mathcal{H}(k)=\left(\begin{array}{cc} -2 t \cos k_{x}-2 t \cos k_{y}-\mu & \Delta\left(\sin k_{x}-i \sin k_{y}\right) \\ \Delta\left(\sin k_{x}+i \sin k_{y}\right) & 2 t \cos k_{x}+2 t \cos k_{y}+\mu \end{array}\right) \] Here, we assume \(\Delta>0\) and \(t>0\) and \(-4|t|<\mu<4\) | \(|t|\). In real space, this Hamiltonian corresponds to a square lattice and on each site there are two quantum states. Please notice that here I use \(\mu\) to refer to a control parameter, which is NOT the chemical potential.

3.1 Two-band models and Pauli matrices

For any two-band models, \(\mathcal{H}\) is a two-by-two Hermitian matrix. For a two-by-two Hermitian matrix, one can always separate it into the identity and Pauli matrices \(I,\) and \(\sigma_{i}\) \[ \mathcal{H}=\mathcal{H}_{0}(k) I+\mathcal{H}_{x}(k) \sigma_{x}+\mathcal{H}_{y}(k) \sigma_{y}+\mathcal{H}_{z}(k) \sigma_{z} \] where

  • \(\mathcal{H}_{0}(k), \mathcal{H}_{x}(k), \mathcal{H}_{y}(k)\) and \(\mathcal{H}_{z}(k)\) are real functions of \(k\).

If we define \(\overrightarrow{\mathcal{H}}(k)=\left(\mathcal{H}_{x}(k), \mathcal{H}_{y}(k), \mathcal{H}_{z}(k)\right),\) we find that \[ \mathcal{H}=\mathcal{H}_{0}(k) I+\overrightarrow{\mathcal{H}}(k) \cdot \vec{\sigma} \] This Hamiltonian is rather similar to a spin \(S=1 / 2\) under magnetic fields. \[ H=\mathrm{constant}+\mu \vec{B} \cdot \vec{\sigma} \] For the case studied here \[ \begin{aligned} &\mathcal{H}_{0}(k)=0\\ &\mathcal{H}_{x}(k)=\Delta \sin \left(k_{x}\right)\\ &\mathcal{H}_{y}(k)=\Delta \sin \left(k_{y}\right)\\ &\mathcal{H}_{z}(k)=-2 t \cos k_{x}-2 t \cos k_{y}-\mu \end{aligned} \]

eigen-energy

The eigenvalues of \(\mathcal{H}\) are \[ E_{\pm}=\mathcal{H}_{0}(k) \pm|\overrightarrow{\mathcal{H}}(k)|=\mathcal{H}_{0}(k) \pm \sqrt{\mathcal{H}_{x}(k)^{2}+\mathcal{H}_{y}(k)^{2}+\mathcal{H}_{z}(k)^{2}} \] This is pretty much the same as the spin system, where the eigen-energies of the are \[ E=\text { constant } \pm \mu|\vec{B}| \] For a two band model, the top band has energy \(E_{+} \geq \mathcal{H}_{0}\), while the lower band has \(E_{-} \leq \mathcal{H}_{0}\). The energy gap between the two bands is: \[ \Delta(k)=E_{+}(k)-E_{-}(k)=2|\overrightarrow{\mathcal{H}}(k)| \] As long as \(\overrightarrow{\mathcal{H}}\) ( \(\mathrm{k}\) ) \(\neq 0,\) the two band will not cross with each other.

eigen-vector

Let's now focus on the lower band \(\left(E_{-}\right)\), its eigenvector is \[ u^{(I)}_-(k)=\frac{1}{\mathcal{N}^{(l)}}\left(\begin{array}{I} \mathcal{H}_{z}(k)-|\overrightarrow{\mathcal{H}}(k)| \\ \mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k) \end{array}\right) \] This wavefunction is singular if \(\mathcal{H}_{x}(k)=\mathcal{H}_{y}(k)=0\) and \(\mathcal{H}_{z}>0\) \[ \begin{aligned} u^{(I)}_-(k)&=\frac{1}{N^{(I)}}\left(\begin{array}{c} \mathcal{H}_{z}-|\overrightarrow{\mathcal{H}}(k)| \\ \mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k) \end{array}\right) \\&=\frac{1}{\mathcal{N}^{(I)}}\left(\begin{array}{c} \mathcal{H}_{z}-\sqrt{\mathcal{H}_{z}(k)^{2}} \\ 0 \end{array}\right) \\&=\frac{1}{\mathcal{N}^{(I)}}\left(\begin{array}{c} \mathcal{H}_{z}-\left|\mathcal{H}_{z}(k)\right| \\ 0 \end{array}\right) \\&=\frac{1}{\mathcal{N}^{(I)}}\left(\begin{array}{l} 0 \\ 0 \end{array}\right) \end{aligned} \] This is indeed a problem for the Hamiltonian shown above. At \(k_{x}=k_{y}=\pi\) \[ \begin{array}{l} \mathcal{H}_{0}(k)=0\\ \mathcal{H}_{x}(k)=\Delta \sin \left(k_{x}\right)=0 \\ \mathcal{H}_{y}(k)=\Delta \sin \left(k_{y}\right)=0 \\ \mathcal{H}_{z}(k)=-2 t \cos k_{x}-2 t \cos k_{y}-\mu=4 t-\mu>0 \end{array} \] Remember that we assumed \(|\mu|<4 t .\)

another way to eigenvector

In fact, there is another way to write down the same eigenvector (i.e. a phase shift) \[ \begin{aligned} u^{(\mathrm{II})}_{-}(k)&=\frac{1}{\mathcal{N}^{(\mathrm{II})}}\left(\begin{array}{c} \mathcal{H}_{z}-|\overrightarrow{\mathcal{H}}(k)| \\ \mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k) \end{array}\right) \times \frac{\frac{\mathcal{H}_{t}+\vec{H}(k)}{\mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k)}}{\left|\frac{\mathcal{H}_{t}+\vec{H}(k)}{\mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k)}\right|} \\ &=\frac{1}{\mathcal{N}^{(\mathrm{II})}}\left(\begin{array}{c} \frac{-\mathcal{H}_{x}\left(k^{2}-\mathcal{H}_{y}(k)^{2}\right.}{\mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k)} \\ \mathcal{H}_{z}+|\overrightarrow{\mathcal{H}}(k)| \end{array}\right) \\ &=\frac{1}{\mathcal{N}^{(\mathrm{II})}}\left(\begin{array}{c} -\mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k) \\ \mathcal{H}_{z}+|\overrightarrow{\mathcal{H}}(k)| \end{array}\right) \end{aligned} \] The two wavefunctions \(u^{(I)}_{-}(k)\) and \(u^{(\mathrm{II})}_{-}(k)\) differ by a phase \[ u^{(\mathrm{II})}_{-}(k)=u^{(I)}_{-}(k) e^{i \phi(k)} \] where \[ e^{i \phi(k)}=\frac{\frac{\mathcal{H}_{z}+|\overrightarrow{\mathcal{H}}(k)|}{\mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k)}}{\left|\frac{\mathcal{H}_{z}+|\overrightarrow{\mathcal{H}}(k)|}{\mathcal{H}_{x}(k)+\dot{i} \mathcal{H}_{y}(k)}\right|} \]

  • This new wavefunction is well defined at \(\mathcal{H}_{x}(k)=\mathcal{H}_{y}(k)=0\) and \(\mathcal{H}_{z}(k)>0\).
  • However, it is NOT well defined at \(\mathcal{H}_{x}(k)=\mathcal{H}_{y}(k)=0\) and \(\mathcal{H}_{z}(k)<0 .\)

There, we find \[ \begin{aligned} u^{(\mathrm{II})}-(k)&=\frac{1}{\mathcal{N}^{(\mathrm{II})}}\left(\begin{array}{c} -\mathcal{H}_{\mathrm{x}}(k)+i \mathcal{H}_{y}(k) \\ \mathcal{H}_{z}(k)+|\overrightarrow{\mathcal{H}}(k)| \end{array}\right) \\&=\frac{1}{\mathcal{N}^{\mathrm{II}}}\left(\begin{array}{c} 0 \\ \mathcal{H}_{z}(k)+\sqrt{\mathcal{H}_{z}(k)^{2}} \end{array}\right) \\&=\frac{1}{\mathcal{N}^{\mathrm{II}}}\left(\begin{array}{c} 0 \\ \mathcal{H}_{z}(k)+\left|\mathcal{H}_{z}(k)\right| \end{array}\right) \\&=\frac{1}{\mathcal{N}^{\mathrm{II}}}\left(\begin{array}{c} 0 \\ \mathcal{H}_{z}(k)-\mathcal{H}_{z}(k) \end{array}\right) \\&=0 \end{aligned} \] For the Hamiltonian shown above, the origin is such a point. \(k_{x}=k_{y}=0\)

cut two

Therefore, we need to cut the \(\mathrm{BZ}\) into two areas and use two different wave functions to describe the Bloch waves. They are connected by a gauge transformation \[ \begin{aligned} &\mathcal{H}_{0}(k)=0\\ &\mathcal{H}_{x}(k)=\Delta \sin \left(k_{x}\right)=0\\ &\mathcal{H}_{y}(k)=\Delta \sin \left(k_{y}\right)=0\\ &\mathcal{H}_{z}(k)=-2 t \cos k_{x}-2 t \cos k_{y}-\mu=-4 t-\mu<0 \end{aligned} \] This is in strong analogy to the magnetic monopole case, where we need to different gauge fields. \[ \begin{array}{l} u^{(\mathrm{II})}_{-}(k)=u^{(I)}_{-}(k) e^{i \phi(k)} \\ \mathcal{A}_{-}^{(\mathrm{II})}(k)=\mathcal{A}_{-}^{(I)}(k)+\nabla_{k} \phi(k) \end{array} \] This is in strong analogy to the magnetic monopole case, where we need to different gauge fields

Some comments

  • These singularities are NOT physical. If one measures any physical observables, there is no singularity anywhere in the momentum space. However. for the wavefunction and the Berry connection, which are not physical observables, there is always some singularity for this Hamiltonian.
  • The location of these singularity points depends on the gauge (phase) choice. In other words, the location of the singularities has no physical meaning either.
  • Only one thing about these singularities is physical, its existence. There must be some singularity points. This statement is independent of gauge choice and it tells us that the topological index is nonzero.

Let's compute the topological index for this model. For these model, there are four special points which satisfy \(\mathcal{H}_{x}=\mathcal{H}_{y}=0 .\) They are \[ \vec{k}=(0,0), \vec{k}=(\pi, \pi), \vec{k}=(0, \pi) , \vec{k}=(\pi, 0) \] At these four points, the values of \(\mathcal{H}_{z}\) are: \(\mathcal{H}_{z}=-4 t-\mu, 4 t-\mu,-\mu\) and \(-\mu\) respectively.

3.2 case study

case l: \(\mu<-4 t\)

If \(\mu<-4 t,\) all the four special points has \(\mathcal{H}_{z}>0 .\) So we can use \(u^{(\mathrm{II})}_{-}(k)\) for the whole Brillouin zone, and there is no singularity points. \[ u^{(\mathrm{II})}_{-}(k)=\frac{1}{\mathcal{N}^{(I)}}\left(\begin{array}{c} \mathcal{H}_{z}(k)-|\overrightarrow{\mathcal{H}}(k)| \\ \mathcal{H}_{x}(k)+\dot{i} \mathcal{H}_{y}(k) \end{array}\right) \] Then we can get Berry connection \[ \mathcal{A}_{-}^{(\mathrm{II})}=-i\left\langle u^{(\mathrm{II})}_{-}(k)\left|\partial_{k}\right|\left(u^{(\mathrm{II})}\right)_{-}(k)\right\rangle \] The Berry curvature for the lower band is \[ \Omega_{-}=\nabla \times \mathcal{A} \] Thus. the total Berry curvature ( \(2 \pi\) times the Chern number) is \[ \oiint_{\mathrm{BZ}} d k \Omega_{-}=\oiint_{\mathrm{BZ}} d k \nabla \times \mathcal{A}_{-}=\oint_{\partial \mathrm{BZ}} d k \mathcal{A}_{-}=0 \] So, the Chern number is zero \[ C=\frac{1}{2 \pi} \oiint_{\mathrm{BZ}} d k \Omega_{-}=0 \]

Marginal case: \(\mu=-4 t\)

\(\mathcal{H}_{x}(k)=\mathcal{H}_{y}(k)=\mathcal{H}_{z}(k)=0\) at \(k=(\pi, \pi)\) Because the gap between the two bands is \[ \Delta(k)=E_{+}-E_{-}=2|\overrightarrow{\mathcal{H}}(k)| \] The gap closes at \(k=(\pi, \pi)\), (because \(\Delta(k)=0\) at \(k=(\pi, \pi)\). )i.e. the two energy bands touch with each other at \(k=(\pi, \pi)\). This gives us a Dirac point. Here, the system is not an insulator, but a metal (a semi-metal). So we cannot define a topological index.

Remember that we showed early on, to change the topology of an insulator, one need to close the gap. And here is the gap closing.

case II: \(-4 t<\mu<0\)

For \(-4 t<\mu<0, \mathcal{H}_{z}<0\) at \(k=(0,0),\) and \(\mathcal{H}_{z}>0\) for the other three points \(k=(0, \pi),(\pi, 0)\) and \((\pi, \pi) .\) Therefore, we need two functions. First, we draw a small circle around the origin.

  • Inside this small circle, which we will call region \(D_{I},\) we

\[ u^{(\mathrm{I})}-(k)=\frac{1}{\mathcal{N}^{(I)}}\left(\begin{array}{c} \mathcal{H}_{z}(k)-|\overrightarrow{\mathcal{H}}(k)| \\ \mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k) \end{array}\right) \]

  • Outside the circle, which we will call region \(D_{\mathrm{II}},\) we choose

\[ \begin{aligned} u^{(\mathrm{II})}_{-}(k)=\frac{1}{\mathcal{N}^{(\mathrm{II})}}\left(\begin{array}{c} -\mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k) \\ \mathcal{H}_{z}+|\overrightarrow{\mathcal{H}}(k)| \end{array}\right) \end{aligned} \]

The wavefunctions for these two regions are connected by a gauge transformation \[ u^{(\mathrm{II})}_{-}(k)=u^{(I)}_{-}(k) e^{i \phi(k)} \] And the Berry connection are related also by the same gauge transformation \[ \mathcal{A}_{-}^{(\mathrm{II})}(k)=\mathcal{A}_{-}^{(I)}(k)+\nabla_{k} \phi(k) \] The total Berry curvature ( \(2 \pi\) times the Chern number) is \[ \begin{aligned} \oiint_{\mathrm{BZ}} d k \Omega_{-}&=\iint_{D_{i}} d k \nabla \times \mathcal{H}_{\mathrm{L}}^{(l)}+\iint_{D_{\mathrm{u}}} d k \nabla \times \mathcal{A}_{\mathrm{L}}^{(\mathrm{M})} \\& =\oint_{o D_{l}} d k \mathcal{A}_{-}^{(l)}+\oint_{\partial D_{\mathrm{u}}} d k \mathcal{R}_{-}^{(\mathrm{m})} \\&=\oint_{\partial D_{i}} d k \mathcal{A}^{(l)}-\oint_{o D_{l}} d \mathrm{k} \mathcal{A}_{-}^{(\mathrm{m})} \\&=\oint_{\partial D_{I}} d k\left(\mathcal{A}_{-}^{(I)}-\mathcal{A}^{(\Pi)}\right) \\&=-\oint_{\partial D_{I}} d k \nabla_{k} \phi(k) \\&=-\int_{0}^{2 \pi} d \theta \partial_{\theta} \phi \\&=-[\phi(\theta=2 \pi)-\phi(\theta=0)] \\&=\phi(\theta=0)-\phi(\theta=2 \pi) \end{aligned} \] We know that \(\phi(\theta=0)-\phi(\theta=2 \pi)\) is quantized \[ \phi(\theta=0)-\phi(\theta=2 \pi)=2 \pi n \]

expand to chern number

Therefore, the Chern number is quantized. For the model we considered here: \[ \begin{aligned} &\begin{array}{l} \mathcal{H}_{0}(k)=0 \\ \mathcal{H}_{x}(k)=\Delta \sin \left(k_{x}\right) \end{array}\\ &\mathcal{H}_{y}(k)=\Delta \sin \left(k_{y}\right)\\ &\mathcal{H}_{z}(k)=-2 t \cos k_{x}-2 t \cos k_{y}-\mu \end{aligned} \] If we choose \(D_{I}\) to be a very small circle with radius \(k \sim 0\), then around the circle, we can expand everything as a power series of small \(k\) \[ \begin{aligned} &\mathcal{H}_{x}(k)=\Delta \sin \left(k_{x}\right) \approx \Delta k_{x}+O\left(k^{2}\right)\\ &\mathcal{H}_{y}(k)=\Delta \sin \left(k_{y}\right) \approx \Delta k_{y}+O\left(k^{2}\right)\\ &\mathcal{H}_{z}(k)=-2 t \cos k_{x}-2 t \cos k_{y}-\mu=-2 t-\mu+O\left(k^{2}\right) \end{aligned} \] the phase become \[ \begin{aligned} e^{i \phi(k)}&=\frac{\frac{\mathcal{H}_{i}+|\vec{H}(k)|}{\mathcal{H}_{i}(k)+i \not H_{j}(k)}}{\left|\frac{\mathcal{H}_{i}+| \vec{\mu}(k)}{\mathcal{H}_{i}(k)+i \mathcal{H}_{y}(k)}\right|}=\frac{\frac{1}{\mathcal{H}_{i}(k)+i \mathcal{H}_{j}(k)}}{\left|\frac{1}{\mathcal{H}_{i}(k)+i \mathcal{H}_{j}(k)}\right|} \\ &=\frac{\left|\mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k)\right|}{\mathcal{H}_{x}(k)+i \mathcal{H}_{y}(k)}=\frac{\left|\Delta k_{x}+i \Delta k_{y}\right|}{\Delta k_{x}+i \Delta k_{y}} \\ &=\frac{\left|k_{x}+i k_{y}\right|}{k_{x}+i k_{y}}=\frac{\left|k e^{i \theta}\right|}{k e^{i \theta}} \\ &=\frac{1}{e^{i \theta}}=e^{-i \theta} \end{aligned} \] Here, we transfer \(k_{x}+i k_{y}\) into polar coordinate, which is \(k e^{i \theta} .\) This result tells us that \[ \phi(k)=-\theta \] Therefore, \[ \oiint_{\mathrm{BZ}} d k \Omega_{-}=\phi(\theta=0)-\phi(\theta=2 \pi)=0-(-2 \pi)=2 \pi \] So, \[ C=\frac{1}{2 \pi} \oiint_{\mathrm{BZ}} d k \Omega_{-}=1 \] This is a topological insulator with \(C=1\)

Marginal case: \(\mu=0\)

The gap closes at \(k=(0, \pi)\) and \((\pi, 0) .\) The system is not an insulator, so we cannot define the Chern number.

case III: \(0<\mu<4 t\)

For \(0<\mu<4 t, \mathcal{H}_{z}>0\) at \(\mathrm{k}=(\pi, \pi),\) and \(\mathcal{H}_{z}<0\) for the other three points. Here,we can draw a small circle centered at \((\pi, \pi)\) to cut the system into two parts. Inside the circle, near \((\pi, \pi),\) we use \(u^{\text {II }}\) and outside the circle, near (0,0) we use \(u^{I}\). Same as case II, we get a topological insulator with \(C\)

Marginal case: \(\mu=4 t\)

The gap closes at \(k=(\pi, \pi)\). Not an insulator.

case IV: \(\mu>4 t\)

All the four special points has \(\mathcal{H}_{z}<0 .\) So we just use \(u^{I}\) for the whole BZ. No singularity, and thus \(C=0\)

  1. This model has four phases. Two topological phases with \(\mathrm{C}=+1\) and two trivial insulator phase with \(\mathrm{C}=0\).
  2. A topological phase and a trivial insulator phase are always separated by a phase transition, which is known as a topological phase transition.

    • Across a topological transition, the insulating gap closes and then reopens (generically true)
  3. Gap closing is the necessary condition for a topological transition, but it is not sufficient. One may close and reopen the gap without changing the topological index. e.g. the \(\mu=0\) point here.

The dispersion \(\epsilon_{\pm}\) is a periodic function of \(\vec{k}\) space (the hexagon repeats itself in the figure shown above). For the lower band, its energy minimum is located at \(k=0\). The maximum of \(\epsilon_{-}\) is reached at the corners of the BZ. The first \(\mathrm{BZ}\) is a hexagon. It has six corners. However, due to the periodic structure in \(\mathrm{k}\) -space,

  • the three corner points with \(\theta=0, \theta=2 \pi / 3\) and \(\theta=4 \pi / 3\) are the same point (their momenta differ by a reciprocal vector).
  • Similarly, the three corners with \(\theta=\pi, \theta=\pi+2 \pi / 3\) and \(\theta=\pi+4 \pi / 3\) are the same point.

common on

The Dirac point (band crossing points) in a honeycomb lattice is stable as long as

  • the space inversion symmetry \(r \rightarrow-r\) and
  • the time reversal symmetry \(t \rightarrow-t\) are preserved.

We will come back to this point latter. No matter how one perturb the systems (e.g. adding longer-range hoppings), the Dirac point is always there as long as the two symmetries mentioned above are preserved.

For graphene, the lower band is filled and the upper band is empty, which is known as "half-filling”. The "half' here means that the number of electrons \(N_{e}\) over the number of sites If \(N_{s}\) is \(N_{e} / N_{s}=1 / 2\). However, it is worthwhile to notice that the ratio between \(N_{e}\) and the number of unit cells \(N\) is \(1,\) because there are two site in each unit cell. So one of the two bands are totally filled. By gating, one can tune the Fermi energy slightly away from the Dirac point, which is known as a dopped graphene.

AB effect and complex hopping

Aharonov–Bohm effect and complex hopping

Q: Can hopping strength \(t\) be complex?

A: Yes, and the phase can come from the Aharonov-Bohm effect in the presence of a magnetic field or spin-orbital couplings.

The Aharonov-Bohm effect: if one moves a charged particle around a closed contour. The phase difference between the final and initial states is proportional to the magnetic flux enclosed by the contour \[ \phi=e / \hbar \iint B \cdot d S=e / \hbar \oint A \cdot d l \] Now, let's consider a discrete lattice. Consider three sites a, b and c. The hopping strength between these three sites are \(t_{\mathrm{ab}} t_{\mathrm{bc}}\) and \(t_{\mathrm{ca}}\) respectively. If a particle hops from a to b and then to c, the hopping strength around this loop is: \[ t_{\mathrm{ab}} t_{\mathrm{bc}} t_{\mathrm{ca}}=\left|t_{\mathrm{ab}}\right| e^{i \phi_{\mathrm{b}}} *\left|t_{\mathrm{bc}}\right| e^{i \phi_{\mathrm{c}}} *\left|t_{\mathrm{ca}}\right| e^{i \delta_{\mathrm{ca}}}=\left|t_{\mathrm{ab}} t_{\mathrm{bc}} t_{\mathrm{ca}}\right| e^{i\left(\phi_{\mathrm{w}}+\phi_{\mathrm{c}}+\phi_{\mathrm{ca}}\right)} \] The phase picked up by the electron is: \[ \phi_{\mathrm{ab}}+\phi_{\mathrm{bc}}+\phi_{\mathrm{ca}}=\frac{e}{\hbar} \iint {B \cdot d S} \] If \(\mathrm{B}\) is nonzero inside the triangle formed by these three sites, the phase for these hoppings are nonzero.

Please notice that

  1. \(t_{\text {ab }}\) and \(t_{\text {ba }}\) has opposite phase, due to the Hermitian condition (One can also prove this using the Fermi's golden rule, which says that the tunneling amplitude from the state \(| \mathrm{f}\rang\) to \(| \mathrm{i}\rang\) is the complex conjugate of the tuning amplitude from \(| \mathrm{i}\rang\) to \(| \mathrm{f}\rang\) .

  2. The individual phases for \(t_{\mathrm{ab}}, t_{\mathrm{bc}}\) and \(t_{\mathrm{ca}}\) have no physical meaning and their phases depend on the gauge choice. However, the total phase around a loop is independent of gauge, and in fact it is a physical observable, i.e. the magnetic flux.

Proof

We know that the phase of the hopping term is \[ \phi_{\mathrm{ab}}=e / \hbar \int_{a}^{b} A \cdot d l \] Underage transformation: \(A \rightarrow A+\nabla \chi\) \[ \begin{aligned}\phi_{\mathrm{ab}} \rightarrow \phi_{\mathrm{ab}}^{\prime}&=e / \hbar \int_{a}^{b}(A+\nabla \chi) \cdot d l\\&=e / \hbar \int_{a}^{b} A \cdot d l+e / \hbar \int_{a}^{b} \nabla \chi \cdot d l\\&=\phi_{\mathrm{ab}}+\left(\chi_{b}-\chi_{a}\right) \frac{e}{\hbar}\end{aligned} \] Obviously, \(\phi_{\mathrm{ab}}\) is not a physical observables, since it relies on the gauge choice. However, the total phase around a loop is different. It is a loop integral of \(A,\) which is gauge independent and the physics meaning of this Integral is the magnetic flux. \[ \begin{aligned}\phi_{\mathrm{ab}}+\phi_{\mathrm{bc}}+\phi_{\mathrm{ca}}&=e / \hbar \oint A \cdot d l \\\rightarrow \phi_{\mathrm{ab}}^{\prime}+\phi_{\mathrm{bc}}^{\prime}+\phi_{\mathrm{ca}^{\prime}} & =e / \hbar \oint\left(A+\nabla_{\chi}\right) \cdot d l\\&=e / \hbar \oint A \cdot d l+e / \hbar \oint \nabla_{\chi }\cdot d l\\&=e / \hbar \oint A \cdot d l\\&=\phi_{\mathrm{ab}}+\phi_{\mathrm{bc}}+\phi_{\mathrm{ca}}\end{aligned} \] The complex hopping strength induced by \(B\) fields breaks the time-reversal symmetry because \(B \rightarrow-B\) under time reversal. (in other words, under time-reversal one needs to flip the sign of all these phases).

Pauli expression

Using Pauli matrices \[ \mathcal{H}=\mathcal{H}_{0}(k) I+\mathcal{H}_{x}(k) \sigma_{x}+\mathcal{H}_{y}(k) \sigma_{y}+\mathcal{H}_{z}(k) \sigma_{z} \] where \[ \begin{aligned} &\mathcal{H}_{0}=\frac{\mathcal{H}_{11}(k)+\mathcal{H}_{22}(k)}{2}=-2 t^{\prime} \cos \phi\left[\cos \left(k \cdot v_{1}\right)+\cos \left(k \cdot v_{2}\right)+\cos \left(k \cdot v_{3}\right)\right]\\&\mathcal{H}_{z}=\frac{\mathcal{H}_{11}(k)-\mathcal{H}_{22}(k)}{2}=-2 t^{\prime} \sin \phi\left[\sin \left(k \cdot v_{1}\right)+\sin \left(k \cdot v_{2}\right)+\sin \left(k \cdot v_{3}\right)\right]\\&\mathcal{H}_{x}=\operatorname{Re}\left[\mathcal{H}_{21}(k)\right]=-t[\cos (\vec{k} \cdot \overrightarrow{e_{1}})+\cos (\vec{k} \cdot \overrightarrow{e_{2}})+\cos (\vec{k} \cdot \overrightarrow{e_{3}})]\\&\mathcal{H}_{y}=\operatorname{Im}\left[\mathcal{H}_{21}(k)\right]=-t\left[\sin \left(\vec{k} \cdot \vec{e}_{1}\right)+\sin \left(\vec{k} \cdot \vec{e}_{2}\right)+\sin \left(\vec{k} \cdot \vec{e}_{3}\right)\right] \end{aligned} \] The energy dispersions \[ \epsilon_{\pm}(\vec{k})=\mathcal{H}_{0}(\vec{k}) \pm \sqrt{\mathcal{H}_{x}(\vec{k})^{2}+\mathcal{H}_{y}(\vec{k})^{2}+\mathcal{H}_{z}(\vec{k})^{2}} \] The gap between the two bands \[ \epsilon_{+}(\vec{k})-\epsilon_{-}(\vec{k})=2 \sqrt{\mathcal{H}_{x}(\vec{k})^{2}+\mathcal{H}_{y}(\vec{k})^{2}+\mathcal{H}_{z}(\vec{k})^{2}} \] At \(\mathrm{K}\) or \(\mathrm{K}^{\prime}, \mathcal{H}_{x}(\vec{k})=\mathcal{H}_{y}(\vec{k})=0\) and the gap is: \[ \begin{aligned}\epsilon_{+}(\vec{k})-\epsilon_{-}(\vec{k})&=2 \sqrt{\mathcal{H}_{x}(\vec{k})^{2}+\mathcal{H}_{y}(\vec{k})^{2}+\mathcal{H}_{2}(\vec{k})^{2}}\\&=2\left|\mathcal{H}_{2}(\vec{k}=K)\right|\\&=6 \sqrt{3} t^{\prime} \sin \phi\end{aligned} \] In fact

  • at the \(\mathrm{K}\) point, \(\mathcal{H}_{z}=-3 \sqrt{3} t^{\prime} \sin \phi\)
  • at the \(\mathrm{K '}\) point, \(\mathcal{H}_{z}=3 \sqrt{3} t^{\prime} \sin \phi\)

They have opposite signs (as long as \(\phi\) is not \(\mathrm{n} \pi\) ). Based on what we learned last time, this means that one cannot define the wavefunction the whole BZ. We need to cut the systems into two regions.

  • The region I contains the K point,
  • the region II contains the K' points.

And we need to use different eigenvectors for these two regions. Using the same method we learned in the last lecture, one finds that the Chern number here is ±1

Potential energy (breaking inversion symmetry)

Let's keep NNN hoppings to be zero for now and add some potential energy to the Hamiltonian. \[ \begin{aligned}H_{\text {Potentiti }}&=(V+M) \sum_{i} a_{i}^{\dagger} a_{i}+(V-M) \sum_{i} b_{i}^{\dagger} b_{i}\\&=V N+M \sum_{i} a_{i}^{\dagger} a_{i}-M \sum_{i} b_{i}^{\dagger} b_{i}\end{aligned} \] The V part (average potential between a and b sites) just adds a constant term to the energy, since the total particle number \(\mathrm{N}\) is conserved. So we can drop the \(\mathrm{V}\) term and only consider the difference between the potential energies at a and \(\mathrm{b}\) sites (M). In k-space \[ \begin{aligned}H_{\mathrm{Potentai}}&=M \sum_{i} a_{i}^{\dagger} a_{i}-M \sum_{i} b_{i}^{\dagger} b_{i}\\&=M \sum_{k k} a_{k}^{\dagger} a_{k}-M \sum_{k} b_{k}^{\dagger} b_{k}\\&=\sum_{k}\left(a_{k}^{\dagger} b_{k}^{\dagger}\right) M \sigma_{z}\left(\begin{array}{l}a_{k} \\b_{k}\end{array}\right)\end{aligned} \] It adds a \(\sigma_{z}\) component to the Hamiltonian. Same as the NNN complex hopping, this term also opens a gap at the Dirac points. At \(\mathrm{K}\) and \(\mathrm{K}^{\prime}, \mathcal{H}\) now has the same sign ( \(\mathcal{H}=M\) for any \(k\) ). So, we can use one wave-function for the whole \(\mathrm{BZ}\) and thus the system is a trivial insulator.

Q: What will happen if we have both \(H_{\text {potential }}\) and \(H_{\mathrm{MN}}\) ?

A: We just need to look at the signs of \(\mathcal{H}_{z}\) at \(\mathrm{K}\) and \(\mathrm{K}\) ' points. If they have the same sign, we can use one wavefunction to cover the whole \(\mathrm{BZ}\), so \(C=0\) (trivial insulator). If they have opposite signs, the system is a topological insulator with \(\mathrm{C}=\pm 1\)

  • At the K point, \(\mathcal{H}_{z}=M-3 \sqrt{3} t^{\prime} \sin \phi\)
  • At the K' point, \(\mathcal{H}_{z}=M+3 \sqrt{3} t^{\prime} \sin \phi\)

Therefore

  1. as long as \(|M|<\left|3 \sqrt{3} t^{\prime} \sin \phi\right|\), the system is an topological insulator ( \(\mathcal{H}_{z}\) flips sign).
  2. If \(|M|>\left|3 \sqrt{3} t^{\prime} \sin \phi\right|, \mathcal{H}_{z}\) is always positive (or negative) and thus the system is topologically trivial.
  3. The marginal case \(|M|=\left|3 \sqrt{3} t^{\prime} \sin \phi\right|\) is a topological transition, where \(\mathcal{H}_{z}=0\) at either the \(\mathrm{K}\) point or the \(\mathrm{K}\) ' point. Because \(\mathcal{H}_{x}=\mathcal{H}_{y}=0\) at these two points, the gap must be zero at one of the two points.

Remarks

  • The model of Haldane is the first example of a topological insulator beyond quantum Hall effect.
  • It demonstrates that topological insulator is a generic concept, which may appear in any insulating systems (NOT just quantum Hall).
  • It also demonstrates that as long as the topological index is nonzero, one will observe all the topological phenomena expected for a quantum Hall state, including the quantized Hall conductivity and the existence of the edge states.
  • The key differences between the model of Haldane and the quantum Hall effects are
      1. the B field is on average zero in the model of Haldane while the QHE has a uniform B field
      1. there is a very strong lattice background in the model of Haldane while the QHE requires weak lattice potential.
  • Systems similar to the Haldane's model are known as topological Chern insulators or Chern insulators (average B is 0 and have a strong lattice potential). But sometimes, Chern insulators are also used to refer to the quantum Hall effect.
  • The model of Haldane is also the foundation to explore more complicated and exotic topological states. For example, the time-reversal invariant topological insulators was first proposed using a modified Haldane's models, which we will study latter in the semester.

6 symmetry properties of the Berry curvature

time-reversal symmetry

In this section, we consider systems with time-reversal symmetry and shows that in the presence of the time-reversal symmetry, the Chern number must be zero. Therefore, to get a topological insulator with nonzero \(C,\) one must break the time-reversal symmetry.

Dispersion relation

Dispersion relation under time-reversal transformation: \[ \mathcal{T} \epsilon_{n}(k)=\epsilon_{n}(-k) \] This is because \(\mathcal{T}\) turns \(k\) into \(-\vec{k}\). There are two ways to see that \(\mathcal{T}\) turns \(\vec{k}\) into \(-\vec{k}\).

  • \(k\) is the momentum. Under time-reversal, velocity changes sign, so does the momentum.
  • \(\vec{k}=-i \partial_{r} .\) Under \(\mathcal{T}, \partial_{r}\) remains invariant. However, because \(\mathcal{T}\) is an anti-unitary transformation (which changes a complex number to its complex conjugate), it flips the sign of \(i .\) Therefore, \(\mathcal{T} \vec{k}=\mathcal{T}\left(-i \partial_{r}\right)=i \partial_{r}=-\vec{k}\)

If the system is time-reversal invariant, the time-reversal symmetry implies that \(\epsilon_{n}(k)\) is invariant under time-reversal transformation \[ \mathcal{T} \epsilon_{n}(k)=\epsilon_{n}(k) \] Compare the two equations, we have: \[ \epsilon_{n}(k)=\epsilon_{n}(-k) \] Bottom line: for systems with time-reversal symmetry, the dispersion is an even function of the momentum \(\vec{k}\).

Bloch wave function

Bloch wave under time-reversal transformation: \[ \mathcal{T} u_{n, k}(r)=u_{n,-k}(r)^{*} \] Here, \(\mathcal{T}\) changes \(k\) to \(-k\). In addition, because \(u_{n, k}(r)\) is in general a complex function, \(\mathcal{T}\) changes the function to its complex conjugate. If the system is time-reversely invariant, \[ \mathcal{T} u_{n, k}(r)=e^{i \phi(k)} u_{n, k}(r) \] By comparing the two equations above, we found that for a system with time-reversal symmetry, \[ u_{n_{,-} k}(r)^{*}=e^{i \phi(k)} u_{n, k}(r) \] This arbitrary phase is here because wavefunctions are defined up to an arbitrary phase

note: \(\psi_{n, k}(r)=u_{n, k}(r) e^{i k r}\) relation between Bloch wave function and wave function

Berry connection

The Berry connection is defined as: \[ \mathcal{A}_{n}(k)=-i\left\langle u_{n, k}\left|\nabla_{k}\right| u_{n, k}\right\rangle=-i \int d \vec{r} u_{n, k}(\vec{r})^{*} \partial_{k} u_{n, k}(\vec{r}) \] The under time-reversal, the new \(\mathcal{H}_{n}(k)\) is defined as \[ \begin{aligned}\mathcal{T} \mathcal{A}_{n}(k)&=-i\left\langle\mathcal{T} u_{n, k}\left|\nabla_{k}\right| \mathcal{T} u_{n, k}\right\rangle\\&=-i \int d \vec{r} u_{n_{n-k}}(\vec{r}) \partial_{k} u_{n_{n}-k}(\vec{r})^{*}\\&=i \int d \vec{r} \partial_{k} u_{n_{n-k}}(\vec{r}) u_{n_{n-k} k}(\vec{r})^{*}\\&=-i \int d \vec{r} u_{n_{n}-k}(\vec{r})^{*} \partial_{-k} u_{n_{n-k}}(\vec{r})\\&=-i\left\langle u_{n_{n-k}}\left|\nabla_{-k}\right| u_{n_{1}-k}\right\rangle\\&=\mathcal{A}_{n}(-k)\end{aligned} \] If the system has time-reversal symmetry \[ \mathcal{T} \mathcal{A}_{n}(k)=\mathcal{H}_{n}(k)+\nabla_{k} \chi(k) \] So we have \[ \mathcal{A}_{n}(-k)=\mathcal{A}_{n}(k)+\nabla_{k} \chi(k) \] Bottom line: For systems with time-reversal symmetry, \(\mathcal{A}_{\mathrm{n}}(-\mathrm{k})\) and \(\mathcal{A}_{\mathrm{n}}\) ( \(\mathrm{k}\) ) differs by a gauge transformation \(\nabla_{k} \chi(k)\)

Berry curvature

The Berry curvature is defined as: \[ \begin{aligned}\mathcal{F}_{n}(k)&=-i \epsilon_{i j}\left\langle\partial_{k_{1} i} u_{n, k} | \partial_{k_{j}} u_{n, k}\right\rangle\\&=-i \epsilon_{i j} \int d \vec{r} \partial_{k_{1} i} u_{n, k}(\vec{r})^{*} \partial_{k_{j} j} u_{n, k}(\vec{r})\end{aligned} \] The under time-reversal, the new \(\mathcal{F}_{n}(k)\) is defined as \[ \begin{aligned}\mathcal{T} \mathcal{F}_{n}(k)&=-i \epsilon_{\mathfrak{i} {\mathfrak{j}}}\left\langle\partial_{\mathrm{k} {\mathfrak{i}}} \mathcal{T}\mathfrak{u}_{n, k} | \partial_{\mathrm{kj} } \mathcal{T}_{\mathfrak{n}, k}\right\rangle\\&= -i \epsilon_{\mathfrak{i} {\mathfrak{j}}} \int d \vec{r} \partial_{\mathrm{k} {\mathfrak{i}}} \mathcal{T} \mathfrak{u}_{n, k}\left(\vec{r}\right)^{*} \partial_{\mathrm{k} {\mathfrak{j}}} \mathcal{T}_{ {n}, -k}(\vec{r})\\&=-i \epsilon_{\mathfrak{i} {\mathfrak{j}}} \int d \vec{r} \partial_{\mathrm{ki}} u_{n_{n}, k}(\vec{r}) \partial_{\mathrm{k} {\mathfrak{j}}} u_{n, -k}(\vec{r})^{*}\\&=-i \epsilon_{\mathfrak{i} {\mathfrak{j}}} \int d \vec{r} \partial_{\mathrm{k} i} u_{n , k}(\vec{r})^{*} \partial_{\mathrm{k} {\mathfrak{j}}} u_{n,-k}(\vec{r})\end{aligned} \] If we swap i and j here \[ \begin{aligned}\mathcal{T} \mathcal{F}_{n}(k)&=-i \epsilon_{\mathrm{ji}} \int d \vec{r} \partial_{\mathrm{k } i} u_{n,-k}(\vec{r})^{*} \partial_{\mathrm{k } j} u_{n-k}(\vec{r})\\&=i \epsilon_{\mathrm{ij}} \int d \vec{r} \partial_{\mathrm{k } i} u_{n,-k}(\vec{r})^{*} \partial_{\mathrm{kj}} u_{n_{1}-k}(\vec{r})\\&=-\mathcal{F}_{n}(-k)\end{aligned} \] If the system has time-reversal symmetry \[ \mathcal{T F}_{n}(k)=\mathcal{F}_{n}(k) \] So we have \[ \mathcal{F}_{n}(-k)=-\mathcal{F}_{n}(k) \] Bottom line: For systems with time-reversal symmetry, \(\mathcal{F}_{n}(k)\) is an odd function of \(\mathrm{k}\)

Because the integral of an odd function over the whole Brillouin zone must be zero, (the contribution from \(k\) and \(- k\) cancels each other), the Chern number for a time-reversally invariant system must be \(C = 0\). The Chern number \(C=\frac{1}{2 \pi} \iint d k \mathcal{F} _{n}(k)\) \[ \begin{aligned} \mathcal{T} C &= \mathcal{T} \frac{1}{2 \pi} \iint d k_{x} d k_{y} \mathcal{F}_{n}(k) \\&=-\frac{1}{2 \pi} \iint d k_{x} d k_{y} \mathcal{F}_{n}(-k) \\&=-\frac{1}{2 \pi} \iint d k_{x} d k_{y} \mathcal{F}_{n}(k) \\&=-C \end{aligned} \] If the system has time-reversal symmetry, \[ \mathcal{T} C=C \] So we have \(C=-C\), which means \(C=0\).

Bottom line: to have a nontrivial Chern number, the system must break the time-reversal symmetry (using external B field or some other method).

space-inversion symmetry

Space inversion transformation \(I\) turns a vector \(\vec{r}\) into \(-\vec{r}\). It also changes \(\vec{k}\) into \(-\vec{k},\) which is similar to \(\mathcal{T}\). However, \(T\) is an unitary transformation, while \(\mathcal{T}\) is anti-unitary.

Dispersion relation

Dispersion relation under time-reversal transformation: \[ I \epsilon_{n}(k)=\epsilon_{n}(-k) \] If the system is invariant under space inversion \[ I \in_{n}(k)=\epsilon_{n}(k) \] Compare the two equations, we have: \[ \epsilon_{n}(k)=\epsilon_{n}(-k) \] Bottom line: For systems invariant under space inversion, the dispersion is an even function of the momentum

If one want make a system where the dispersion is NOT an even function, one need to break both space inversion symmetry and time reversal symmetry.

Bloch wave function

\[ I u_{n, k}(r)=u_{n,-k}(r) \]

If the system is invariant under space inversion \[ I u_{n, k}(r)=e^{i \phi(k)} u_{n, k}(r) \] By comparing the two equations above, we found that for a system with time-reversal symmetry, \[ u_{n,-k}(r)=e^{i \phi(k)} u_{n, k}(r) \]

Berry connection

Berry connection \(\mathcal{A}_{n}(\boldsymbol{k})=-i\left\langle\boldsymbol{u}_{n, k}\left|\nabla_{k}\right| \boldsymbol{u}_{n, k}\right\rangle\) \[ I \mathcal{A}_{n}(k)=\mathcal{A}_{n}(-k) \] If the system is invariant under space inversion \[ I \mathcal{A}_{n}(k)=\mathcal{A}_{n}(k)+\nabla_{k} \chi(k) \] So we have \[ \mathcal{A}_{n}(-k)=\mathcal{A}_{n}(k)+\nabla_{k} \chi(k) \] Bottom line: For systems invariant under space inversion, \(\mathcal{A}(-\mathrm{k})\) and \(\mathcal{A}_{\mathrm{n}}\) ( \(\mathrm{k}\) ) differs by a gauge transformation \(\nabla_{k} \chi(k)\)

Berry curvature

\(\mathcal{F}_{n}(k)=-i \epsilon_{i j}\left\langle\partial_{i} u_{n, k} | \partial_{j} u_{n, k}\right\rangle\) \[ I \mathcal{F}_{n}(k)=\mathcal{F}_{n}(-k) \] If the system is invariant under space inversion \[ I \mathcal{F}_{n}(k)=\mathcal{F}_{n}(k) \] So we have \[ \mathcal{F}_{n}(k)=\mathcal{F}_{n}(-k) \] Bottom line: For systems invariant under space inversion, \(\mathcal{F}_{n}(k)\) is an even function of \(\mathrm{k}\)

Time-reversal symmetry tell us \(\mathcal{F}_{n}(k)\) is an odd function of \(\mathrm{k}\). If system have both \(\mathcal{T}\) and \(I\) symmetry, \(\mathcal{F}\) must be both an even function and an odd function. So \(\mathcal{F}=0\) at any k point.

The Chern number

\(C=\frac{1}{2 \pi} \iint d k \mathcal{F}_{n}(k)\) \[ \mathcal{I} C=C \]

IT symmetry

Under \(I T\), \[ I T F _{n}(k)=I\left[- F _{n}(-k)\right]=- F _{n}(k) \] If system is invariant under \(I T\), \[ I T F _{n}(k)=- F _{n}(k) \] So, we have \[ F _{n}(k)=- F _{n}(k) \] Bottom line: For systems invariant under \(I T , F _{n}(k)=0\) at every \(k\).

  • For \(T\) symmetry, the integral of \(F\) is zero \(( C =0)\), but \(F\) can be nonzero at each \(k\) point.
  • For \(I T\) symmetry, \(F\) is zero at every single k point. This is a stronger statement!

Please notice that all these conclusions assumes that the system is gapped. For gapless systems with \(I T\) symmetry, \(F\) can actually be nonzero and quantized, which will be discussed in the next section.

flux and Dirac points

1 magnetic field, Aharonov-Bohm effect and magnetic \(\pi\) - flux

Q: Why \(B\) changes sign under \(T\) ?

A: Many ways to see it. Here, I use the \(A - B\) effect.

In the presence of \(B\) field, Aharonov and Bohm tells us that if we move a charged-particle around a closed loop, the electron will pick up a phase, and the phase is the total magnetic field enclosed by the loop (times \(e / \hbar)\) \[ \phi=e / \hbar \iint B \cdot d S=e / \hbar \oint A \cdot d l \] Under time-reversal \(e^{i \phi}\) turns into \(e^{-i \phi}\), because time-reversal is anti-unitary. So the A-B phase changes sign \[ T \phi=-\phi \] In other words, the integral of \(B\) changes sign \[ T \iint_{D} B \cdot d S=-\iint_{D} B \cdot d S \] Because this equation is true for any region \(D\), we have \(B=-B\).

If a system has time-reversal symmetry, everything must remain the same before and after we flip the arrow of time, including the \(A - B\) phase \[ e^{i \phi}=e^{-i \phi} \] Therefore, we have \(\phi=-\phi\). So we have \(\phi=0\) for any region \(D\), which implies that \(B=0\).

Q: Do we really need to have \(B=0\) to preserve the time-reversal symmetry.

A: NO!

This is because phase is only well-define up to mod \(2 \pi\). The time-reversal symmetry requires \(e^{i \phi}=e^{-i \phi}\), which does NOT implies \(\phi=-\phi\). In fact, as long as \(\phi=-\phi+2 n \pi\), the \(A - B\) phase will be the same before and after the time-reversal transformation. This means that to preserve the time-reversal symmetry we just need \(\phi=n \pi\) while \(n\) don't need to be 0 . So we need to have \(\iint_{D} B \cdot d S=n \pi\) for any \(D\). This means that \(B\) can contains some delta functions. \[ B=\frac{\hbar}{e} \sum_{i} n_{i} \pi \delta\left(r-r_{i}\right) \] For this \(B\) field, the integral of \(B\) in region \(D\) is just \[ \phi=e / \hbar \iint B \cdot d S=\left(\sum_{i}^{\prime} n_{i}\right) \pi \] Here the sum \(\sum_{i}^{\prime}\) is over all \(r_{i}\) inside \(D\). A delta function in \(B\) is a magnetic flux, which is known as a \(n \times \pi\) -flux. These fluxes don't break the time-reversal symmetry. In addition, it worthwhile to point out that for the \(A - B\) phase is only well defined up to mod \(2 \pi\). So all the \(2 n \pi\) fluxes give the same \(A - B\) phase, which is 0 . All the \((2 n +1) \pi\) fluxes give the same \(A - B\) phase, which is \(\pi\).

2 Berry flux

We can do the same thing for the Berry curvature \(F ( k )\). If we goes around a contour in the \(k\) -space, the phase change of the Bloch wave function around this contour is \[ \phi=\iint_{D} F d k^{2} \] This is known as the Berry phase. Under \(I T , D\) is invariant so \[ I T \phi=I T \iint_{D} F (k) d k^{2}=-\iint_{D} F (k) d k^{2}=-\phi \] If the system is \(I T\) invariant, we must have \(\phi=-\phi\) (but up to \(\bmod 2 \pi)\) \[ \phi=-\phi+2 n \pi \] so \[ \phi=n \pi \] As a result, if we have \(I T\) symmetry, \(\iint_{D} F d k^{2}= n \pi\) for any region \(D\). So \(F\) must be either 0 or some delta functions. \[ F (k)=\sum_{i} n_{i} \pi \delta\left(k-k_{i}\right) \] These delta functions are known as Berry fluxes. Because the Berry phase is only well defined up to mod \(2 \pi\), we have in general two types of Berry fluxes: 0 and \(\pi\). And a \(\pi\) flux is a Dirac point.

3 Dirac point

Near a Dirac point, the kernel of the Hamiltonian is: \[ H =v_{F}\left(\begin{array}{cc} 0 & kx -i ky \\ kx +i ky & 0 \end{array}\right) \] Using polar coordinates \[ H =v_{F} k\left(\begin{array}{cc} 0 & e^{-i \phi} \\ e^{i \phi} & 0 \end{array}\right) \] The eigenvalues for this matrix is \(\epsilon_{\pm}=\pm v_{F} k .\) The eigenvectors are: \[ u_{+}=\frac{1}{\sqrt{2}}\left(\begin{array}{c} e^{-i \phi} \\ 1 \end{array}\right) \text { and } u_{-}=\frac{1}{\sqrt{2}}\left(\begin{array}{c} -e^{-i \phi} \\ 1 \end{array}\right) \] The Berry connection in the polar coordinates has two components, radius and angle. \[ A_{+k}=-i\left\langle u_{+}\left|\partial_{k}\right| u_{+}\right\rangle=0 \]

\[ \begin{aligned} A_{+\phi}&=-\frac{i}{k}\left\langle u_{+}\left|\partial_{\phi}\right| u_{+}\right\rangle \\&=-\frac{i}{k} \frac{1}{\sqrt{2}}\left(-e^{i \phi} \quad 1\right) \partial_{\phi} \frac{1}{\sqrt{2}}\left(\begin{array}{c} -e^{-i \phi} \\ 1 \end{array}\right) \\&=-\frac{i}{k} \frac{1}{2}\left(e^{i \phi} \quad 1\right)\left(\begin{array}{c} -i e^{-i \phi} \\ 1 \end{array}\right) \\&=-\frac{1}{2 k} \end{aligned} \] Let's choose a small circle around the Dirac point, the flux through this circle is: \[ \begin{aligned} \chi &=\iint F d k\\&=\oint A \cdot d k \\&=\int_{0}^{2 \pi} A _{\phi} k d \phi \\&=-\int_{0}^{2 \pi} \frac{1}{2 k} k d \phi \\&=-\frac{1}{2} \int_{0}^{2 \pi} d \phi \\&=-\pi \end{aligned} \]

Why is a Dirac point stable?

This is because the Berry fluxes are very stable. If one has \(T I\) symmetry, the Berry flux is quantized to integer values. Once its value is fixed (say \(\pi)\), no perturbation can change it. The only thing that can change continuously is the location of the delta functions. In other words, without break the \(T I\) symmetry, one can only move the Dirac points around in the \(k\) -space, but they cannot just disappear. To get ride of Dirac points, one need to move two Dirac points together and let them annihilate each other (let a \(\pi\) flux meet a \(-\pi\) flux, so that we get zero flux). Another way to get ride of Dirac points is to break either the \(T\) or the \(I\) symmetry.

7 Edge states (numerical calculations)

Why energy is a function of \(k_{x}\) and \(k_{y}\)

Consider one electron moving in an infinite 2D lattice (for simplicity, we consider a square lattice here). Because the lattice is invariant under lattice translations \((x \rightarrow x+a\) and \(y \rightarrow y+a)\), lattice momentum is a good quantum number (a conserved quantity). Momentum conservation means that \[ \left[k_{x}, H\right]=\left[k_{y}, H\right]=0 \] In addition, we know \(\left[k_{x}, k_{y}\right]=0\). With three operators which commute with each other, quantum mechanics tells us that we can find common eigenstates for all these three operators, and we can use these eigenstates as the basis for the Hilbert space. This set of basis looks like: \[ \left|\psi_{\epsilon_{n}, k_{x}, k_{y}}\right\rangle \] This state is the eigenstate of \(H , k_{x}\) and \(k_{y}\) with eigenvalues \(\epsilon_{n}, k_{x}\) and \(k_{y}\) respectively. Here, one need a sub-index (n) to distinguish different bands. So we have the band structure: \[ \epsilon_{n}\left(k_{x}, k_{y}\right) \]

What if we have an finite system?

To study edge states, we must have an edge. So we cannot use an infinite 2D plan. Let's consider the simplest case here, a infinitely long stripe. We assumes that the system is infinitely long along \(x\), but it has a finite width along y. Now, the translational symmetry along \(x\) is still preserved. But there is no translational symmetry along y (due to the existence of the edge). So \(k_{x}\) is still a good quantum number but \(k_{y}\) is NOT. \[ \left[k_{x}, H\right]=0 \quad \text { but } \quad\left[k_{y}, H\right] \neq 0 \] So we cannot define common eigenstates for \(H, k_{x}\) and \(k_{y}\), but we can still find eigenstates for \(H\) and \(k_{x}\), because they commute with each other. \[ \left|\psi_{\epsilon_{m}, k_{x}}\right\rangle \] Here, again, we need an extra index distinguish different states with the same \(k_{x}\). Here I emphasize that \(m\) is NOT just the band index \(n\). It includes information for both \(n\) and \(k_{y}\). So now, energy is a bunch of function of \(k_{x}\) \[ \epsilon_{m}\left(k_{x}\right) \] where \(m=1,2,3 \ldots N _{S}\). We have \(N_{S}\) functions of \(k_{x}\). We can draw these \(N_{S}\) functions on the \(k_{x}-\epsilon\) plane, which gives us \(N_{S}\) curves, which is our 1D dispersion.

Examples

We consider the model of Haldane for a system with infinite size along \(x\), by finite width along y. The Hamiltonian is invariant under translations along the \(x\) axis \((x \rightarrow x+\sqrt{3} a)\), so the \(x\) -component of the momentum is a conserved quantity \(\left[k_{x}, H\right]=0\). So we can find common eigenstates of \(k_{x}\) and \(H\). The system has two edges: one on the top, one at the bottom. We can consider this lattice as a bunch of \(1 D\) (horizontal) lines coupled together, so the width are described by the number of chains one have in this infinite stripe \((N)\). For \(\epsilon_{m}\left(k_{x}\right)\), we have \(m =1,2, \ldots 2 N\). This factor of 2 comes from the fact that we have two sites in each unit cell for a honeycomb lattice.

The second panel is the same calculation for a system with width \(N =100\left(N_{S}=2 N=200\right)\). The last panel shows the bulk band structure (side view).
