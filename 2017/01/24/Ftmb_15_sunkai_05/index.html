spontaneous symmetry breaking

Q1: Symmetry: if a the Hamiltonian of a system has certain symmetry, can the system have a lower symmetry?

Q2: Analyticity: if we have only analytic functions, can we get something non-analytic?

As will discussed below, the answers are yes and yes. The phenomenon, where the system has a lower symmetry than the Hamiltonian is called spontaneous symmetry breaking.

1 ferromagnetic transition in the Ising model

Consider a spin lattice where each site has spin-1/2 (say a square lattice). Assuming that the spin want to point in the same direction along the z-axis, \[ H=-J \sum_{(i, j\rangle} S_{i}^{z} S_{j}^{z} \] where

  • \(\langle i, j\rangle\) represents the nearest-neighbors
  • the coupling constant \(J>0 .\)

Ising symmetry

This system has the Ising symmetry \(S^{z} \rightarrow-S^{z}\) for all the spins, but the state may not have this symmetry.

  • At high temperature, when entropy dominates, the spins are pointing in random directions, which maximizes the entropy.
  • At low temperature, when entropy contribution is small, the spins wants to align in the same direction (either \(+z\) or \(-z\) ) to minimize the energy.

We can use the average value of \(S^{z},\left\langle S^{z}\right\rangle\) as our order parameter (assuming all sites have the same \(\left.\left\langle S^{2}\right\rangle\right)\). This order parameter shows two different phases.

  • At high temperature, \(\left\langle S^{z}\right\rangle=0,\) which is the paramagnetic phase.
  • At low temperature, \(\left\langle S_{z}\right\rangle \neq 0,\) which is the ferromagnetic phase.

The temperature where \(\left\langle S^{2}\right\rangle\) turns from zero to nonzero is called the transition point (for a second order phase transition, it is also called the critical point, \(T_{c}\) ).

In the low temperature phase, \(\left\langle S^{z}\right\rangle\) is nonzero. In this ferromagnetic phase, the state is not invariant under \(S^{z} \rightarrow-S^{z},\) i.e. the system doesn't have the Ising symmetry.

Statistical physics tells us that \[ \left\langle S_{i}^{z}\right\rangle=\frac{1}{Z} \sum_{n} S_{n i}^{z} \exp \left(-E_{n} / k_{B} T\right) \] where we sum over all possible spin configurations (labeled by the integer \(n\) ). For a specific spin configuration, we assume its energy is \(E_{n}\) and the spin at site \(i\) is \(S_{n i}{ }^{z}\)

Q1: Everything on the right hand side are an analytic functions. However, \(\left\langle S_{i}^{z}\right\rangle\) as a function of temperature \(T\) is obviously an non-analytic function with singularity at \(T_{c}\). How can this be possible? Q2: For the right hand side, \(E_{n}\) is even under \(S_{z} \rightarrow-S_{z}\) but \(S_{n i}{ }^{z}\) is odd. Therefore, \(S_{n i}{ }^{z} \exp \left(-E_{n} / k_{B} T\right)\) is an odd function under \(S_{z} \rightarrow-S_{z}\). For an odd function, by summing over all possible states \(n,\) the total value must be zero. How can we get nonzero \(\left\langle S^{z}\right\rangle ?\) In fact, for spontaneous symmetry breaking, there are two important ingredients

  1. we need a small symmetry breaking field.
  2. the system size needs to be infinitely large.

The first ingredient is automatically satisfied in most real systems. For example, for the ferromagnetic transition, we always have some small but finite magnetic field in any experiments (lets assume it is along the \(+ z\) direction), this field will favor the state with \(\left\langle S_{z}\right\rangle>0 .\) Therefore, in reality, what we obtained in an experiment is \[ \left\langle S_{z}\right\rangle_{+}=\operatorname{Lim}_{h \rightarrow 0^{+}}\left\langle S_{z}(h)\right\rangle \] Or \[ \left\langle S_{z}\right\rangle_{-}=\operatorname{Lim}_{h \rightarrow 0^{-}}\left\langle S_{z}(h)\right\rangle \] where \(h\) is a small external magnetic field and we try our best to reduce the strength of this field. When we approaches zero field, we have two options, from the \(+\) or - side. These two options give us \(\left\langle S_{z}\right\rangle_{+}\) and \(\left\langle S_{z}\right\rangle_{-}\) respectively. Due to the Ising symmetry, we know that \(\left\langle S_{z}\right\rangle_{+}=-\left\langle S_{z}\right\rangle_{-}\). If \(\left\langle S_{z}(h)\right\rangle\) is an analytic function (not singular \(h=0\) ), it is easy to prove that \[ \lim _{h \rightarrow 0^{+}}\left\langle S_{z}(h)\right\rangle=\operatorname{Lim}_{h \rightarrow 0^{-}}\left\langle S_{z}(h)\right\rangle \] So \[ \left\langle S_{z}\right\rangle_{+}=\left\langle S_{z}\right\rangle_{-} \] Because we know that \(\left\langle S_{z}\right\rangle_{+}=-\left\langle S_{z}\right\rangle_{-},\) we find immediately that \[ \left\langle S_{z}\right\rangle_{+}=\left\langle S_{z}\right\rangle_{-}=0 \] However, if \(\left\langle S_{z}(h)\right\rangle\) is NOT an analytic function at \(h=0\) \[ \lim _{h \rightarrow 0^{+}}\left\langle S_{z}(h)\right\rangle \neq \operatorname{Lim}_{h \rightarrow 0^{-}}\left\langle S_{z}(h)\right\rangle \] If we approach the zero field limit from the \(+\) side, we get \(\left\langle S_{z}\right\rangle=\left\langle S_{z}\right\rangle_{+} .\) If we approaches \(h=0\) from the \(-\) side, we get \(\left\langle S_{z}\right\rangle=\left\langle S_{z}\right\rangle_{-}=-\left\langle S_{z}\right\rangle_{+} .\) This is the reason, why we have two possible values for the magnetization in the low-temperature. Here, I used the fact \(\left\langle S_{z}\right\rangle_{+}=-\left\langle S_{z}\right\rangle_{-}\). But we know that everything formula in statistic mechanics is analytic, e.g. \[ \left\langle S_{i}^{z}(h)\right\rangle=\frac{1}{Z} \sum_{n} S_{n i}^{z}(h) \exp \left(-E_{n}(h) / k_{B} T\right) \] Q: How can we get an non-analytic \(\left\langle S_{z}(h)\right\rangle\) using analytic functions? A: This is because we have one singular object in our system, i.e. the system size \(L \rightarrow+\infty\). We know that if we have a series of functions \(f_{i}(x)\) with \(i=1,2,3, \ldots .\) If every function is analytic, the infinite \(i\) limit, \(\lim _{i \rightarrow+\infty} f_{i}(x),\) may not necessarily be an analytic function. For example, \[ f_{i}(x)=\frac{x}{x^{2}+1 / i^{2}} \] For any value of \(i, f_{i}(x)\) is analytic, but \[ \lim _{i \rightarrow+\infty} f_{i}(x)=\frac{1}{x} \] This is an singular function at \(x=0\). If we approaches \(x=0\) from the \(+\) side, we find \[ \lim _{x \rightarrow 0^{+}} \lim _{i \rightarrow+\infty} f_{i}(x)=\lim _{x \rightarrow 0^{*}} \frac{1}{x}=+\infty \] If we approaches \(x=0\) from the - side, we get \[ \begin{array}{l} \operatorname{Lim}_{x \rightarrow 0^{-}} \lim _{i \rightarrow+\infty} f_{i}(x)=\lim _{x \rightarrow 0^{-}} \frac{1}{x}=-\infty \\ \lim _{x \rightarrow 0^{+}} \lim _{i \rightarrow+\infty} f_{i}(x) \neq \lim _{x \rightarrow 0^{-}} \lim _{i \rightarrow+\infty} f_{i}(x) \end{array} \] Similarly, for \(\left\langle S_{z}(h)\right\rangle\) discussed above, if we take into account the system size \(L, \quad\left\langle S_{z}(h, L)\right\rangle\) is an analytic function for any finite \(L,\) but the \(L \rightarrow \infty\) may be singular. If that is the case, the following two quantities may not be the same in general \[ \lim _{h \rightarrow+0} \lim _{L \rightarrow+\infty}\left\langle S_{z}(h, L)\right\rangle \neq \lim _{h \rightarrow-0} \lim _{L \rightarrow+\infty}\left\langle S_{z}(h, L)\right\rangle \] This is what happens in an phase which spontaneously break some symmetry of the Hamiltonian.

2 The Ginzburg-Landau approach (phenomenological)

If we zoom out and coarse grain the lattice, we can define an order parameter in continuum. \[ m(\vec{r})=\frac{1}{\delta V } \sum_{i \in \delta V }\left\langle S_{i}^{z}\right\rangle \] Here, we draw a small box around the position \(\vec{r},\) which we called \(\delta V\). Inside \(\delta V ,\) we get the total spin by summing over all sites inside this box. Then, we divided the number by the volume of this box \(\delta V\). We choose \(\delta V\) to be much larger than the size of a unit cell, but much smaller than the system size. Under the Ising transformation, \(S_{z} \rightarrow-S_{z},\) it is easy to check that \(m(\vec{r}) \rightarrow-m(\vec{r}) .\) With this quantity, we can write down the Ginzburg-Landau free-energy \(F=\int d \vec{r}\left(\partial_{i} m \partial_{i} m+\alpha m^{2}+g m^{4}+\right.\) higher order terms \()\) and the minimum of \(F\) give us the expectation value of \(m(\vec{r})\). Here, we refines the unit of length so that the \(\partial_{i} m \partial_{i} m\) term has coefficient unity. The Ginzburg-Landau theory is an phenomenological theory, which means that we cannot determine the value of the control parameters, which depends on the microscopic details of the system. The rule to write down the GL free energy is to write down every single term allowed by symmetry \((m \rightarrow-m) .\) For example, a term proportional to \(m\) or \(m^{3}\) is not allowed. If we have those terms in the theory, it will violate the \(m \rightarrow-m\) symmetry. Q: why the spatial derivative starts from second order? A: because \(m \partial_{i} m=\partial_{i} \frac{m^{2}}{2},\) which is a total derivative term. The integral of a total derivative is zero (up to a surface term, which typically can be ignored, unless the system is a topological state).

Microscopic approach (fix the numbers)

The control parameters (e.g. \(\alpha\) and \(g\) ) can be determined using diagrammatic techniques we discussed early on. They are known as the vertex functions. Here, I briefly describe the construction and details can be found in D.J. Amit, field theory, the renormalization group, and critical phenomena, World Scientific. First, we add some external fields to the Hamiltonian (notice that the \(h_{i}=0\) limit recovers the original Ising model) \[ H\left(\left\{h_{i}\right\}\right)=-J \sum_{\langle i, j\rangle} S_{i}^{Z} S_{j}^{z}+\sum_{i} S_{i}^{z} h_{i} \] Then, using the new Hamiltonian we compute the partition function \(Z,\) which now relies on \(\left\{h_{i}\right\}\) \[ Z\left(\left\{h_{i}\right\}\right)=\sum_{n} \exp \left(-E_{n}\left(\left\{h_{i}\right\}\right) / k_{B} T\right)=\operatorname{tr}\left[\exp \left(-H\left(\left\{h_{i}\right\}\right) / k_{B} T\right)\right] \] Then, we compute take a log and get the Gibbs free energy \[ G\left(\left\{h_{i}\right\}\right)=-k_{B} T \ln Z\left(\left\{h_{i}\right\}\right) \] Then, we define \[ \bar{\phi}_{i}=\frac{\delta G (\{h\})}{\delta h _{i}}=\frac{1}{Z(\{h\})} \operatorname{tr}\left[S_{i}^{z} \exp \left(-H\left(\left\{h_{i}\right\}\right) / k_{B} T\right)\right] \] which is the expectation value of \(S_{i}^{z}\) for site \(i\) Then, we make a Legendre transformation to get the free energy \(F\) \[ F\left(\left\{\bar{\phi}_{i}\right\}\right)=\sum \bar{\phi}_{i} h_{i}-G\left(\left\{h_{i}\right\}\right) \] Now, the free energy is a function of \(\bar{\phi}_{i},\) and one can easily prove that \[ \frac{\delta F\left(\left\{\bar{\phi}_{i}\right\}\right)}{\delta \bar{\phi}_{i}}=\sum_{i} h_{i} \] Therefore, the extremum condition of \(F\left(\left\{\bar{\phi}_{i}\right\}\right)\) (i.e. \(\frac{\left.\delta F\left(\bar{\phi}_{i}\right\}\right)}{\delta \bar{\phi}_{i}}=0\) ) coincides with the zero field condition. And the value of \(\bar{\phi}_{i}\) that minimizes \(\left.F\left(\bar{\phi}_{i}\right\}\right)\) are precisely the expectation value of spins at each site at zero field. i.e. \(\left.F\left(\bar{\phi}_{i}\right\}\right)\) is our GL free energy. Then, we coarse grain the lattice and get a continues theory as describe above.

Comment 1, In reality, the computation is much easier, thanks to some tricks people found. It turns out that if we want the coefficient \(\alpha\), we just need to compute two-point correlation functions. If we want \(g\) we just need four-point correlation functions. Comment 2 , Because the coefficients in the GL free energy are obtained by computing n-point correlation functions, there the loop integral may have singular values and thus require UV cut off: \(\alpha(\Lambda)\) and \(g(\Lambda)\). This is called running couplings in renormalization group, i.e. the control parameter relies on cut offs.

Disordered and ordered phases

If we assume that \(g>0\) and higher order terms don't have any important contributions, we have two phases depending on the sign of \(\alpha\) \(\alpha>0\) is the disordered phase (the paramagnetic phase). Here, we have \(\langle m\rangle=0\) and the state has the same symmetry as the Hamiltonian (or say the GL free energy). \(\alpha<0\) is the ordered phase (the ferromagnetic state). Here, \(|\langle m\rangle| \neq 0,\) but because the system has the \(m \rightarrow-m\) symmetry, if we have a state with \(\langle m\rangle=a,\) there must be another possible state with \(\langle m\rangle=-a\). In the ordered phase, we have (at least) two different possible states, and they have opposite \(\langle m\rangle\). The energy barrier that separates these two states is infinitely high (if the system size is finite). Therefore, once the system fall into one of the two states, it will take infinite long time and infinite energy to flip into the other state. Of course, in reality, the system size can never be infinite, but they are typically large \(\sim 10^{23}\). This means that it will not take infinite long time for use to flip the ground states, but it will take a very long time, too long to observe.

magnetization and frustrated magnets

Consider a lattice of spins. The Heisenberg model has the following Hamiltonian \[ H=J \sum_{\langle i, j\rangle} \vec{S}_{i} \cdot \vec{S}_{j}=J \sum_{\langle i, j\rangle}\left(S_{i}^{x} S_{j}^{x}+S_{i}^{y} S_{j}^{y}+S_{i}^{z} S_{j}^{z}\right) \] where \(\langle i, j\rangle\) stands for nearest neighbors.

  • If \(J<0\), spins want to point in the same direction (ferromagnetic).
  • If \(J>0\), neighboring spins want to point in the opposite direction to minimize the energy (anti-ferromagnetic).

In real materials (which may be anisotropic), the coupling constants in different directions may be different, which can be described by the XYZ model: \[ H=\sum_{\langle i j\rangle}\left(J_{x} S_{i}^{x} S_{j}^{x}+J_{y} S_{i}^{y} S_{j}^{y}+J_{z} S_{i}^{z} S_{j}^{z}\right) \] If \(J_{x}=J_{y},\) we get the XXZ model \[ H=\sum_{\langle i j\rangle}\left(J_{x} S_{i}^{x^{x}} S_{j^{x}}^{x}+J_{x} S_{f}^{y} S_{j}^{y}+J_{z} S_{i}^{z^{z}} S_{j}^{z^{2}}\right) \] If \(J_{x}=J_{y}\) and \(J_{z}=0,\) we get the XY model \[ H=J \sum_{\langle i j\rangle}\left(S_{i}^{x} S_{j}^{x}+S_{i}^{y} S_{j}^{y}\right) \] If \(J_{x}=J_{y}=0\) and \(J_{z} \neq 0\), the model is known as the Ising model \[ H=J \sum_{\langle i j\rangle} S_{i}^{z} S_{j}^{z} \]

1.Frustration and spin liquids

The key question for a spin system is to determine the ground state at \(T=0 K\). For ferromagnetic couplings, the ground state is simple: all spin pointing in the same directions (a ferromagnetic state). For anti-ferromagnetic couplings, it depends on geometry. For example, on a square lattice, the spins can find a way to make all spins happy (all neighboring spins stay in opposite directions). But on a triangle, there is no way to make all the spin happy in an Ising model with \(J_{Z}>0 .\) So the spins are "firust xrated'.". This type of systems are known as frustrated magnets. The reason that frustrated magnets are interesting is because their ground states are not obvious. Therefore, some of these system may have very strange ground state, with exotic properties, such as spin liquids. As \(T\) goes down, in most systems, some ordering will develop (e.g. liquid turns in to solids, paramagnetism turns into ferromagnetism, etc.). There, we will get a phase transition from an disordered state at high temperature to an ordered state at low temperature. The ordered states are often considered as "solids", while the disordered states are sometimes called "liquids”, using the terminology from the solid-liquid transition. In most systems, the low-temperature phase is a "solid" state, in which some type of ordering is developed. However, occasionally, the system may not develop any ordering even at \(T=0 \mathrm{K}\). There, we say that the system remains a "liquid" at \(0 \mathrm{K}\) (a quantum liquid that cannot be frozen by cooling down to zero temperature). For spin systems, if the spin doesn't form any magnetic ordering, the \(T=0 K\) state is call a "spin liquid" or "quantum spin liquid" (quantum here stands for \(T=0 K\), at which the system is governed by quantum physics instead of classical statistics). Typically, in theoretical studies, one maps spins into a strongly-correlated Fermi gas (or boson gas), which is known as the slave fermion (or slave boson) approach. For the slave fermion approach, the fermion may form a insulator or superconductor (which is a gaped spin liquid), semimetal like graphene (which is a gapless spin liquid, known as the algebra spin liquid), or a metal (which is known as a Bose metal and the Fermi surfaces for this Bose metal is known as a Bose surface or a spinon Fermi surface or a spinon surface).

The mapping from spin to fermions/bosons can be rigorously constructed in any dimensions. However, one needs to pay a price; these fermions/bosons have strong interactions, and thus the problem cannot be solved in general. Very typically, one needs to make some approximations, which cannot be fully justified. Because these approximation are not well-controlled, it is not obvious that whether we can trust their predictions or not. Large scale numerical simulations are used to compare with these theories. Recently, some good news are obtained using density matrix renormalization group (DMRG).

In 1D, there is a very straightforward mapping which can map the XY model into a free fermion problem. This is because in \(1 \mathrm{D},\) bosons and fermions are not fundamentally different from each other (one cannot exchange two particles in \(1 \mathrm{D}\) ).

2. Spin-1/2 and hard-core bosons (in any dimensions)

This section describes a baby version of the slave boson approach.

Hard-core bosons are bosons hopping on a lattice, but we assume that there can be at most one boson per site. Hard-core bosons can be realized using ordinary bosons, if we introduce strong (short-range) repulsive interactions between these bosons. When the repulsion is infinitely strong, bosons will not want to stay together, so we will have at most one boson per site (i.e. hard-core bosons).

Hard-core bosons has an obvious connection to spin-1/2. Let's consider one single lattice site. For a single site, the size (dimension) of the Hilbert space of hard core bosons is 2 (occupied \(|1\rangle\) or empty \(|0\rangle\) ). For spin-1/2, the Hilbert space has the same size (spin up | up \(\rangle\) and spin down | down \(\rangle\) ). Because the two Hilbert space has the same size, we can constructe a mapping between them. We map the spin up state for spin- \(1 / 2\) to the occupied state for a hard core boson, and map the spin down state to the empty state. \[ |0\rangle=\mid \text { down }\rangle \quad|1\rangle=| up \rangle \] For hard-core bosons, we can define creation and annihilation operators \(b_{i}\) and \(b_{i}^{\dagger}\), which satisfy \[ b^{\dagger}|0\rangle=|1\rangle \quad b|0\rangle=0 \] \[ b^{\dagger}|1\rangle=0 \quad b|1\rangle=|0\rangle \] For spins, we have the spin rising and lowering operators \(S_{i}{ }^{+}=S_{x}+i S_{y}\) and \(S_{i}^{-}=S_{x}-i S_{y}\), which satisfy \[ \begin{array}{ll} \left.\left.S^{+} \mid \text {down }\right\rangle=\mid \text { up }\right\rangle & \left.S_{i}^{-} \mid \text {down }\right\rangle=0 \\ \left.S^{+} \mid \text {up }\right\rangle=0 & \left.\left.S_{i}^{-} \mid \text {up }\right\rangle=\mid \text { down }\right\rangle \end{array} \] Under this mapping, we find that \(b^{\dagger}=S^{+}\) and \(b=S^{-}\). The particle number operator for hard-core bosons is \(n=b^{\dagger} b\). For spin- \(1 / 2\), this operator is mapped to \(n=S^{+} S^{-}=\left(S_{x}+i S_{y}\right)\left(S_{x}-i S_{y}\right)=S_{z}+1 / 2\) \[ \begin{aligned} n=& S^{+} S^{-} \\=& \left(S_{x}+i S_{y}\right)\left(S_{x}-i S_{y}\right) \\=& S_{x}^{2}+S_{y}^{2}-i\left(S_{x} S_{y}-S_{y} S_{x}\right) \\=& (|\vec{S}|)^{2}-S_{z}^{2}-i\left[S_{x}, S_{y}\right] \\=& s(s+1)-S_{z}^{2}+S_{z} \\=& \frac{1}{2}\left(\frac{1}{2}+1\right)-\frac{1}{4}+S_{z} \\=& S_{z}+1 / 2 \end{aligned} \] It is easy to check that \(\left(S_{z}+1 / 2\right) \mid\) up \(\rangle=1 \times \mid\) up \(\rangle\) and \(\left(S_{z}+1 / 2\right) \mid\) down \(\rangle=0 \times \mid\) down \(\rangle\), which is the same as the particle number operator \(n=b^{\dagger} b\)

One can further verify this mapping by checking the commutation relations \[ [b, b]=\left[b^{\dagger}, b^{\dagger}\right]=0 \] and \[ \left[b, b^{\dagger}\right]=(1-2 n) \] The last fomular can be proved using \[ \left[b, b^{\dagger}\right]|0\rangle=b b^{\dagger}|0\rangle-b^{\dagger} b|0\rangle=|0\rangle=(1-2 n)|0\rangle \]

\[ \left[b, b^{\dagger}\right]|1\rangle=b b^{\dagger}|1\rangle-b^{\dagger} b|1\rangle=-|1\rangle=(1-2 n)|1\rangle \] Note: \(\left[b, b^{\dagger}\right]=(1-2 n)\) is the commutation for hard-core bosons, which is a little bit different from ordinary bosons (without the hard-core constrain). For ordinary bosons, we have \(\left[b, b^{\dagger}\right]=1\).

For a lattice with many different sites. \[ \left[b_{i}, b_{j}^{\dagger}\right]=\left(1-2 n_{i}\right) \delta_{ ij } \] For spins, we have the same commutation relations. \[ \begin{array}{l} {\left[S_{i}^{-}, S_{j}^{-}\right]=\left[S_{i}^{+}, S_{j}^{+}\right]=0} \\ {\left[S_{i}^{+}, S_{j}^{-}\right]=\left[b_{j}^{\dagger}, b_{i}\right]} \end{array} \] proof \[ \begin{aligned} \left[S_{i}^{+}, S_{j}^{-}\right]=&\left[S_{i x}+i S_{i y}, S_{j x}-i S_{j y}\right] \\=&-i\left[S_{i x}, S_{j y}\right]+i\left[S_{i y}, S_{j x}\right] \\=& 2 i\left[S_{i y}, S_{i x}\right] \delta_{i, j} \\=& 2 S_{i z} \delta_{i j} \\=& \left(2 n_{i}-1\right) \delta_{i j} \\=& \left[b_{j}^{\dagger}, b_{i}\right] \end{aligned} \]

XXZ model

Consider the XXZ model \[ H=J_{x} \sum_{\left.k_{i j}\right\rangle}\left(S_{i}^{x} S_{j}^{x}+S_{i}^{y} S_{j}^{y}\right)+J_{z} \sum_{k i j\rangle} S_{i}^{z} S_{j}^{z} \] It is easy to check that \[ \begin{aligned} &S_{i}^{+} S_{j}^{-}+S_{j}^{+} S_{i}^{-} \\=&\left(S_{i}^{x}+i S_{i}^{y}\right)\left(S_{j}^{x}-i S_{j}^{y}\right)+\left(S_{j}^{x}+i S_{j}^{y}\right)\left(S_{i}^{x}-i S_{i}^{y}\right) \\=& 2 S_{i}^{x} S_{j}^{x}+2 S_{i}^{y} S_{j}^{y}+i S_{i}^{y} S_{j}^{x}-i S_{i}^{x} S_{j}^{y}+i S_{j}^{y} S_{i}^{x}-i S_{j}^{x} S_{i}^{y} \\=& 2 S_{i}^{x} S_{j}^{x}+2 S_{i}^{y} S_{j}^{y} \end{aligned} \] Therefore, we have \[ \begin{aligned} H=& \frac{J_{x}}{2} \sum_{k i j>}\left(S_{i}^{+} S_{j}^{-}+S_{j}^{+} S_{i}^{-}\right)+J_{z} \sum_{k i j>}\left(n_{i}-1 / 2\right)\left(n_{j}-1 / 2\right) \\=& \frac{J_{x}}{2} \sum_{k_{i j}>}\left(b_{i}^{\dagger} b_{j}+b_{j}^{\dagger} b_{i}\right)+J_{z} \sum_{k j>} n_{i} n_{j}-J_{z} \sum_{(i j\rangle}\left(n_{i}-1 / 4\right) \\=&\frac{J_{x}}{2} \sum_{k j>}\left(b_{i}^{+} b_{j}^{-}+b_{j}^{+} b_{i}^{-}\right)+J_{z} \sum_{i j>} n_{i} n_{j}-\text { constant } \end{aligned} \] If we ignore the unimportant constant term, we get \[ H=t \sum_{k i j\rangle}\left(b_{i}^{\dagger} b_{j}+b_{j}^{\dagger} b_{i}\right)+V \sum_{k_{i j}>} n_{i} n_{j} \] where \(t=J_{x} / 2\) and \(V=J_{z} .\) So, we mapped the XXZ model into hard-core bosons with nearest-neighbor hoppings \((t)\) and nearest-neighbor repulsions \((V)\).

It is worthwhile to emphasize that these bosons are hard-core bosons, which are bosons with a huge on-site repulsion. Using ordinary bosons, this model becomes \[ H=t \sum_{\left.k_{i j}\right\rangle}\left(a_{i}^{\dagger} a_{j}+a_{j}^{\dagger} a_{i}\right)+V \sum_{\left.k_{i j}\right\rangle} n_{i} n_{j}+U \sum_{i} n_{i}\left(n_{i}-1\right) \] with \(U \rightarrow \infty\). The \(U \rightarrow \infty\) limit prevents two bosons to stay on the same site. This boson model has infinite interactions, which cannot be solved.
