Semantics & history

  1. Many-body Green's function theory: set of techniques that originated in quantum field theory and have then been imported to the many-body problem.
  2. Green's function: mathematical object
  3. Many-body Green's functions are applicable to all sorts of complex/many-body systems: crystals, molecules, atoms, atomic nuclei, ...
  4. Self-consistent Green's functions: many-body Green's functions with dressed propagators (see later).
  5. Many-body Green's functions are not Green's function Monte Carlo.
  6. Late 1950 s, 1960 s: import of ideas from QFT and development of formalism.
  7. \(1970 s \rightarrow\) today: applications and technical developments.

Green's function in maths

  1. In mathematics: Green's function is solution of a inhomogeneous differential equation

\[ [z-L( r )] G\left( r , r ^{\prime} ; z\right)=\delta\left( r - r ^{\prime}\right) \]

  1. Green's function Contains information about eigenstates & eigenvalues of \(L\)

\[ \begin{aligned} G\left( r , r ^{\prime} ; z\right)&= \langle r |\frac{1}{z-L} \left[\sum_{n} |\phi_{n} \rangle \langle\phi_{n} | \right] | r ^{\prime} \rangle \\&=\sum_{n}\langle r |\frac{1}{z-L} | \phi_{n}\rangle\left\langle\phi_{n} \mid r^{\prime}\right\rangle \\&=\sum_{n} \frac{\left\langle r \mid \phi_{n}\right\rangle\left\langle\phi_{n} \mid r ^{\prime}\right\rangle}{z-\lambda_{n}} \end{aligned} \]

more generally \[ G\left( r , r ^{\prime} ; z\right)=\sum_{n} \frac{\phi_{n}( r ) \phi_{n}^{*}\left( r ^{\prime}\right)}{z-\lambda_{n}}+\int d c \frac{\phi_{c}( r ) \phi_{c}^{*}\left( r ^{\prime}\right)}{z-\lambda_{c}} \] where

  • \(\sum_{n} \frac{\phi_{n}(r) \phi_{n}^{*}\left(r^{\prime}\right)}{z-\lambda_{n}}\) is discrete spectrum
  • \(\int d c \frac{\phi_{c}(r) \phi_{c}^{*}\left(r^{\prime}\right)}{z-\lambda_{c}}\) is continuous spectrum

One-body system

Substituting \(L( r ) \rightarrow H ( r ), \lambda \rightarrow E\) with \(H ( r )\) a one-particle Hamiltonian \[ [E- H ( r )] G\left( r , r ^{\prime} ; E\right)=\delta\left( r - r ^{\prime}\right) \] which corresponds to the one-particle Schrödinger equation \[ [E- H ( r )] \psi( r )=0 \] The Green's function in the case reads \[ G\left( r , r ^{\prime} ; z\right)=\sum_{n} \frac{\phi_{n}( r ) \phi_{n}^{*}\left( r ^{\prime}\right)}{z-E_{n}} \]

Second quantization

There is a unique operator \(\hat{\psi}^{\dagger}( x )\) that generates the position basis. It is defined by \[ \begin{aligned} \left| x _{1}\right\rangle &=\hat{\psi}^{\dagger}\left( x _{1}\right)|0\rangle \\ \left| x _{1} x _{2}\right\rangle &=\hat{\psi}^{\dagger}\left( x _{2}\right)\left| x _{1}\right\rangle=\hat{\psi}^{\dagger}\left( x _{2}\right) \hat{\psi}^{\dagger}\left( x _{1}\right)|0\rangle \\ \left| x _{1} \ldots x _{N}\right\rangle &=\hat{\psi}^{\dagger}\left( x _{N}\right)\left| x _{1} \ldots x _{N-1}\right\rangle=\hat{\psi}^{\dagger}\left( x _{N}\right) \ldots \hat{\psi}^{\dagger}\left( x _{1}\right)|0\rangle \end{aligned} \]

Second quantization: Take home message

  • Second quantization is nothing but a convenient way to generate a many-particle basis that automatically has the correct (anti)symmetry.
  • Basis states are created by (anti)-commuting operators with simple (anti)-commutation relations
  • As we will see, second quantization is very convenient in many-body theory as it allows for simple manipulation of perturbative terms without the need to deal with (anti)-symmetrized orbital products
  • The derivation of the Hamiltonian in second quantization is easy in position basis as the Hamiltonian is almost diagonal in this basis

格林函数分母中无穷小的来源

\[ \begin{aligned} A &=\int_{0}^{+\infty} d t f(t) G(t) \\ &=\int_{0}^{+\infty} d t \cdot \int_{-\infty}^{+\infty} d \omega e^{ i \omega t} f(\omega) \cdot \int_{-\infty}^{+\infty} d \omega^{\prime} e^{ i \omega^{\prime} t} G\left(\omega^{\prime}\right) \\ &=\int_{-\infty}^{+\infty} d \omega f(\omega) \cdot \int_{-\infty}^{+\infty} d \omega^{\prime} G\left(\omega^{\prime}\right) \cdot \int_{0}^{+\infty} d t e^{ i \left(\omega+\omega^{\prime}\right) t}\\ &=\int_{-\infty}^{+\infty} d \omega f(\omega) \cdot \int_{-\infty}^{+\infty} d \omega^{\prime} G\left(\omega^{\prime}\right) \frac{-1}{ i \left(\omega+\omega^{\prime}+ i 0^{+}\right)} \end{aligned} \]

\[ e^{ i \left(\omega+\omega^{\prime}\right) t}=\cos \left[\left(\omega+\omega^{\prime}\right) t\right]+ i \sin \left[\left(\omega+\omega^{\prime}\right) t\right] \] 是一个振荡的三角函数函数，对t的积分是不收玫的。原来的积分顺序是先对 \(\omega\) 和 \(\omega^{\prime}\) 积分，再对 \(t\) 积分，但是我们想交换积分 的顺序，想先对 \(t\) 积分。但是交换顺序后发现，对 \(t\) 的积分不收敘，但早就已知整个积分是收敘的，所以直接交换积分是不被允许的，它会导致发散的错误的结果。是也是一个不一致收玫不能交换积分顺序的例子。

但是我们可以对其改造，使得其满足一致收玫，而且改造后的积分与原来的积分的值是相等的。这样，经过改造我们就可以交换积分顺序，从而把t给积掉。

定理

先不加证明地说明一个结论。如果积分 \[ \int_{0}^{+\infty} d t f(t) \] 收敛，那么将有下面的结论 \[ \int_{0}^{+\infty} d t f(t)=\lim _{\eta \rightarrow 0^{+}} \int_{0}^{+\infty} d t f(t) e^{-\eta t} \]

加入这一因子改造的本质是加入一个衰减因子，当 \(\eta \rightarrow 0^{+}\) 时, 这个哀减因子decay地非常慢，而原来的积分是收敘的，也就是在无穷远处的值应该趋近于零，类似于一个波包，所以可以认为在波包的值不为零时，这个decay对于它的作用是可以忽略的。将 \(t\) 的积分算出 \[ \lim _{\eta \rightarrow 0^{+}} \int_{0}^{+\infty} d t e^{ i \left(\omega+\omega^{\prime}\right) t-\eta t}=\lim _{\eta \rightarrow 0^{+}} \frac{-1}{ i \left(\omega+\omega^{\prime}+ i \eta\right)} \] 般就简记为 \[ \lim _{\eta \rightarrow 0^{+}} \frac{-1}{ i \left(\omega+\omega^{\prime}+ i \eta\right)} \equiv \frac{-1}{ i \left(\omega+\omega^{\prime}+ i 0^{+}\right)} \] \(i 0^{+}\) 在中间计算过程中的出现原因有两点： 1. 数学上的事实： \(i 0^{+}\) 的出现是正确的，不会使得原积分的值发生改变。 2. 操作上的便利性： \(i 0^{+}\) 的出现能够使得我们交换积分的顺序，从而先把 \(t\) 积分掉。
