多体系统

虽然关于单个电子、原子的基础物理都比较清楚了，但在研究多体系统的过程中，我们需要应对指数墙问题：系统的自由度随系统尺寸指数增加，无法直接求解薛定谔方程。

  1. 一种可能的解决方案是单电子近似，将多体效应约化为各个电子感受到的势场，将多体相互作用看作微扰，相关的理论有能带理论、密度泛函理论（DFT）[4]。
  2. 但对于强关联系统，系统中各电子之间的相互. 作用起着决定性作用，在此情况下，能带理论失效，我们很难从单电子近似角 度入手来研究体系的性质，大家特别关注的高温超导模型就属于这一类。

强关联领域发展出的数值方法

  1. 量子蒙特卡洛方法 (Quantum Monte Carlo）
  2. 密度矩阵重正化方法(Density Matrix Renormalization Group)
  3. 密度矩阵嵌套理论(Density Matrix Embedding Theory)
  4. 张量网络态（Tensor Network States, TNS)

DMRG表象与轨道优化

DMRG常见的表象有

  1. 实空间表象
  2. 动量空间表象
  3. 任何给定幺正矩阵，都可以得到一个新表象

同一个态在不同的表象中具有不同的纠缠熵

那么寻找最合适的表象，降低波函数的纠缠熵

基于这样的需求，我们发展了一种轨道优化的DMRG方法，

  • 在计算过程中 不断地变换表象，以便降低纠缠熵。

例子：对于赫伯德模型

  1. 当U较小时，可以认为 是在自由费米子模型上作微扰。动量空间DMRG与很多其它数值方法都能比较成功。
  2. 在 \(U\) 较大的极限下，同位排斥能使得电子出现反铁磁序，相对非常局域。 实表象即最佳表象。
  3. 更多情况下，我们关心的是中间大小的U \(=4 \sim 8\) ，带宽8。

张量网络的典型算法

  1. 块时间演化算法(Time Evolving Block Decimation, TEBD)
  2. 高阶奇异值分解重正化群 (High-Order Tensor Renormalization Group, HOTRG)

投影纠缠对态PEPS

（a）矩阵乘积态MPS的张量表示，每个色点是一个三阶张量，通过两个虚拟指标与相邻张量相连，另一个为物理指标。

（b）投影纠缠对态PEPS的张量表示。每个 蓝色的点代表一个五阶张量，通过四个虚拟指标与相邻张量相连。

投影纠缠对态（Projected Entangled Pair State, PEPS）是MPS在二维的推广。我们以正方晶格为例

  1. 在二维的正方晶格上，每个格点上都定义了一个五阶张量
  2. 其中一个指标代表 物理自由度
  3. 另外四个指标分别与邻近张量共享。
  4. 每个虚拟指标都是一个辅助 自由度，此张量可以看作是从四个辅助指标到真实指标的一个映射。
  5. PEPS态不仅可以用于表示有限大小的系统波函数，也可以用于表示具有平移不变的无 穷大系统，即iPEPS

时间演化算法 TEBD

时间演化算法（TEBD）是一种用于求哈密顿量基态或实时演化的方法。 它相当于矩阵用幂迭代（power iteration）方法求矩阵的本征态。 \[ \left|\Psi_{G}\right\rangle=\lim _{\beta \rightarrow \infty} e^{-\beta H}\left|\Psi_{0}\right\rangle \] 其中

  • \(\left|\Psi_{0}\right\rangle\) 是一个任意的初态

经过 \(\beta \rightarrow \infty\) 时间演化后，收敛到基态

  1. 对于一维模型， \(\left|\Psi_{0}\right\rangle\) 由MPS表示
  2. 对于二维模型， \(\left|\Psi_{0}\right\rangle\) 则由PEPS表示

重正化TRG

Levin在2007年第一次提出了张量网络重正化(Tensor Renormalization Group, \(TRG )\) 方法[14]，并且用于计算经典统计模型，这是张量方法的里程碑。

  1. 在考虑了环境对系统的影响逐渐发展出二次张量网络重正化方法（Second Renormalization Group, SRG）[38],
  2. 为了解决三维经典问题
    • 相应地又发展出高阶奇异值重正化方法（HOTRG）[2]
