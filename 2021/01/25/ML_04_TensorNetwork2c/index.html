1 量子多体物理的指数墙与面积律

量子多体物理

量子多体物理一直是凝聚态物理中一座魏峨的高山。此处峰林俊秀，有

  1. 高温超导 [5]，
  2. 量子自旋液体 [6]，
  3. 量子相变 [7]，
  4. 分数霍尔效应 [8]

等新奇物理齐聚。然而此处也云遮雾绕，像

  1. 赝能隙 [9]，
  2. 奇异金属相 [5]，
  3. 自旋电荷分离 [10]，
  4. 解禁闭量子临界 [11]

等等让人迷惑的现象层出不穷。当然，此处亦是十分的陡肖险峻，

  • 理论还没有统一的范式，
  • 平均场近似失效，
  • 严格对角化计算代价过大，
  • 量子蒙卡会遭遇本质的负符号问题。

正应了王安石那句：

“世之奇伟，瑰怪，非常之观，常在于险远，而人之所罕至焉，故非有志者不能至也。”

指数墙难题

量子多体理论研究的起点往往是一个与实验现象相联系的简化类型，例如可能是高温超导母体的莫特绝缘体 [12] 和与之联系的 Hubbard 类型， \[ \hat{H}=-t \sum_{\sigma,<i, j\rangle}\left(\hat{c}_{i, \sigma}^{\dagger} \hat{c}_{i, \sigma}^{\dagger}+\text { h.c. }\right)+U \sum_{i} \hat{n}_{i \uparrow} \hat{n}_{i \downarrow} \] 这里所谓的简化仅仅是相对于真实材料来说的，在 Hubbard 类型中

  1. 我们忽略了所有与原子核相关的相互作用，
  2. 我们也忽略了不同价带间电子的相互作用，
  3. 我们仅考虑巡游电子与同一个局域轨道上的电子的相互作用。

然而，即便是这样的简化类型也完全不意味着计算上的简单。Hubbard 类型这样的量子多体类型描述的是一个由多粒子构成的系统，且粒子之间的相互作用无法忽略也无法简单地运用平均场进行单体近似。对这样的系统，如果我们知道单个粒子的波函数\(|\varphi(x)\rangle,\) 那么系统的一般波函数 \(|\Psi( x )\rangle\) 应该表示为, \[ |\Psi( x )\rangle=\sum_{i_{1}, i_{2}, \cdots, i_{N}} C_{i_{1}, i_{2}, \cdots, i_{N}}\left|\varphi_{1}\left(x_{1}\right)\right\rangle \otimes\left|\varphi_{2}\left(x_{2}\right)\right\rangle \otimes \cdots \otimes\left|\varphi_{N}\left(x_{N}\right)\right\rangle \] 这里 \(C\) 是叠加系数。注意到，如果单粒子波函数 \(|\varphi(x)\rangle\) 具有 \(d\) 种状态的话, 那么整个系统的希尔伯特空间的维度将达到 \(d^{N}\) 。如果我们假设每个单粒子具有10 个状态的话，那么 80 个这样的单粒子的基矢的规模就相当于已知宇宙中所有原子的总数了。因此这种指数增长的自由度使得严格对角哈密顿量根本无从实现。这个困难称为量子多体物理中的“指数墙”难题 [13]。

峰回路转

然而“指数墙”难题并不意味着量子多体物理研究完全无法推动。指数墙”强调的是一种没有任何先验知识下的最一般情况的波函数，但事实上希尔伯特空间中的绝大多数基矢被证明是我们完全不用去关心的，因为可以证明任意一个局域哈密顿量经过多项式时间的演化能够遍历的空间只是希尔伯特空间中极小的一部分 [14]。我们关心的真实物理类型往往比最一般的情况要特殊很多。

  1. 在最幸运的情况下，我们甚至可以利用天才的物理直觉和敏锐的洞察直接猜出波函数的形式。一个很好的例子是分数量子霍尔效应中的 Laughling 波函数 [15]。
  2. 在不那么幸运的情况下，我们也可以利用物理类型的具体形式, 对称性，局域性等对目标波函数进行非常有效的近似。例如对于基态有能隙的局域哈密顿量, 可以证明其基态纠缠熵满足“面积律” [16]。

Fig 1.1 The system is coupled to the environment through boundaries. The entanglement is dominated by local short-range entanglement.

我们将整个系统分为系统和环境两部分。定义系统的约化密度矩阵 \[ \rho_{s y s}=\operatorname{Tr}_{ env }(|\Psi\rangle\langle\Psi|) \] 此处 \(\rho\) 为整个系统的密度矩阵。则两部分的纠缠熵可定义为 \[ S=-\operatorname{Tr}\left(\rho_{ sys } \ln \rho_{ sys }\right) \] 纠缠熵的面积律指出，对基态有能隙的局域哈密顿量，其纠缠熵与系统与环境交界区域的尺度正比, 即 \[ S \sim L^{d-1} \] - \(L\) 是系统尺寸, - \(d\) 是系统的维度。

  1. 对于一维系统，满足面积律的纠缠熵是常数;
  2. 对于二维系统，纠缠熵与系统尺寸是线性关系。

基态波函数满足面积律意味着系统的纠缠是由局域的短程纠缠所主导的。这启发我们在对波函数做近似的时候可以优先考虑近邻的局域纠缠，丢弃不必要的长程信息。事实上，上述观察孕育出了基于局域低秩展开的波函数的张量网络态表示方法。

2 张量表示与张量网络

张量网络是由多个张量通过缩并掉相同指标连接起来的一个网络，通常张量网络的代数表示涉及复杂繁銷的指标求和与求积。为了直观，我们首先引入张量网络的图示法。

如图 1.2 所示，我们用

  • 一个圆圈代表一个独立的代数对象
  • 圆圈上伸出来的每条腿代表代数对象的一个自由指标

张量求和

将两个代数对象的相同指标连接起来表示我们要对这个指标进行求和。

  • 图 1.3 (a) 表示了两个矩阵 \(A_{i j}\) 和 \(B_{j k}\) 对共同指标 \(j\) 的缩并 \(\sum_{j} A_{i j} B_{j k}\) 有了上述定义后，我们就可以把复杂的代数运算用直观的图示展示出来。

张量网络态

多体波函数 \[ |\Psi( x )\rangle=\sum_{i_{1}, i_{2}, \cdots, i_{N}} C_{i_{1}, i_{2}, \cdots, i_{N}}\left|\varphi_{1}\left(x_{1}\right)\right\rangle \otimes\left|\varphi_{2}\left(x_{2}\right)\right\rangle \otimes \cdots \otimes\left|\varphi_{N}\left(x_{N}\right)\right\rangle \] 所有系数 \(C_{i_{1}, i_{2}, \cdots, i_{N}}\) 可以被理解为总共拥有 \(d^{N}\) 个参数的 \(N\) 阶张量 \(C\) 。

同时，如果我们的量子多体波函数满足“面积律”, 那么确定 \(C\) 中的每个系数的值实际上将变成一件非常低效和不必要的事。

如图 1.4，所谓张量网络态即是根据对波函数的纠缠特性的准确分析，利用一系列较低阶的张量的互相缩并组成的网络来近似 \(C\) 这样的高阶张量，在不过多地降低波函数 \(\Psi\) 的精度的前提下，将参数个数从指数多压缩到了实际计算可容忍的范围。

从物理的角度来说，利用只有局域连接的低秩张量组成的网络来近似一个高阶张量, 等价于忽略掉高阶张量非局域指标之间的一些长程关联。

可以说，张量网络态表示的合理性是与量子态的面积律密切相关的。

常见的张量网络态有

  1. 矩阵乘积态 (MPS)[17],
  2. 投影纠细对态 (PEPS) [18],
  3. 树状张量网络态（TTN）[19]，
  4. 多尺度纠缠重整化假设（MERA）[20]

张量网络算法

张量网络算法是在张量网络态的基础之上发展处理解决真实物理问题的计算方法。一般来说，利用张量网络态计算人们关心的物理量，我们还需要解决两个主要问题：

  1. 第一个问题是如何找到正确的张量值。除开极少数特殊情况，我们并不能从哈密顿量或者波函数中直接严格地把张量网络中的张量构造出来。张量的初值. 往往是随机给定的。这时候我们需要首先利用合适地算法更新各个张量以便整个张量网络逐渐地收敛到真实的波函数上去。
  2. 第二个问题是找到正确的张量值之后, 如何准确地收缩掉整个张量网络。从张量网络态中计算物理量往往需要我们将整个张量网络收缩掉，但对像 PEPS 这样二维或更高维的张量网络态来说，严格收缩也是需要调用指数多的计算资源的。所以我们在收缩张量网络时需要巧妙的近似算法。

张量网络的算法主要可以分为三大类：

  1. 第一类叫做投影类算法，其中既有用来寻找基态的

    • 时间演化块消去算法(TEBD)
    • 时间依赖变分原则算法( TDVP )

    也有用来收缩张量网络的时间

    • 角转移矩阵算法（CTM） [23]
    • 边界 MPS 算法等 [24]

  2. 第二类叫做粗粒化类算法，这类算法主要用来收缩二维及更高维度的张量 网络。包括

    • 张量重整化群算法 (TRG) [25],
    • 二次张量重整化群算法 (SRG) [26],
    • 高阶张量重整化群算法（HOTRG）[27]，
    • 张量纠缠筛去重整化算法（TEFR） [28]

  3. 第三类算法叫做变分类算法，包括用来寻找基态的

    • 密度矩阵重整化群方法(DMRG) [29]，
    • 变分 PEPS [24]
    • 变分 MERA [30]

    变分类算法与前两类互有交叉。比如用来收缩高维张量的

    • 张量网络重整化算法 (TNR) [31]
    • 圈张量网络重整化方法（loop-TNR）[32]

3 当量子物理遇上机器学习

类似于量子多体物理算法，机器学习算法可能也只需在构型空间中的一个极小角落工作。其实机器学习和量子物理处理的问题是类似的。即根据数据的先验知识，利用较少的多项式复杂度的参数来拟合一个具有指数多种可能的函数构型。

  • 量子物理中一个很好的例子是张量网络态，它利用了系统局域性和基态波函数的纠缠熵往往满足面积律这个先验知识，用一组张量的乘积来表示波函数系数，非常高效地抓住了波函数的主要信息。
  • 机器学习中一个很好的例子是卷积神经网络, 利用卷积操作将特征的平移不变性编码进了类型中, 利用深层网络将图片多个层级上的特征各种编码，深层卷积神经网络在图片识别任务中的表现非常突出。

认识到两个领域问题的相似性，研究人员开始尝试互相借鉴两者的经验，引入机器学习算法来解决量子物理问题。

  1. 这个领域的开创性工作是利用监督学习的限制玻尔兹曼机识别不同的物理相 [37]。
  2. 之后无监督版本的相识别机器学习算法 [38] 也很快被提出。

但对机器学习类型来说，真正困难的是从构型开始 (而不是从事先给出拓扑特征或拓扑不变量开始）识别一个非局域且对单个变量的值敏感的拓扑相。这一块目前还处于探索阶段。

限制玻尔兹曼机

在利用机器学习类型作为量子波函数变分假设的问题中，一个相当有代表性的工作是利用限制玻尔兹曼机做为量子蒙特卡罗的变分假设，在无阻挫量子自旋系统中得到了非常好的变分基态能量。[39] 这种变分假设可以看作是 Jastrows的替代。但是它更加灵活，因为它以一种更经济的方式对多体关联进行了编码。

同时，另一些研究也发现限制玻尔兹曼机的结构可以跟张量网络态联系起来 [40, 41]，文献 [42, 43] 分别从量子纠缠和计算复杂性的角度分析了它们的表达能力。从这些工作中，我们可以閒到神经网络状态对于描述高度纠细的量子态和具有长程相互作用的类型是有利的。

重整化群

重整化群是一个重要的物理学概念。它是一种在降低数据维数的同时保留有效信息的方法。重整化群与深度学习也有着非常有趣的联系。

  • 这种联系一方面为用重整化群解决物理问题提供了深度学习的工具
  • 另一方面也可以为深度学习提供理论上的理解。

文献导览

  1. 第一个引起广泛讨论的工作是 [44]，它提出了一中具有 MERA 结构启发的生成型贝叶斯网络。
  2. 之后文献 [45] 首次给出了一个实空间重整化群与深层玻尔兹曼机的严格映射。
  3. 文献 [46] 将动量空间重整化群与主成分分析联系了起来。
  4. 最后，文献 [47] 利用机器学习中的 Real-NVP 类型提出了一种类似于 MERA 结构的叫做神经网络重整化群的变分重整化群。
  5. 这种神经网络重整化群提供了一种保证信息守恒的重整化过程，这一特性可以应用于全息映射中 [48]。

另一个有意思的应用是利用机器学习来加速量子蒙卡的抽样速度 [49, 50]。这些工作的基本想法是：利用蒙卡构型训练出一个代理函数，再利用这个代理函数生成新的构型。由于代理函数与原构型的关联是高度非局域的, 所以通过代理函数的一步更新可以相当于翻转局域自旋的几十上百步更新。这个方法的短板是并不是对所有系统，我们都能够快速训练出一个接近目标问题势能面的代理函数。所以实际效率是问题依赖的。

基于张量网络的机器学习算法

在机器学习的方法被应用于量子物理问题的同时，量子多体物理中的经典类型也在被引入到不同的机器学习问题中去。张量网络便是这样的一个例子。

除了系统的局域性等考虑之外，我们认为发展基于张量网络的机器学习类型至少有两点动机。

  1. 第一, 张量网络的表示和张量网络的算法为判别学习和生成学习提供了具有更强表达能力的类型。事实上，如果我们试图把概率图类型拓展到量子版本同时还希望保证概率的正定性的话，张量网络的结构可以自然地涌现出来 [51,52]\(。\)
  2. 第二，张量网络这种形式是打开量子机器学习的门户，因为量子电路在形式上等价于一种特殊的张量网络。张量网络态为量子电路提供了架构设计和参数初始化的方法 [53]。

监督学习

  1. 第一个利用张量网络处理监督判别型学习的工作是 [54]，它利用 MPS 模型处理 MNIST 数据的分类问题并达到了相当不错的准确度。
  2. 随后基于树状张量网络的分类模型被提出 [3]。
  3. Miles Stoudenmire 通过首先使用树张量网络执行无监. 督特征提取进一步提高了 MPS 类型的分类准确度 [55]。
  4. 更一般的张量网络类型也在文献 [2] 被讨论。上述类型我们会在第 3 章详细介绍。

在无监督生成型学习问题中

  1. 第一个代表性工作同样是将 MPS 类型应用于二进制化的 MNIST 类型中 [4]。MPS 在某些方面发展出了传统机器学习生成型类型不具备的优势。
  2. 之后的树状张量网络生成型类型继承了 MPS 类型的这些优势，同时可以更好地实现长程关联的建模 [56]。相比判别型类型，生成型类型难度更大，我们认为正是在这类问题中，张量网络强大的表达能力的优势才能被更多的释放出来。这部分的细节我们将放在第 5 章。

张量网络中的术语与机器学习中的术语存在着对应关系。

到今天，张量网络也同样吸引了很多机器学习领域研究人员的注意 [57]。他们开始注意到机器学习中的术语与张量网络中的术语的对应关系，张量网络的技术与机器学习的降维算法存在着紧密联系 [1]。同时，张量网络也可以用来分析机器学习类型的表达能力 [58-60]。

不过，张量网络依然还在等待一个类似 AlexNet 一样的时刻，即张量网络方法能够以比较大的差距战胜传统的机器学习算法。随着越来越多的技术与结果的积累。至少在某一个特定的应用领域下，这样的时候很有可能在未来几年之内发生。

张量网络在监督学习中的应用

在本章，我们会对将张量网络应用于监督学习的工作做一个简要的回顾。

最近十年, 物理学和机器学习领域开始认识到张量方法的力量。

  • 在机器学习中，张量分解可以用来解决非凸优化任务 [141, 142] 并在许多其他重要问题上取得进展 [143-145]，
  • 而在物理学方面，张量网络是量子多体物理中一种非常强大的工具，这种方法通过将多体量子物理中产生的维度指数大的多体量子波函数分解为张量网络来处理，已经取得了很大的进展 [146]。
    • 最成功的几类张量网络可以通过低阶张量近似来避免维数灾难，同时还可以通过对这些张量的收缩精确地再现多体波函数 [147]。

图模型与张量网络对偶

在正式介绍张量网络监督学习算法之前，我们先略施笔墨介绍一下张量网 络与机器学礼中的图类型的关系。

  • 考虑一组离散随机变量 \(X =\left\{X_{1}, \ldots, X_{N}\right\},\)
  • 随机变量的构刑 \(x =\left(x_{1}, \ldots, x_{N}\right)\)
  • 一组数据集中样本 $D ={ d {1}, , d {|D|}} $

这组样本背后暗含的概率分布 \(p( x )\) 可以通过最大化对数似然度来估计 \[ L =\sum_{i=1}^{|D|} \log p\left( d _{i}\right) \] 其中一种常见的对 \(p\) 建模的方式是图类型 [150]，概括地说，图类型就是一种用图来表示的概率分布的因式分解。考虑一个图 \[ G=(V, E) \] 其中

  • $ V$ 是顶点的集合
  • \(E\) 是边的集合
  • \(\operatorname{cl}(G)\) 是图中因式分解的最大集团的集合。

一个无向图或者一个马尔科夫随机场定义的概率分布的因式分解写作 \[ p( X = x )=\frac{1}{Z} \prod_{C \in cl (G)} \phi_{C}\left( x _{C}\right) \] 其中

  • \(x _{C}\) 是在 \(C\) 十的进机变量的值
  • \(\phi_{C}\) 叫做势函数，他说一个正定的函数
  • \(Z\) 是概率分布的归一化因子，也称为概率分布的配分函数。

图类型也可以转化为一个因子图 [151]，所谓因子图是一种由变量和因子组成的二元图。 图类型到因子图的转化只需把每个最大集团定义为一个因子苹点 \(f_{s}\) (如图 3.1)。

此时概率分在的因式分解写作, \[ p( X = x )=\frac{1}{Z} \prod_{s} f_{s}\left( x _{c}\right) \] 在因子图上我们可以利用较戌熟的信念传递算法和 sum-product 算法来对进行统计推

张量网络在无监督生成型学习中的应用

生成型建类 [181]，要求从训练数据中学习数据自身的联合概率分布，并根据训练得到的联合概率分布生成新样本。

  • 它是无监督学习的核心问题之一，被广泛应用于寻找数据的更好更有效的表示或从初始数据生成新样本的现实任务中。
  • 与仅仅捕获数据判别标签的条件概率的判别型建类相比，生成型建类试图捕获的是数据集的整个联合概率分布，因此是一个更加困难的任务 [182]

经过最近几十年的发展，机器学习领域已经提出来了非常多种的生成型类 型，在这些类型中非常庞大的一支叫做概率图类模型（PGM），常见的概率图类型包括

  • 贝叶斯网络 [183]
  • 隐形马尔科夫类型 [184]
  • 限制玻尔兹曼机 [108] 等。

同时也存在着基于神经网络的生成型类型，比如说

  • 深度信念网络 [185]，
  • 变分自动编码器（VAE）[186],
  • realNVP [187],
  • PixelCNN [188, 189] 等。

另一种值得一提的生成型类型是最近几年非常热门的生成对抗网络（GAN） [190]。

  1. 一种就是玻尔兹曼机 [64]，它用经典统计物理学的玻尔兹曼分布来诱导出数据的联合概率分布：
  2. 另一类叫做玻恩机，它用量子量子物理学中的玻恩诠释来表示数据的联合概率分布，即将联合概率分布理解为一个波函数的类的平方 \([34,191 ] 。\)
    • 玻恩机中一类常用的类型是基于张量网络的玻恩机，换言之，也就是用张量网络来表达一个多体波函数的玻恩机。
