摘要

量子自旋液体为什么重要？

1987 年，Anderson提出通过渗杂可变高温超导的Mott 绝缘体La \(_{2} CuO _{4}\) 的基态实际是量子自旋液体 [3]，开辟了人们研究 高温超导机制的新思路 [4-8]。

量子自旋液体是怎么诞生的？

1973 年，P. W. Anderson 在研究三角晶格中反铁磁海森堡相互作用的阻挫体系时提出了RVB（resonating valence bonds, 共振价键）模型描述该无序态， 并首次提出了量子自旋液体的概念 [1]。

量子液体是什么？

量子自旋液体是一种由于强量子涨落的存在，使得体系中磁有序被抑制，即使到绝对零度也无长程磁序建立的新奇量子态。

Kitaev蜂窝模型和量子自旋液体模型是什么关系？

Kitaev 量子自旋液体是量子自旋液体模型中重要的一员

  • 该模型具有拓扑序，存在 非阿贝尔任意子激发。通过对任意子的操作，可以实现量子计算 [11-19]。
  • 一般量子自旋液体模型无法精确解，但是Kitaev 量子自旋液体有精确解

解释一下量子自旋液体模型中重要的概念“阻挫(frustration)”

在自然界中，物质会自然趋于能量最低态使系统达到稳定。对于一般磁性材料体系而言，电子的自旋在低温下会由于热涨落减弱行成有序排列以达到系统自由能最小。

  • 但是因为阻挫的存在，使得系统在 极低温下也无法建立长程磁有序，并存在很强的量子涨落。
  • 何为阻挫？阻挫是指各种能量标度相近的相互作用间产生竞争，使得组成体系 的各个单元不能同时进入能量最低态 [20]。

“阻挫(frustration)”形成的原因有哪些？

  1. 几何阻挫——代表：RVB 模型
  2. 由于不同方向的键在自旋取向上不同，使得通一个格点上相互不对易，产生阻挫。——代表：Kitaev 模型

toric code 模型实际上是Kitaev 模型的一个低能有效哈密顿量

背景

自旋之间的相互作用及由此而生的自旋序 (磁相变) 似乎是凝聚态物理学的永恒主题之一。 如果我们山易入难，从经典自旋系统开始，

  1. 伊辛所完成的坊次老师布置的作业：维自旋链是否存在相变？应是自旋系统 “格物致知”历史之升始。
  2. 高对称点阵中，如果只考虑是近邻相互作用，则一维 Ising 自旋链不存在有序态
  3. 二维点阵存在有限的相变温度并由 Onsager 给出严格解、杨振宁给出相变点附近磁矩的严格解..
  4. 三维点阵中 Ising 模型到目前还没有严格解，但也不妨碍近年的数值模拟对三维模型的深刻物理认识。

这些历史的黄金烙印奠定了经典磁学的基石。当海森堡提出自旋相互作用的量子理论后，这些烙印也可以看成是现代磁学和自旋电子学的基石。自然，有人会将伊辛模型推广到空间上自由度更 多的体系，如

  • XY 模型、
  • Potts 模型
  • 海森堡模型的自旋可以沿任意空间方向排列

物理学界众多好事之士茶余饭后总会提出一些新的说辞。 比如说:

1.二维、三维甚至更高维空间的自旋系统是否就一定得有序?
2.有无极端例外，即使温度降到零度也不会长程有序?

事实上，我们早就知道很多人在研究这个极端。例如，二维各向同性空间中海森堡自旋系统有无相变?这种问题时至今日仍然未得圆满。逻辑上, 至少有两类情况可能逼近这个极端。

  1. 一是空间对称性下降
  2. 一是交互作用不限于最近邻。

凝聚态物理有个名词定义这种现象，叫“自旋失措 ( 阻抨） spin frustration”。
