1 基本语法

用 Clear 清除 x 和 y 的值：

Clear[x, y]

清除在当前 Wolfram 系统会话中的所有定义：

Clear["Global`*"]

定义函数 f

f[x_] := x^2

清除一个函数定义：

Clear[f]

  • 共轭
  • 转置

mathematica的向量是不能转置的

v = {3 + 4 I, 5, I};
v // MatrixForm
ConjugateTranspose[m] // MatrixForm

需要把向量转化为(1$$3)矩阵，才能转置：

v={{ 3 + 4 I },{ 5},{ I}}
v // MatrixForm
ConjugateTranspose[v] // MatrixForm

把v化为矩阵的方式有两种

v={{ 3 + 4 I },{ 5},{ I}}

w = {3 + 4 I, 5, I};
w={w}
  • 行向量与列向量的乘积，是一个数字
  • 列向量与行向量的乘积，是一个矩阵
v = {{3 + 4 I}, {5}, {I}}
ConjugateTranspose[v].v // MatrixForm
v. ConjugateTranspose[v] // MatrixForm

高等理论计算

姚期智教授给清华大学新入学研究生开的一门课，课程内容：

  • 经典计算复杂性
    1. NP 完全性，
    2. 多项式空间复杂性，
    3. 对数空间复杂性，
    4. 交互式证明系统，
    5. 随机复杂性，
    6. 去随机化，
    7. 概率检验证明系统，
    8. 电路复杂性，
    9. 通信复杂性，
    10. 判定树复杂性等。
  • 在量子计算复杂性方面将包括：
    1. 量子计算模型，
    2. 量子电路，
    3. 量子 Fourier 变换算法，
    4. Shor 算法，
    5. Grover 量子搜索算法，
    6. 量子纠错码，
    7. 冯诺依曼熵等。
  • 参考教材
    1. Complexity Theory: A Modern Approach, Sanjeev Arora and Boaz Barak
    2. Lecture nots: Quantum Computation, instructor Umesh Vazirani

高等理论计算机科学

旨为介绍理论计算机科学的一些前沿的主题：

  1. 近似算法、
  2. 随机算法、
  3. 计算复杂性、
  4. 优化理论、
  5. 机器学习理论

并就大家感兴趣的问题进行深入的探讨；通过专题探讨，帮助学生确立今后研究的方向，明确研究的目标。

课程将快速回顾关于凸优化和机器学习的基础知识：

线性/逻辑回归，正则化，牛顿法，随机梯度下降（不同步，方差减少法），生成与歧视，方差与偏差。现有的机器学习和预测算法方面：k-nn，SVM，内核技巧，聚类，Adaboost，梯度提升，随机森林。在线学习和顺序预测方面：多臂老虎机问题，通用投资组合，乘法加权法，在线凸优化，基本时间序列。基于线性代数的学习算法方面：SVD，主成分分析（PCA），独立分量分析（ICA），非负矩阵分解（NMF），主题建模，矩阵完成，字典学习，张量法，谱聚类。
