1 基本语法

用 Clear 清除 x 和 y 的值：

Clear[x, y]

清除在当前 Wolfram 系统会话中的所有定义：

Clear["Global`*"]

定义函数 f

f[x_] := x^2

清除一个函数定义：

Clear[f]

  • 转置

mathematica的向量是不能转置的

v = {3 + 4 I, 5, I};
v // MatrixForm
ConjugateTranspose[m] // MatrixForm

需要把向量转化为(1$$3)矩阵，才能转置：

v={{ 3 + 4 I },{ 5},{ I}}
v // MatrixForm
ConjugateTranspose[v] // MatrixForm

把v化为矩阵的方式有两种

v={{ 3 + 4 I },{ 5},{ I}}

w = {3 + 4 I, 5, I};
w={w}
  • 行向量与列向量的乘积，是一个数字
  • 列向量与行向量的乘积，是一个矩阵
v = {{3 + 4 I}, {5}, {I}}
ConjugateTranspose[v].v // MatrixForm
v. ConjugateTranspose[v] // MatrixForm
