狄拉克方程

狄拉克方程由 Paul A.M. Dirac 在 1928 年提出，用以描述自旋为 \(1 / 2\) 的粒子的相对论量子行为 \(^{[10]} \circ\) 自由粒子的不含时狄拉克方程写作， \[ \begin{aligned} H _{D} \Psi&=\left(c \alpha \cdot p+m c^{2} \beta\right) \Psi\\ &=E \Psi \end{aligned} \] 其中

  • \(m\) 是粒子的静止质量
  • \(c\) 是光速
  • \(p=-i \hbar \nabla\) 是动量算符。

系数的反对易关系

系数 \(\alpha_{i}(i=x, y, z)\) 和 \(\beta\) 需要满足 \[ \alpha_{i}^{2}=\beta^{2}=1\\ \left\{\alpha_{i}\beta\right\}=0\\ \left\{\alpha_{i}, \alpha_{j}\right\}=2 \delta_{i j} \] 要满足上述条件，这些系数至少是 \(2 \times 2\) 矩阵。泡利矩阵 \(\sigma_{i}(i=x, y, z)\) 表示为 \[ \sigma_{x}=\left(\begin{array}{cc} 0 & 1 \\ 1 & 0 \end{array}\right), \sigma_{y}=\left(\begin{array}{cc} 0 & -i \\ i & 0 \end{array}\right), \sigma_{z}=\left(\begin{array}{cc} 1 & 0 \\ 0 & -1 \end{array}\right) \] 它们满足反对易关系。即 \(\left\{\sigma_{i}, \sigma_{j}\right\}=2 \delta_{i j \circ}\)

  1. 对于一维空间的狄拉克方程， \(\alpha_{i}\) 和 \(\beta\) 可取数值

\[ \alpha_{x}=\sigma_{x}, \alpha_{y}=\alpha_{z}=0, \beta=\sigma_{z} \] 2. 对于二维空间的狄拉克方程， \(\alpha_{i}\) 和 \(\beta\) 可取

\[ \alpha_{x}=\sigma_{x}, \alpha_{y}=\sigma_{y}, \alpha_{z}=0, \beta=\sigma_{z} \] 3. 对于三维空间的狄拉克方程，并没有四个 \(2 \times 2\) 矩阵满足关系 \(1.2 。 \alpha_{i}\) 和 \(\beta\) 需要扩展为 \(4 \times 4\) 的矩阵

\[ \alpha_{i}=\left(\begin{array}{cc} 0 & \sigma_{i} \\ \sigma_{i} & 0 \end{array}\right), \beta=\left(\begin{array}{cc} \sigma_{0} & 0 \\ 0 & -\sigma_{0} \end{array}\right) \] #### 色散

利用泡利矩阵，哈密顿量所对应的相对论的能量 \(-\) 动量关系可表示为 \[ E^{2}=p^{2} c^{2}+m^{2} c^{4} \] 如果质量项趋近于零，能量和动量为线性关系。这同由非相对论自由粒子的薛定谔方程 \(p^{2} \Psi / 2 m=E \Psi\) 得到的能量对动量的平方依赖关系形成鲜明对比。

石墨烯: 无质量狄拉克电子

石墨烯无自旋 (spinless) 的紧束溥哈密顿量可表示为 \[ H _{0}=t \sum_{r_{A}} \sum_{j=1}^{3} c_{B}^{\dagger}\left(r_{A}+ d _{j}\right) c_{A}\left( r _{A}\right)+H . c \] 其中

  • \(t\) 是最近邻的 \(p_{z}\) 轨道之间的跃迁能量
  • \(d _{j}\) 是上述的由 \(A\) 子晶格碳原子指向三个最近邻 \(B\) 子晶格碳原子的矢量。
  • \(c_{\alpha}\left(r_{i}\right)\) 是在位置 \(r_{i}\) 处的电子潭灭算符， \(c_{\alpha}^{\dagger}\left(r_{i}\right)\) 在位置 \(r_{i}\) 处的电子产生算符。
  • \(\alpha= A , B\) 是对应 \(A , B\) 子晶格的指标。 \(H . c .\) 代表厄米共轨。

借助傅立叶变换， \(c_{\alpha}\left(r_{i}\right)=\frac{1}{\sqrt{N}} \sum_{k} e^{-i k \cdot r_{i}} c_{\alpha}(k)(N\) 为原胞的总数)，可以将哈密顿量写为， \[ H _{0}=\sum_{ k } c_{\alpha}^{\dagger}( k )\left[ H _{ k }\right]_{\alpha \beta} c_{\beta}( k ) \] 其中

  • 以两个不等价的碳原子的 \(p_{z}\) 轨道作为布洛赫基，无自旋的布洛赫哈密顿量, \(H _{k},\) 表示为

\[ H _{k}=t_{k}^{\prime} \sigma_{x}+t_{k}^{\prime \prime} \sigma_{y} \]

系数 \[ t_{k}=\sum_{j=1}^{3}-t \exp \left(-i k \cdot d _{j}\right) \equiv t_{k}^{\prime}+i t_{k}^{\prime \prime} \] 其中

  • \(t_{k}^{\prime}\) 和 \(t_{k}^{\prime \prime}\) 分别表示 \(t_{k}\) 的实部和虚部。

对称性

  1. 空间反演操作交换 A 和 B 两套子晶格: \(P =\sigma_{x}\)
  2. 时间反演操作使波函数的相位取复共斩，它的操作算符表示为 \(T =\sigma_{0} K\)

能带

在 \(K _{\pm}\) 附近，将波矢 \(k\) 分为 \(K _{\pm}\) 和 \(q\) 两部分，即 \[ k = K _{\pm}+ q \] 其中 \(q \ll\left| K _{\pm}\right|\)

对哈密顿量进行低能展开，由此得到在 \(K _{\pm}\) 附近石墨烯的低能有效模型 \[ \begin{aligned} H _{0}&=\hbar v_{ F }\left(\begin{array}{cc} 0 & \tau q_{x}-i q_{y} \\ \tau q_{x}+i q_{y} & 0 \end{array}\right) \\&=\hbar v_{ F }\left(\tau \sigma_{x} q_{x}+\sigma_{y} q_{y}\right) \end{aligned} \] 其中

  • \(\tau=\pm 1\) 是谷指标，分别对应 \(K _{\pm}\)
  • \(v_{ F }\) 是费米速度。

能谱 \[ \varepsilon_{\pm}( q )=\pm \hbar v_{ F } q \] 石墨烯的低能能带呈线性量色散，满足无质量的二维狄拉克方程。因此，石墨烯的价带和导带所组成的雉形被称为狄拉克雉，价带和导带的交点被称为狄拉克点，相应的电子态被称为无质量的狄拉克费米子。石墨烯是一类典型的狄拉克材料。

说明

在此，需要说明几点：

    1. 对于狄拉克材料 (例如石墨烯)，狄拉克方程只用来描述低能区域的电子能带特性，这一区域只占整个布里渊区的很小的一部分 $( q | K _{}|) $
    1. 在狄拉克材料中，哈密顿量中的 \(\sigma\) 矩阵可以对应真实的自旋，也可以对应赝自旋。像在上述石墨烯中， \(\sigma\) 矩阵对应子晶格的赝自旋。上自旋”和“下自旋”分别对应 A 和 B 两套子晶格。不同于真自旋，这一赝自旋并没有对磁场的响应。上述石墨烯的低能哈密顿量中没有描述电子的真自旋，是一个无自旋的模型。石墨烯的狄拉克锥，具有两重自旋简并。
    1. 石墨烯中狄拉克点是稳定的。在不考虑自旋轨道耦合作用的前提下，系统的时间反演对称和空间反演对称性确保了其狄拉克点的存在。
    1. 在哈密顿量中考虑更高阶的跃迁项(比如，次近邻跃迁) 或者破坏了石墨烯其它对称性 (比如，\(C_{3}\) 旋转对称性),只会改变狄拉克雉的形状或者移动狄拉克点的位置，狄拉克点仍是存在的。

有质量狄拉克电子

对于石墨烯的低能有效模型，它只存在反对角元，而没有对角元。 \[ H _{0}=\hbar v_{ F }\left(\begin{array}{cc} 0 & \tau q_{x}-i q_{y} \\ \tau q_{x}+i q_{y} & 0 \end{array}\right) \] 考虑自旋轨道塊合相互作用，或者打破系统的时间反演对称性或空间反演对称性 \[ H _{1}=\hbar v_{ F }\left(\tau \sigma_{x} q_{x}+\sigma_{y} q_{y}\right)+M \sigma_{z \circ} \] 其相应的能量色散关系为 \[ \varepsilon_{\pm}( q )=\pm \sqrt{\left(\hbar v_{ F } q\right)^{2}+M^{2}} \] 由此，狄拉克点处打开了带隙，狄拉克电子获得质量

值得注意的是，高能物理中，自由粒子的质量项 \(M\) 同波矢 \(k\) 无关。而对于狄拉克材料体系，等效质量 \(M\) 可以依赖于波矢 \(k(\) 低能能带中表现为谷指标 \(),\) 也可以与自旋相关。

通过不同的方式引入质量项，为有带隙的石墨烯体系中带来了很多新奇的物理，引起了凝聚态物理界广泛关注。

  1. 引入打破空间反演对称性的 AB 子晶格交错势场，人们在石墨烯中提出了谷相关的物理概念。其中，谷材料能呈现谷依靠的 旋光选择性和谷霍尔效应。
  2. 当引入较强的自旋轨道相互作用时，石墨烯呈现量子自旋霍尔效应(QSHE)，亦称为二维拓扑绝缘体，它的边界态呈线性色散，满足一维狄拉克方程。量子自旋霍尔效应在三维体系中对应拓扑绝缘体，其表面态满足二维狄拉克方程。因而，拓扑绝缘体是一种新颖的狄拉克材料体系。
  3. 引入打破时间反演对称性的交换场项，石墨烯会有量子反常霍尔效应(QAHE)。在磁性渗杂的拓扑绝缘体中，量子反常霍尔效应最近在实验上得以实现。谷，作为新颖的电子自由度，通过谷霍尔效应能提供了新的信息传递的方式; 而量子化的自旋或反常霍尔效应则能提供了无耗散的受拓扑保护的界面导电通道

import numpy as np
from math import *
import matplotlib.pyplot as plt


def main():
    n =  0.5
    k1 = np.arange(-n*pi, n*pi, n/20)
    k2 = np.arange(-n*pi, n*pi, n/20)
    plot_bands_two_dimension(k1, k2, hamiltonian)


def hamiltonian(kx,kz,ky=0):  # Weyl semimetal
    A = 1
    M0 = 1
    M1 = 1
    H = A*(kx*sigma_x()+ky*sigma_y())+(M0-M1*(kx**2+ky**2+kz**2))*sigma_z()
    return H


def sigma_x():
    return np.array([[0, 1],[1, 0]])


def sigma_y():
    return np.array([[0, -1j],[1j, 0]])


def sigma_z():
    return np.array([[1, 0],[0, -1]])


def plot_bands_two_dimension(k1, k2, hamiltonian):
    from mpl_toolkits.mplot3d import Axes3D
    from matplotlib import cm
    from matplotlib.ticker import LinearLocator, FormatStrFormatter
    dim = hamiltonian(0, 0).shape[0]
    dim1 = k1.shape[0]
    dim2 = k2.shape[0]
    eigenvalue_k = np.zeros((dim2, dim1, dim))
    i0 = 0
    for k10 in k1:
        j0 = 0
        for k20 in k2:
            matrix0 = hamiltonian(k10, k20)
            eigenvalue, eigenvector = np.linalg.eig(matrix0)
            eigenvalue_k[j0, i0, :] = np.sort(np.real(eigenvalue[:]))
            j0 += 1
        i0 += 1
    fig = plt.figure()
    ax = fig.gca(projection='3d')
    k1, k2 = np.meshgrid(k1, k2)
    for dim0 in range(dim):
        ax.plot_surface(k1, k2, eigenvalue_k[:, :, dim0], cmap=cm.coolwarm, linewidth=0, antialiased=False)
    plt.xlabel('kx')
    plt.ylabel('kz') 
    ax.set_zlabel('E')  
    plt.show()


if __name__ == '__main__':
    main()

import numpy as np
from math import *
import matplotlib.pyplot as plt


def main():
    n =  0.5
    k1 = np.arange(-n*pi, n*pi, n/50)
    k2 = np.arange(-n*pi, n*pi, n/50)
    plot_bands_two_dimension_direct(k1, k2, hamiltonian)


def hamiltonian(kx,kz,ky=0):  # surface states of Weyl semimetal
    A = 1
    H = A*kx
    return H


def sigma_x():
    return np.array([[0, 1],[1, 0]])


def sigma_y():
    return np.array([[0, -1j],[1j, 0]])


def sigma_z():
    return np.array([[1, 0],[0, -1]])


def plot_bands_two_dimension_direct(k1, k2, hamiltonian):
    from mpl_toolkits.mplot3d import Axes3D
    from matplotlib import cm
    from matplotlib.ticker import LinearLocator, FormatStrFormatter
    fig = plt.figure()
    ax = fig.gca(projection='3d')
    dim1 = k1.shape[0]
    dim2 = k2.shape[0]
    eigenvalue_k = np.zeros((dim2, dim1))
    i0 = 0   
    for k10 in k1:
        j0 = 0
        for k20 in k2:
            if (k10**2+k20**2 <= 1):
                eigenvalue_k[j0, i0] = hamiltonian(k10, k20)
            else:
                eigenvalue_k[j0, i0] = 'nan'
            j0 += 1
        i0 += 1
    k1, k2 = np.meshgrid(k1, k2)
    ax.plot_surface(k1, k2, eigenvalue_k, cmap=cm.coolwarm, linewidth=0, antialiased=False) 
    plt.xlabel('kx')
    plt.ylabel('kz')
    ax.set_zlabel('E')  
    plt.show()


if __name__ == '__main__':
    main()
