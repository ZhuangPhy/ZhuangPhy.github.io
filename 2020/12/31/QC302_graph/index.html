前言

文1：变分量子态本征求解器

量子计算在求解一些问题时较经典计算有指数加速。例如，在查询复杂度意义下

  1. 对于Deutsch-Jozsa Problem，其量子算法相对于确定性经典算法有指数量级的加速 [1]；
  2. 对于Simon's Problem，其量子算法相对于有界错误经典算法有指数量级的加速 [2]。

基于这些研究，Shor 设计了高效求解素因子分解的量子算法 [3]，该问题在经典计算机上目前还未有多项式时间算法。

自然地，人们会问，量子计算对所有问题都有指数加速吗？答案是否定的。

可以证明，对定义在全局上的布尔函数（total function） \[ P : \Sigma^{ n } \rightarrow\{0,1\} \] 量子计算至多可以提供多项式量级的加速。

更准确地说，其经典确定性查询复杂度 \(D(P)\) (忽略常数因子意义下) 至多是量子查询复杂度 \(Q ( P )\) 的 6 次方[4], 即 \[ D ( P )= O \left( Q ^{6}( P )\right) \] 更进一步地, 即使 \(P\) 是部分的布尔函数（partial function）, 即其仅定义在 \(\Sigma^{ n }\) 的某个子集 \(S\) 上, 若 \(S\) 在S \(_{ n }\) 下保持封闭, 且 \(P\) 是对称函数, 则量子计算仍至多提供多项式量级的加速 [5]。Ben-David 进一步证明了即使允许 \(P\) 是 \(S\) 上的任意布尔函数 (不特指对称函数) ，量子计算相较于经典随机查询算法至多仅能提供多项式量级加速 [11]。

一个重要的问题是，什么样的函数类量子存在指数加速？

由以上研究可知，此函数类必须是部分函数且对称性不能太强。图属性函数是一种广泛研究的布尔函数，其对称性略弱于对称函数。2011年，Ambainis, Childs, 和Liu提出公开问题 [9]：量子计算对图属性函数有指数量级加速吗？

  1. 事实上，对于定义在全局上的图属性函数，04 年文献 [10] 就已经证明了 \(D ( P )= O \left( Q ^{4}( P )\right)\)。
  2. 对更一般的情况 ，文献 [6, 7] 证明了，若输入采用邻接矩阵，量子算法对图属性函数的计算至多有多项式量级加速。
  3. 除此之外，文献 [6] 证明了在邻接链表查询模型下，量子算法*对图属性函数的计算*有指数量级加速。
  4. 文献 [7] 还证明了如果函数 P 和定义域 S 在具有充分传递性的群作用下保持不变，则量子算法就只有多项式加速。
