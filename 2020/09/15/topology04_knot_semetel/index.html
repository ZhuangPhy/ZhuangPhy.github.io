纽结理论

纽结理论（Knot theory）是拓扑学的一个分支，研究抽象的绳子缠绕打结的性质。

为了完成对不同纽结的分类，大量的数学工具被发掘出来，比如（Jones polynomial）。 \[ \begin{array}{|l|l|} \hline \text { Chern–Simons gauge group G } & {\text { Knot polynomial or invariant }} \\ \hline SO ( N ) & \text { Kauffman polynomial } \\ \hline SU ( N ) & \text { HOMFLY polynomial } \\ \hline SU (2) / SO(3) & \text {Jones polynomial } \\ \hline U (1) & \text { Winding Number } \\ \hline \end{array} \]

\[ \begin{array}{|l|l|} \hline \text { 陈西规范群G } & {\text { 纽结多项式或不变量 }} \\ \hline SO ( N ) & \text { 考夫曼多项式 } \\ \hline SU ( N ) & \text { HOMFLY多项式 } \\ \hline SU (2) \text { 或SO(3) } & \text { 铺斯多项式 (跟括号多项式有关) } \\ \hline U (1) & \text { 环绕数 } \\ \hline \end{array} \]

3D的Chern–Simons理论生成很多重要的纽结多项式和纽结不变量，Chern–Simons理论描述三维拓扑量子场论，可以描述描述分数量子霍尔效应（FQHE）。所以纽结理论在以下领域是一个重要的数学工具

  • 不受任何对称性保护的非厄米奇异线(exceptional line)半金属

在这些纽结半金属中，能带的简并会在三维布里渊区中形成丰富结构，比如环，链，或者是非平凡的纽结。如何系统的分类和刻画这些非平凡的物理结构呢？

  • 对于陈绝缘体（Chern insulator）：

Chern Number的改变可以通过分析在相变点附近的狄拉克方程的质量项得到。在相变点附近，狄拉克质量项控制着布里渊区贝里曲率（Berry curvature）的变化，陈数的改变可以通过计算布里渊区内所有狄拉克点附近的质量项的变化之和求得。

  • 类似的，对于纽结半金属（nodal line semimetals）：

在纽结相变点附近，节线或者奇异线的局域演化可以和琼斯多项式的整体变化一一对应联系起来。相变点对应的节线存在一些离散的节线相交点，比如数字8可以看作是一个临界点，它有一个节线相交点。通过分析这些相交点附近节线的局域演化，就可以得到对应的琼斯多项式的整体变化。这些变化同时也会和系统的低能物理变化联系起来。

上述结果也可以推广到纽结超导体。

Introduction and motivation

Knot theory

Knot: an embedding of a circle in \(3 D\) space Knot diagram: a projection of the knot, together with the data of over- and under-crossings.

Knot theory: a mathematical theory that classifies different knots and links.

Knot invariant: a quantity defined for each knot which is the same for equivalent knots.

Our approach

  1. For a given critical phase, what kind of knot phases (including the trivial knot phase) can be generated by the perturbation, namely, we need to find a method to systematically write down all the possible generated phase.
  2. What is the topological invariant of these generated phases, namely, we need to find a knot invariant to classify them.
  3. We need to consider whether there exist any additional physical constraints on the generated phases.
For the first step: Perturbation generated phase

\[ \begin{aligned} &\text { Critical phase }\\ & \Downarrow \\ &\text { Perturbation generated phases } \end{aligned} \]

since the perturbation is weak, only the local evolutions around the TPs finally determine the linking or knotting properties of the nodal lines. Then, we focus on the touching point, where two nodal lines touch together.

Knot invariant

\[ \text { We need a knot invariant to classify them } \] The Jones polynomial \(J\) is a topological invariant for oriented links with the following properties:

  • it takes values in \(Z \left[t^{1 / 2}, t^{-1 / 2}\right]\)
  • it satisfies \(J(O)=1,\) where \(O\) denotes the unknot, as shown in Fig. \(5 .\) it satisfies the skein relation

\[ t^{-1} J\left(L_{+}\right)-t J\left(L_{-}\right)+\left(t^{-1 / 2}-t^{1 / 2}\right) J\left(L_{0}\right)=0 \]

whenever \(L_{+}, L_{-}\) and \(L_{0}\) are three oriented links, which have diagrams that are identical except in a small region where they are given as shown in Fig. \(6 .\)

Calculation of Jones polynomial

The first example we considered is shown in Fig. \(7 .\) Obviously, \(L_{+000}\) and \(L_{-000}\) are both equivalent to an unknot. Hence \(J\left(L_{+000}\right)=J\left(L_{-000}\right)=1 .\) Using the skein relation, one can obtain, \[ t^{-1}-t+\left(t^{-1 / 2}-t^{1 / 2}\right) J\left(L_{0000}\right)=0 \] After some simple calculation, we have \(J\left(L_{0000}\right)=-t^{1 / 2}-t^{-1 / 2} .\) Hence, the Jones polynomials of single unknot is different from the one of the two separated unknots.

second example

The second example is shown in Fig. \(8 .\) The skein relation relates the Hopf-link to the unknot. since we know \(J\left(L_{+00-}\right)=-t^{1 / 2}-t^{-1 / 2}\) and \(J\left(L_{+000}\right)=1 .\) Hence based on the skein relation, \[ t^{-1} J\left(L_{+00+}\right)-t\left(-t^{1 / 2}-t^{-1 / 2}\right)+\left(t^{-1 / 2}-t^{1 / 2}\right)=0 \] One can obtain, the Hopf-link has \(J\left(L_{+00+}\right)=-t^{5 / 2}-t^{1 / 2}\)
