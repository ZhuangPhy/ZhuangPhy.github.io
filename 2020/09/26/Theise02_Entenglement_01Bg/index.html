摘要

低维关联量子物态是当前凝聚态物理的研究热点，其非平庸拓扑信息很大程度上隐藏在基态波函数的量子纠缠特性中。 \[ 拓扑 \leftrightarrows 纠缠 \] 通过将体系一个子部分的约化密度矩阵取对数得到的纠细哈密顿量，以及其本征值谱、即纠细谱，则是定量描述纠细的工具。通过左右划分得到的边缘纠细谱的低能部分与拓扑态边缘激发能谱一一对应，从而可以反映拓扑信息。

本文集中于一维有对称性的短程纠缠态、

  • 即对称性保护拓扑态(SPT态)
  • 通过研究其共振价键固体(VBS)

构型的严格 可解点基态波函数的量子纠缠特性，来刻画SPT态的拓扑性质。主要内容如下。

第一部分

在第一部分我们首先介绍了反铁磁整数自旋链的Haldane相，以及作为其严格可解点的AKLT模型。我们说明只有奇数自旋AKLT态才属于非平庸的SPT态，并以该模型为例证明，SPT态边缘纠缠谱简并度中，既有拓扑相关的部分、也有非普适的部分，而非平庸的前者可通过由对称性决定的拓扑退纠缠算符除去。该工作厘清了SPT态边缘纠缠细谱的结构，对用纠缠谱刻画拓扑态有指导作用。

第二部分

从第二部分起，我们研究如何通过量子纠缠，从VBS型基态波函数中读取原SPT态与平庸相之间的量子临界点的信息。该临界点不能用朗道的经典理论描述，其低能元激发是原VBS的边缘自由度。我们对VBS波函数引入对称的延展划分：将每l个格点组合成 一个了块，以所有奇数了块作为了系统，得到体纠缠细哈密顿量\(H_{E}\)。我们说明，\(H_{E}\) 在 \(l\) 选取合适的情况下就描述退禁闭的边缘自由度之间的渗透，从而给出量子临界点的完整能谱。我们证明自旋S-AKLT 态的\(H_{E}\)是自旋 \(S / 2\) 的海森堡模型，对于非平庸的奇数 $S $ 并取 \(l=\)偶数，\(H_{E}\)就描述Haldane相与平庸相之间的量子临界点，低能有效理论是\(S U(2)\),WZW场论。

第三部分

在第三部分，我们将延展划分的方法运用到具有 \(SO (5)\) 对称性的自旋2-VBS态，并证明\(H_{E}\) 在 \(l=\) 奇数 时就描述与原SPT 态相连的量子临界点，低能有效 理论是 \(S U(4)\), WZW场论。在第四部分，我们用自旋构造了具有 \(SU(N)(N \geq 3)\) 结柄端边缘互为共斩，破坏镜像对称性; 而当总长度为奇数时，两端边缘相同，属于非平庸SPT 态，且其对应的量子临界点由取 \(l=\)奇数的体纠缠哈密顿量描述，低能有效理论是 \(S U(N)\), WZW场论。

通过延展划分我们不仅可以得到量子临界点低能有效理论，更能写出对应的哈密顿量，从而建立了体-边缘-量子临界点对应，深化了对拓扑态的理解。

以波函数为重心: 一种新的研究范式

新旧

从还原论到演生论

  • 还原论的作用

在物理学的历史中，还原论(Reductionism)一直是研究者的信仰。人们相信，一切复杂系统中出现的现象都可以还原到基本组成单元的行为上，只要我们透彻认识了最基本组成单元的运动规律，我们就可以以此为基础反方向推导所有系统的运动规律。以此信念为指导，人们对物质的认知从宏观物质深化到分子、原子、质子中子、夸克乃至于超弦，可以说，还原论在人类科学的发展历程中起了巨大的推进作用。

  • 还原论的局限

但是，随着人们对复杂体系认识的深入，还原论越来越显得力不从心。这是因为，复杂体系的自由度数随着粒子数的增加而指数增长，很快就超出了数值计算的极限，让整个体系的行为预测变得不可能。

  • 演生(Emergence)

20世纪中叶以来，人们认识到，物质世界根据能量尺度的不同可以划分为不同的层次，各个层次之内各有其运动的基本规律，这些基本规律作为本层次内的有效理论存在，而与别的能量尺度上的规律并无强烈关系。于是，不同层次的体系会展现出多样的物理现象，这些现象独立于组成部分的微观细节，也无法用组成单元本身的运动规律来预测，这就是演生(Emergence)现象。一个典型的例子就是连续相变的重 整化群理论，作为其基础的标度不变性只是在对应尺度上的有效理论，并不能由更基本粒子的运动规律推导出来。诺贝尔奖得主.W.Anderson对演生现象有一 句著名的概括: More is differen

强关联

现代凝聚态物理、尤其是强关联体系，由于其粒子数和粒子间的强关联大大增加了体系的复杂性，成为新奇宏观量子现象的频发地，典型的如

  • 量子霍尔效应
  • 高温超导体

人们在这些体系中观测到大量基本粒子中无法想象的新奇现象，比如

  • 电荷分数化
  • 满足分数统计甚至非阿贝尔统计的任意子

并由此开发出新的研究方法、并预言更新奇的量子现象。

这些发展除了有其深刻的理论意义，还具有潜在的巨大利用价值，比如利用任意子构造容错率极高的量子计算机 、发展性能更好的量子材料等。甚至有可能利用凝聚态体系中的演生现象，来解决基本粒子物理中由来已久的问题，比如Majorana费米子的存在性问题。因为这些原因，凝聚态物理成为当代物理学最蓬勃和兴盛的分支之一。

相变理论的传统范式

凝聚态物理的基本任务之一是刻画物态以及物态之间的转变。对于经典物态这个任务可以用朗道的对称性破缺理论很好地完成。

  • 传统范式的局限

而对极低温接近量子极限下的物态，则更为复杂。一个典型的例子是分数量子霍尔效应，在每一个平台上的横向霍尔电阻都是数学精度的值，但是不同电阻平台上的态都具有相同的对称性，也不具有局域的序参量，这说明朗道理论在描述这些态的时候并不 足够，这些平台上的物态包含一种超出经典范畴的结构。

在传统的少体或者弱关联量子问题中，典型的研究方式是用平均场理论求解哈密顿量，再根据需要做微扰展开，但在分数量子霍尔效应以及现代凝聚态物理面对的一系列体系中，粒子间的关联程度如此强大以至于平均场理论失效，而热力学量级的自由度数又让求解哈密顿量成为不可能，于是提出一种新的研究范式(Paradigm)迫在眉捷。

新研究范式的起源

绕开哈密顿量，构造出有效的基态波函数，研究其中包含的物态信息。

这种新研究范式的突破起源于Laughlin，他在解释分数量子霍尔效应实验的 时候，开创性地绕开哈密顿量，而直接写出了在不同霍尔电阻平台上的基态波函数，即著名的Laughlin 波函数。对于填充因子为 \(v=1 / m\) 的平台，其基态波函数的形式为: \[ \Psi_{L}=\prod_{i<j}\left(z_{i}-z_{j}\right)^{m} e^{-\sum_{i}\left|z_{i}\right|^{2} / 4} \] 其中 \(z_{i}=x_{i}+i y_{i}\) 是第i个粒子的坐标。在此波函数的基础上，Laughlin预言了这个态的边缘激发形式、和体激发满足的分数电荷性质，这些都在后来的实验中被验证，Laughlin也因此获得了1998年的诺贝尔物理学奖。

在此之后，人们意识到Laughlin的方法是一类有效的研究强关联量子物态的方法，即直接构造出有效的基态波函数，研究其中包含的物态信息，之后再根据需要构造以此波函数为基态的母哈密顿量(Parent Hamiltonian) 。在很多时候，尽管构造出的母哈密顿量与体系真实的哈密顿量有较大的不同，但是其基态波函数的性质却非常相似。鉴于在极低温的量子极限下，我们只关心低能部分的物理，所以这种研究范式是有效的。从此，对于强关联量子体系的研究重点，就转向了对各种具有非平庸结构的基态波函数的研究，并大大深化了人们对量子世界物质的理解。

拓扑序的刻画：纠缠

非局域的拓扑序

人们认识到，对于这类新奇量子物态，其波函数中通常并不存在局域的序参量，但是存在非平庸的量子序。这种序与体系的微观局域细节无关，而仅取决于体系的拓扑结构，因而在局域扰动下有稳定(Robust)的结构，并可以被反映在各种可观测性质中，比如体激发的分数统计性质、分数化的边缘激发等等。这种稳定的新型的序就被称为拓扑序，具有拓扑序的量子物质就被称为拓扑物态。至今为止，人们对于无能隙的拓扑物态理解还比较欠缺，但对于有能隙态已经有比较深刻的认识。

纠缠态分类

近年来，凝聚态物理学家逐渐认识到量子纠缠在刻画拓扑序的时候发挥着重要的作用，并根据纠缠特性的不同，有能隙拓扑物态可以分为长程纠缠 和短程纠缠两种。

定义是，在封闭的空间中(如一维的周期性链和二维的Torus等)中，

  • 基态波函数的简并度大于1，则称其为长程纠缠态
  • 反之则称为短程纠缠态。

根据对称性和量子纠缠特性，有能隙的拓扑态可以被分为四类，如表1.1所示。 \[ \begin{array}{lcc} \hline & \text { 短程纠缠 } & \text { 长程纠缠 } \\ \hline \text { 有对称性 } & \text { SPT 态 } & \text { SET 态 } \\ \text { 无对称性 } & \text { 平庸态 } & \text { LRE 态 } \\ \hline \end{array} \] 我们的研究集中在具有对称性的短程纠缠态、即对称性保护拓扑态(Symmetry-Protected-Topological Phase，以下简称SPT态)。

SPT对称性保护拓扑态

SPT“对称性 保护”定义：

所谓“对称性 保护”，物理上的体现为，在保证对称性的前提下，我们无法将一个SPT态绝 热地(Adiabatically) 转变为平庸的真空态，除非经历能隙的关闭过程。

SPT态的基本特征是在边缘存在分数(Fractionalized) 的自由度，该自由度禁闭(Confined)在边缘附近，构成对称群的投影表示(Projective epresentation) \(^{[17]}\)

对于SPT态，人们认识到它大致可以分为以下几种：

  • (1)玻色子体系，其拓扑分类由文小刚等人利用群上同调(Group Cohomology)理论给出[17]，典型例子是一维自旋模型中的Haldane相;
  • (2)无相互作用费米子，如近年引起广泛关注的拓扑绝缘体，其分类由Kitaev 和Ryu 等人给出 ;
  • (3)相互作用费米子体系（还不完备）。

在本文中，我们研究一维系统中的SPT态，其典型的不动点(Fixed Point)构型是共振价键固体态(Valence-Bond-Solid State，以下简称VBS态)

纠缠熵与纠缠谱：刻画拓扑物态的有效工具

纠缠熵

如何计算纠缠熵

当我们刻画量子纠缠时，需要将体系分为两部分，记为A与B，写出体系的

  • 密度算符 \(\rho=|\psi\rangle\langle\psi|\)

然后对B部分的自由度积分，得到A部分的

  • 约化密度矩阵 \(\rho_{A}=T r_{B} \rho_{\circ}\)

信息纠缠，都包含在 \(\rho_{A}\) 中。我们引入的第一个量是纠缠熵，它被定义 为 \(\rho_{A}\) 的von Neumann熵: \[ S=-\operatorname{Tr}\left(\rho_{A} \ln \rho_{A}\right) \] 纠缠熵刻画了A、B部分之间纠缠程度的大小。

面积定律

可以证明，对于有能隙的态，纠缠熵满足所谓面积定律: \[ S=\alpha L-\gamma \] 其中

  • \(L\)表示A、B部分之间接触面积的大小，对于一维体系接触面就是一个点，于是\(L\) =常数，
  • \(\alpha\) 是一个与体系有关的系数，
  • 而 \(\gamma\) 是一个面积定律的修正项。可以 证明，对于短程纠细的态， \(\gamma=0,\) 而对于长程纠细态 \(\gamma \neq 0,\) 所以 \(\gamma\) 被称为拓扑纠缠熵 (Topological Entanglement Entropy) \(^{[29,30]}\)

example：

而对于一维的无能隙临界态，因为存在一类共形不变性(Conformal Invariance，它是标度不变性的一种推广)，利用共形场论的知识，可以证明，纠缠熵 满足如下Calabrese-Cardy公式 \[ S=\frac{c}{3} \ln \left[\frac{L}{\pi} \sin \left(\frac{\pi L_{A}}{L}\right)\right]+S_{0} \] 其中

  • $L $, \(L_{A}\) 分别是总系统与A部分子系统的长度。这个公式在周期性边条件(PBC)下成立，在开放边条件下会有一定的修正。
  • \(S_{0}\)是一个非普适的量
  • 而c是普适的，被称为中心荷(Central Charge)，它是标记临界态普适类(Universality Class) 的一个重要参数。
    1. 对于c < 1的情况，普适类被c 唯一确定，
    2. 对于 \(c<1\) 则不是,我们需要更多的信息

纠缠熵的不足

纠缠熵在刻画拓扑序中发挥了重要的作用，但是它也有各种不足[33]。

  • 比如它的一般形式是基于标度理论给出的，在数值计算中难以达到很高的精确度。
  • 最重要的是，纠缠熵只是一个数字，难以完全确定丰富的拓扑序。

纠缠谱

作为纠缠熵的深化和推广，在2008年，Haldane引入纠缠谱的概念。将约化密度矩阵写成如下形式: \[ \rho_{A}=e^{-H_{E}} \] 其中

  • \(H_{E}\) 被称为纠缠哈密顿量(Entanglement Hamiltonian)，它有类似于量子体系的 热力学哈密顿量的形式
  • 纠缠谱就被定义为 \(H_{E}\) 的能谱 \(\left\{\xi_{i}\right\}\)

显然，纠缠熵就是纠缠哈密顿量的热力学熵(取纠缠温度$==1 $) ，而纠缠谱包含着更多的信息。

纠缠谱的方法一经提出，很快被利用到各种不同的体系中，如

  • FQHE体系
  • 自旋链体系
  • 拓扑绝缘体和拓扑超导体体系
  • 量子临界体系

在特定的系统中，该方法的有效性也可以被解析地证明

对应：纠缠谱-拓扑边缘激发

一个重要的观点是：我们在划分体系的时候相当于在A、B分界处产生了一条虚拟的边界，于是 \(H_{E}\) 作为描述A部分的哈密顿量会包含有A部分边缘自由度的信息，于是，纠缠谱中很可能包含原体系边缘激发的信息，这也是Haldane最初引入纠缠谱时做的假设，即：拓扑态基态波函数纠缠谱的低能部分，与原拓扑体系的边缘激发能谱有一一对应的关系。

如何划分

显然，纠缠谱的性质与我们如何划分体系有密切关系。至今为止，大致 显有四种不同种类的划分方式，

    1. 轨道空间的划分，这也是Haldane最早引入的划分方式;
    1. 粒子数空间划分，这样得到的纠塵谱在一定体系中可以得到体激发的信息
  • (3)动量空间的划分，这在某些无能隙的临界体系中可以运 用
  • (4)实空间的划分，这也是我们当前工作采用的划分方式。

在前人的工作中，有两类实空间的划分被应用(见图 1.2，以一维为例):

    1. 将体系沿中间一分为二的划分，这样的划分可以得到单个边缘的激发能谱，我们称之为边缘纠缠谱(Edge Entanglement Spectrum);
  • (2)将体系中截取一段的划分,对于有能隙态，这样得到的纠缠谱可以反映两个边缘自由度之间的相互作用形式，对于一维临界态，此时的纠缠谱可以确定普适类与临界指数，这种纠缠谱被称为块纠缠谱(Block Entanglement Spectrum)。

在我们的工作中，我们将对于SPT态的VBS构型基态波函数引入第三种划 分，即延展性的划分(Extensive Bipartition)，如图 1.2(c)所示。容易看出，这样的划分可以产生一系列的边缘自由度，于是此时 \(H_{E}\) 是一个描述边缘自由度在体中相互作用的哈密顿量，因而我们称此时的纠缠谱为体纠缠谱(Bulk Entanglement Spectrum)。我们认为，在取合适的子块长度下，体纠缠谱可以反映原SPT态与平庸态之间的量子临界点能谱。

在此之前，我们先介绍一种典型的、也是人们认识最为充分的强关联体 系SPT态，即一维反铁磁自旋链中的Haldane相(Haldane Phase)。

Haldane相: 一种典型的SPT

Haldane相

反铁磁海森堡自旋模型

Haldane相产生于如下形式的反铁磁海森堡自旋模型中: \[ H=J \sum_{i} S _{i} \cdot S _{i+1}, \quad J>0 \] 对于自旋1/2的情况，Bethe在1931年给出的严格解证明，其能谱无能隙，自旋-自旋关联函数在长距离情况下成审函数衰减。在之后的半个世纪里，人们认为该结论对任意自旋都定性成立。

直到1983年，Haldane利用自旋相干态路径积分推导低能有效场论的方法证明，在自旋S很大的情况下，

  • 当S为整数时，有效作用量是1+1维的O(3)非线性 \(\sigma\) 模型，能谱有能隙;
  • 而在自旋为半奇数的时候，作用量会在 \(\sigma\) 模型的基础上增加一个拓扑项，受自旋1/2的严格解启发，Haldane认为该拓扑项的量子干涉作用会让整个能谱呈现无能隙状态。

Haldane 猜想

Haldane的结论虽然是基于大S 极限情况给出的，但他猜测对小自旋情况也成立，这就是著名的Haldane 猜想 :

Haldane 猜想：一维自旋S的反铁磁海森堡模型

  • 当S为半奇数时是无能隙的，自旋-自旋关联函数以幂函数形式衰减：
  • 当S为整数时是有能隙的，自旋-自旋关联函数在长距离下指数衰减。

这一猜想的正确性在后来的实验和数值研究中得到证实。自此，整数自旋反铁磁链的有能隙相就被称为Haldane相。

奇偶 Haldane相

近年来，人们认识到整数自旋Haldane相可以进一步分成两类：

  1. 奇数自旋的Haldane相受对称性保护，属于真正的非平庸SPT态，起保护作用的是以下任意一种或多种对称性：
    • (1)时间反演对称性(Time Reversal Symmetry)
    • (2)空间镜像对称性(Inversion Symmetry);
      1. \(Z_{2} \times Z_{2}\) 对称性，它由绕三个自旋轴中 的任意两个旋转180度的操作构成，是SO(3)的最大离散子群。
  2. 而偶数自旋的Haldane相则不是SPT态，它可以与平庸的直积态绝热相连。

Haldane相的区分标准

为了给这两类Haldane相提供一种可观测的区分标准，Pollmann等人提出，对于非平庸的奇数自旋情况，边缘纠缠谱的每个能级都是二重简并。为说明这个概念，考虑如下的自旋1哈密顿量: \[ H=J \sum_{i} S _{i} \cdot S _{i+1}+U_{z z} \sum_{i}\left(S_{i}^{z}\right)^{2}, \quad J>0 \] 显 然， \(U_{z z} / J\) 可 以 作 为 调 制 参 数 , 在 其 等 于0的 时 候 就 得 到Haldane相。Pollmann等人利用iTEBD的方法计算了在不同参数取值情况下的基态波函，并将该波函数沿中间分成左右两部分，计算边缘纠缠谱，他们的结果如图 所示。

可见

  • 在中间的Haldane相区间内，每一个纠缠能级的简并度都是偶数(大多数为2，还有偶然的4重简并)
  • 而在两端的非Haldane相，则会出现简并度为奇数的纠缠能级。于是，边缘纠缠谱的二重简并可以作为非平庸Haldane相的基本定义特征。

在后来的工作中，他们将这个结论推广到了任意整数自旋的情况，,并说明正是上述三种对称性中的任意一种保护了奇数自旋链中这一非平庸的二 重简并度。进一步，他们将这一观点做了推广：

任何基态波函数边缘纠缠谱貝有简并度的态都是拓扑非平庸的。

但是，纠缠谱的简并度都能作为SPT态的标志吗？我们将在一维SPT 态的框架内深入探究这个问题，并厘清边缘纠缠谱中简并度的涵义。

SPT与平庸态之间的相变: 超越朗道理论的量子临界点

无能隙的未解之谜

在第一小节里我们已经提到，对于有能隙的拓扑物态人们已经有比较深刻 的理解，但对于无能隙的量子物态则较为欠缺。顾名思义，无能隙则意味着存在能量任意小的激发，那么体系为何能在扰动下稳定地存在，对于这一最基础的问题，人们仍然没有很好的理解。

  1. 一种可能的假设是，d维无能隙的拓扑态都是某个d + 1维有能隙拓扑态的边缘态，那么该无能隙态的稳定性就由高一维的拓扑态稳定性所保证。
  2. 而另一种可能是，无能隙的拓扑态可以作为同一维度的有能隙拓扑态与平庸相(Trivial Phase，或称真空态)之间的量子临界点(Quantum Critical Point，以下简称QCP)存在，从而保证其稳定性。于是，研究拓扑物态与平庸相之间的量子临界点，就有了更重要的意义。

量子相变

正如SPT态的定义指出，在保持对称性的前提下，SPT与平庸态之间的转变必须经历能隙闭合的相变过程。可以证明，该相变一定是连续相变，从而是一个量子临界点。在相变点两侧的态具有一样的对称性，因而这个相变不能用朗道的对称性破缺理论来描述，它属于一类零温时的拓扑量子相变，其相变点被称为退禁闭(Deconfined)量子临界点。一个深刻而自然的问题由此产生:

这个量子临界点的信息，是否也像边缘激发能谱一样，被隐含在SPT 态的基态波函数中？如果是，该如何读取这个信息？

我们认为，拓扑态基态波函数就像这个态的DNA，所有的非平庸信息都被包括在内，包括量子临界点的信息。至于提取该信息的方法，首先注意到，SPT态与具有相同对称性的平庸态之间的区别在于，边缘自由度在前者中被禁闭，在后者中则退禁闭，并且作为临界点的低能元激发(Elementary Excitation)形式存在。于是，我们就需要从基态波函数中演生出一系列延展分布的退禁闭边缘激发。

对应关系

在上节我们论述过，拓扑态基态波函数的边缘纠缠谱与边缘激发的能谱有对应关系，尤其是，对VBS型不动点波函数，其边缘纠缠谱只包含边缘自由度的信息，而不包含非拓扑的信息。这提示我们对VBS型基态波函数采取延展性 的划分，由此产生一系列的边缘激发，如图 1.2(c) 所示。可以看出，此时纠缠哈密顿量 \(H_{E}\) 描述的是边缘自由度在体中的相互作用，因而我们称此时的纠缠谱为体纠缠谱。在本文中，我们将通过几个例子说明，在对VBS型基态波函数做延展 划分，并取合适的子块长度和适当的近似下，得到的体纠缠能谱就是与原SPT相连的量子临界点的能谱，一个简化的框架示意见图 1.4(a)。

显然，我们的工作是对用纠缠谱读取拓扑态基态波函数信息的一个深化，它将人们原有的体-边缘对应(Bulk-Edge Correspondence)推广到体-边缘-量子临界点对应(Bulk-Edge-QCP Correspondence)，如图 1.4(b)所示，在这三角对应中，量子纠缠起到了桥梁的作用。
